## <a name="create-an-aspnet-web-app"></a>Erstellen einer ASP.NET-Web-App
Erstellen Sie in Visual Studio 2017 ein neues Projekt. Dabei muss es sich derzeit um eine **ASP.NET Core-Webanwendung** handeln. Geben Sie dem Projekt den Namen **webfrontend**.

![](images/NewProjectDialog1.png)

Klicken Sie auf die Vorlage **Webanwendung (Model-View-Controller)**, und stellen Sie sicher, dass Sie aus den beiden Dropdownlisten im oberen Bereich des Dialogfelds **.NET Core** und **ASP.NET Core 2.0** auswÃ¤hlen. Klicken Sie auf **OK**, um das Projekt zu erstellen.
