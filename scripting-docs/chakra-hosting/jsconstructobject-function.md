# JsConstructObject-Funktion
Ruft eine Funktion als Konstruktor auf.

```

   _In_ JsValueRef function,  
   _In_reads_(argumentCount) JsValueRef *arguments,  
   _In_ unsigned short argumentCount,  
   _Out_opt_ JsValueRef *result  
);
```  
