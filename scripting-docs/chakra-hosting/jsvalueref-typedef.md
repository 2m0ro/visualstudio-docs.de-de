# <a name="jsvalueref-typedef"></a>JsValueRef-TypeDef
Ein Verweis auf einen JavaScript-Wert.  
  
## <a name="syntax"></a>Syntax  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>Hinweise  
 Ein JavaScript-Wert weist einen der folgenden Wertetypen auf: Nicht definiert, Null, Boolesch, Zeichenfolge, Zahl oder Objekt.  
  
## <a name="requirements"></a>Anforderungen  
 **Header:** jsrt.h
