# JsPropertyIdType Enumeration
Typenumeration einer JavaScript\-Eigenschaft.  
  
## Syntax  
  
```vb  
enum JsPropertyIdType;  
  
```  
  
## Mitglieder  
  
### Werte  
  
|Name|Beschreibung|  
|----------|------------------|  
|JsPropertyIdTypeString|Typenumeration einer JavaScript\-Zeichenfolgeneigenschaft.|  
|JsPropertyIdTypeSymbol|Typenumeration einer JavaScript\-Symboleigenschaft.|
