# <a name="global-object-javascript"></a>Global-Objekt (JavaScript)
Ein systeminternes Objekt, dessen Zweck darin, globale Funktionen und Konstanten in einem Objekt zu erfassen.  
  
## <a name="remarks"></a>Hinweise  
 Die `Global` Objekt wird nie direkt verwendet und kann nicht erstellt werden, mithilfe der `new` Operator. Sie wird bei der Initialisierung des Skriptmoduls Aufgabenstellungen seine Funktionen und Konstanten verf√ºgbaren sofort erstellt.
