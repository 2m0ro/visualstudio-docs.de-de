# Math.asin-Funktion (JavaScript)
Gibt den Arkussinus einer Zahl zurück.  
  
## Syntax  
  
```  
  
Math.asin(number)   
```  
  
## Hinweise  
 Das erforderliche `number`\-Argument ist ein numerischer Ausdruck, für den der Arkussinus benötigt wird.  
  
 Der Rückgabewert ist der im Bogenmaß angegebene Arkussinus des `number`\-Arguments.
