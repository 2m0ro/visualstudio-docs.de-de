# <a name="mathasin-function-javascript"></a>Math.asin-Funktion (JavaScript)
Gibt den Arkussinus einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche `number` Argument ist ein numerischer Ausdruck, der für den Arkussinus erforderlich ist.  
  
 Der zurückgegebene Wert besteht aus den Arkussinus der `number` -Arguments im Bogenmaß.
