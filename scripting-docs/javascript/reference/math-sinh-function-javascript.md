# <a name="mathsinh-function-javascript"></a>Math.sinh-Funktion (JavaScript)
Gibt den umgekehrten Hyperbelsinus einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Hinweise  
 Der Rückgabewert ist identisch mit `(exp(x) - exp(-x))/2`. Wenn x gleich `NaN` ist, lautet das Ergebnis `NaN`.
