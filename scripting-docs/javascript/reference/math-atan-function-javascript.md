# <a name="mathatan-function-javascript"></a>Math.atan-Funktion (JavaScript)
Gibt den Arkustangens einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche `number` Argument ist ein numerischer Ausdruck, der für den Arkustangens erforderlich ist.  
  
 Der zurückgegebene Wert besteht aus den Arkustangens der `number` -Arguments im Bogenmaß.
