# toString-Methode (WeakMap) (JavaScript)
Gibt eine Zeichenfolgendarstellung eines `WeakMap`\-Objekts zurück.  
  
## Syntax  
  
```javascript  
weakmapObj.toString()  
```  
  
#### Parameter  
 `weakmapObj`  
 Erforderlich.  Ein `WeakMap`\-Objekt.  
  
## Eigenschaftswert\/Rückgabewert  
 Die Zeichenfolgendarstellung des `WeakMap`.
