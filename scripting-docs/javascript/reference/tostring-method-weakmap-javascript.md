# <a name="tostring-method-weakmap-javascript"></a>toString-Methode (WeakMap) (JavaScript)
Gibt eine Zeichenfolgendarstellung eines `WeakMap`-Objekts zurück.  
  
## <a name="syntax"></a>Syntax  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>Parameter  
 `weakmapObj`  
 Erforderlich. Ein `WeakMap`-Objekt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Die Zeichenfolgendarstellung des `WeakMap`.
