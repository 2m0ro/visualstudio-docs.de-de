# Math.asinh-Funktion (JavaScript)
Gibt den umgekehrten Hyperbelsinus einer Zahl zurück.  
  
## Syntax  
  
```  
  
Math.asinh(x)   
```  
  
## Hinweise  
 Wenn `x` gleich `NaN` ist, lautet das Ergebnis `NaN`.
