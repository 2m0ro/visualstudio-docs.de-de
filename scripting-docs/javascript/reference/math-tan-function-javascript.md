# <a name="mathtan-function-javascript"></a>Math.tan-Funktion (JavaScript)
Gibt den Tangens einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche `radians` Argument ist ein numerischer Ausdruck, der einen im Bogenmaß angegebener Winkel enthält.  
  
 Der Rückgabewert ist der Tangens des Arguments numerischen `radians`.
