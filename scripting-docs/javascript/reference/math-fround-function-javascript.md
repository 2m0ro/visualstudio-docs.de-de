# Math.fround-Funktion (JavaScript)
Gibt das nächste Gleitkommaformat einer Zahl mit einfacher Genauigkeit zurück.  
  
## Syntax  
  
```  
  
Math.fround(  
number  
)   
```  
  
## Hinweise  
 Wenn `number` gleich `NaN` ist, lautet das Ergebnis `NaN`.
