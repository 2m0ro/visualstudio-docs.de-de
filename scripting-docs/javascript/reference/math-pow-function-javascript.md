# Math.pow-Funktion (JavaScript)
Gibt den um einen angegebenen Exponenten potenzierten Wert eines Basisausdrucks zurück.  
  
## Syntax  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## Parameter  
 *base*  
 Erforderlich.  Der Basiswert des Ausdrucks.  
  
 *exponent*  
 Erforderlich.  Der Exponentenwert des Ausdrucks.  
  
## Beispiel  
 Im folgenden Beispiel gibt ein numerischer, baseexponent gleicher Ausdruck 1000 zurück.  
  
```javascript  
Math.pow(10,3);  
```
