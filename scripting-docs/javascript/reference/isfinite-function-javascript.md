# <a name="isfinite-function-javascript"></a>isFinite-Funktion (JavaScript)
Bestimmt, ob eine angegebene Zahl endlich ist.  
  
## <a name="syntax"></a>Syntax  
  
```  
isFinite(number)   
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche `number` Argument ist jeden beliebigen numerischen Wert.  
  
 Die `isFinite` -Funktion gibt `true` Wenn `number` ist jeder Wert außer `NaN`, negativ unendlich oder plus unendlich. In diesen drei Fällen gibt es **"false"**.
