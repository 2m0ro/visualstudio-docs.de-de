# <a name="mathcbrt-function-javascript"></a>Math.cbrt-Funktion (JavaScript)
Gibt die Kubikwurzel einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>Hinweise  
 Wenn `number` gleich `NaN` ist, lautet das Ergebnis `NaN`.
