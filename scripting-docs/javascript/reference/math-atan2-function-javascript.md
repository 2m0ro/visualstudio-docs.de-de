# Math.atan2-Funktion (JavaScript)
Gibt den Winkel \(im Bogenmaß\) zwischen der X\-Achse und einem Punkt \(*y*,*x*\) zurück.  
  
## Syntax  
  
```  
  
Math.atan2(y, x)  
```  
  
## Parameter  
 *x*  
 Erforderlich.  Ein numerischer Ausdruck, der die kartesische x\-Koordinate darstellt.  
  
 *y*  
 Erforderlich.  Ein numerischer Ausdruck, der die kartesische y\-Koordinate darstellt.  
  
## Hinweise  
 Der Rückgabewert liegt zwischen *\-Pi* und *Pi*.  Er stellt den Winkel des angegebenen \(*y*,*x*\)\-Punkts im Bogenmaß dar.
