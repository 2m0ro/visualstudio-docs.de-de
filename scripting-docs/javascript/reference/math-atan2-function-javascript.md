# <a name="mathatan2-function-javascript"></a>Math.atan2-Funktion (JavaScript)
Gibt den Winkel (im Bogenmaß) von der x-Achse zu einem Zeitpunkt (*y*,*x*).  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Parameter  
 *x*  
 Erforderlich. Ein numerischer Ausdruck, der die kartesische X-Koordinate darstellt.  
  
 *y*  
 Erforderlich. Ein numerischer Ausdruck, der die kartesische y-Koordinate darstellt.  
  
## <a name="remarks"></a>Hinweise  
 Der Rückgabewert liegt zwischen *-Pi* und *Pi*. Gibt den Winkel des angegebenen (*y*,*x*) zeigen, im Bogenmaß.
