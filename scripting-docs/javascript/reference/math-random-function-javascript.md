# <a name="mathrandom-function-javascript"></a>Math.random-Funktion (JavaScript)
Gibt eine Pseudozufallszahl zwischen 0 und 1 zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Die generierte Pseudozufallszahl liegt zwischen 0 (inklusiv) und 1 (ausschließlich), d. h., die zurückgegebene Zahl kann 0 (null) sein, aber es wird immer kleiner als eins sein. Zufallszahlen-Generator wird automatisch ausgeführt, wenn [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] zuerst geladen wird.
