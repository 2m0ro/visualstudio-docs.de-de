# <a name="todatestring-method-date-javascript"></a>toDateString-Methode (Datum) (JavaScript)
Gibt ein Datum als Zeichenfolgenwert zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche *ObjDate* Verweis ist ein `Date` Objekt.  
  
 Die `toDateString` -Methode gibt einen Zeichenfolgenwert, der das Datum an, in der aktuellen Zeitzone in einem benutzerfreundlichen, leicht lesbaren Format.
