# valueOf-Methode (WeakMap) (JavaScript)
Gibt den einfachen Wert des angegebenen Objekts zurück.  
  
## Syntax  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### Parameter  
 `weakmapObj`  
 Erforderlich.  Ein `WeakMap`\-Objekt.  
  
## Eigenschaftswert\/Rückgabewert  
 Gibt die `WeakMap`\-Instanz zurück.
