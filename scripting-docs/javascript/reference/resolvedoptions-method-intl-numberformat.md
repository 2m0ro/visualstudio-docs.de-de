# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions-Methode (Intl.NumberFormat)
Gibt ein Objekt zurück, das die Eigenschaften und die Werte des `NumberFormat`-Formatierungsobjekts enthält.  
  
## <a name="syntax"></a>Syntax  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parameter  
 `numberFormatObj`  
 Das `NumberFormat`-Objekt, von dem die Informationen abgerufen werden.  
  
## <a name="remarks"></a>Hinweise  
 Die Eigenschaften des zurückgegebenen Objekts entsprechen den berechneten Eigenschaften des `NumberFormat`-Objekts.
