# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions-Methode (Intl.DateTimeFormat)
Gibt ein Objekt zurück, das die Eigenschaften und die Werte des Datums-/Formatierungsobjekts enthält.  
  
## <a name="syntax"></a>Syntax  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parameter  
 `dateTimeFormatObj`  
 Das `DateTimeFormat`-Objekt, von dem die Informationen abgerufen werden.  
  
## <a name="remarks"></a>Hinweise  
 Die Eigenschaften des zurückgegebenen Objekts entsprechen den berechneten Eigenschaften des `DateTimeFormat`-Objekts.
