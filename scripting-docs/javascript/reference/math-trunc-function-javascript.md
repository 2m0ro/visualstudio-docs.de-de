# Math.trunc-Funktion (JavaScript)
Gibt den ganzzahligen Teil einer Zahl ohne Nachkommastellen zurück.  
  
## Syntax  
  
```vb  
Math.trunc(number)  
```  
  
## Hinweise  
 Das erforderliche `number`\-Argument ist ein numerischer Ausdruck, für den die Ganzzahl ohne Dezimalanteil erforderlich ist.  
  
 Wenn `number` gleich `NaN` ist, wird `NaN` zurückgegeben.
