# F&#252;r die zuk&#252;nftige Verwendung reserviertes Wort (JavaScript)
In JavaScript dürfen für die künftige Verwendung reservierte Schlüsselwörter nicht als Bezeichner verwendet werden, obwohl sie in der aktuellen Version keine spezielle Bedeutung haben.  
  
 Eine Liste der reservierten Wörter in JavaScript finden Sie unter [Reservierte Wörter in JavaScript](../../javascript/reference/javascript-reserved-words.md).  
  
## Für die zukünftige Verwendung reservierte Wörter  
 Nachfolgend sind die für die zukünftige Verwendung reservierten Wörter aufgeführt:  
  
1.  class  
  
2.  enum  
  
3.  "export"  
  
4.  extends  
  
5.  import  
  
6.  super  
  
## Für die zukünftige Verwendung reservierte Wörter im Strict\-Modus  
 Für die zukünftige Verwendung im Strict\-Modus sind folgende Wörter reserviert.  Weitere Informationen zum Strict\-Modus finden Sie unter [Use Strict\-Direktive](../../javascript/reference/use-strict-directive.md).  
  
1.  implements  
  
2.  interface  
  
3.  package  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
