# <a name="javascript-future-reserved-words"></a>Für die zukünftige Verwendung reserviertes Wort (JavaScript)
In JavaScript dürfen für die künftige Verwendung reservierte Schlüsselwörter nicht als Bezeichner verwendet werden, obwohl sie in der aktuellen Version keine spezielle Bedeutung haben.  
  
 Eine Liste der reservierte Wörter in JavaScript, finden Sie unter [JavaScript reservierte Wörter](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Für die zukünftige Verwendung reservierte Wörter  
 Nachfolgend sind die für die zukünftige Verwendung reservierten Wörter aufgeführt:  
  
1.  Klasse  
  
2.  enum  
  
3.  "export"  
  
4.  extends  
  
5.  import  
  
6.  super  
  
## <a name="future-reserved-words-in-strict-mode"></a>Für die zukünftige Verwendung reservierte Wörter im Strict-Modus  
 Für die zukünftige Verwendung im Strict-Modus sind folgende Wörter reserviert. Weitere Informationen zum strict-Modus finden Sie unter [use strict-Direktive](../../javascript/reference/use-strict-directive.md).  
  
1.  implements  
  
2.  interface  
  
3.  package  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
