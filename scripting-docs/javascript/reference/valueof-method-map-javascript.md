# valueOf-Methode (Map) (JavaScript)
Gibt den einfachen Wert des angegebenen Objekts zurück.  
  
## Syntax  
  
```javascript  
mapObj.valueOf()  
```  
  
#### Parameter  
 `mapObj`  
 Erforderlich.  Ein `Map`\-Objekt.  
  
## Eigenschaftswert\/Rückgabewert  
 Gibt die Zuordnungsinstanz zurück.
