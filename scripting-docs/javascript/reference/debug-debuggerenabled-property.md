# Debug.debuggerEnabled-Eigenschaft
Bestimmt, ob das Debuggen für den Skriptkontext aktiviert ist.  Das Debuggen kann unabhängig davon, ob ein Debugger angefügt ist, aktiviert oder deaktiviert werden.  
  
## Syntax  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
