# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled-Eigenschaft
Bestimmt, ob Debuggen für Script-Kontext aktiviert ist. Debuggen kann aktiviert oder deaktiviert, und zwar unabhängig davon, ob ein Debugger angefügt ist.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
