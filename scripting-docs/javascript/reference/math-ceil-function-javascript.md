# Math.ceil-Funktion (JavaScript)
Gibt die kleinste ganze Zahl zurück, die größer oder gleich dem numerischen Argument ist.  
  
## Syntax  
  
```  
  
Math.ceil(number)   
```  
  
## Hinweise  
 Das erforderliche `number`\-Argument ist ein numerischer Ausdruck.  
  
 Der Rückgabewert ist ein ganzzahliger Wert gleich der kleinsten ganzen Zahl, die größer oder gleich dem numerischen Argument ist.
