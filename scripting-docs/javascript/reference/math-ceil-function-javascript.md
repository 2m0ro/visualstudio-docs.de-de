# <a name="mathceil-function-javascript"></a>Math.ceil-Funktion (JavaScript)
Gibt die kleinste ganze Zahl zurück, größer als oder gleich dem numerischen Argument.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Hinweise  
 Das erforderliche `number`-Argument ist ein numerischer Ausdruck.  
  
 Der Rückgabewert ist ein Ganzzahlwert, der die kleinste ganze Zahl größer als oder gleich dem numerischen Argument gleich.
