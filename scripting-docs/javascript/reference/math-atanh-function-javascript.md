# <a name="mathatanh-function-javascript"></a>Math.atanh-Funktion (JavaScript)
Gibt den umgekehrten Hyperbeltangens einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Hinweise  
 Wenn `x` gleich `NaN` ist, lautet das Ergebnis `NaN`.  
  
 **Gilt für**: [Math-Objekt](../../javascript/reference/math-object-javascript.md)
