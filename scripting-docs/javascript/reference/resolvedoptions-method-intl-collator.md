# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions-Methode (Intl.Collator)
Gibt ein Objekt zurück, das die Eigenschaften und die Werte des Collator-Objekts enthält.  
  
## <a name="syntax"></a>Syntax  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parameter  
 `collatorObj`  
 Das `Collator`-Objekt, von dem die Informationen abgerufen werden.  
  
## <a name="remarks"></a>Hinweise  
 Die Eigenschaften des zurückgegebenen Objekts entsprechen den berechneten Eigenschaften des `Collator`-Objekts.
