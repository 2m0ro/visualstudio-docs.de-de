# valueOf-Methode (Set) (JavaScript)
Gibt den einfachen Wert des angegebenen Objekts zurück.  
  
## Syntax  
  
```javascript  
setObj.valueOf()  
```  
  
#### Parameter  
 `setObj`  
 Erforderlich.  Ein `Set`\-Objekt.  
  
## Eigenschaftswert\/Rückgabewert  
 Gibt die festgelegte Setinstanz zurück.
