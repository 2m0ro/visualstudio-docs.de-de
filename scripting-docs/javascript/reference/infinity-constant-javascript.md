# <a name="infinity-constant-javascript"></a>Infinity-Konstante (JavaScript)
Eine Zahl ist, die größer als die größte Gleitkommazahl. Gibt einen Anfangswert von `Number.POSITIVE_INFINITY`. Minus unendlich (-unendlich) ist kleiner als der kleinste Gleitkommatyp Gleitkommazahl.  
  
## <a name="syntax"></a>Syntax  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Hinweise  
 Die `Infinity` konstant ist ein Mitglied der `Global` Objekt und zur Verfügung gestellt wird, wenn das Skriptmodul initialisiert wird.
