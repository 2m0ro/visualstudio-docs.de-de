# Infinity-Konstante (JavaScript)
Eine Zahl, die größer als die größte Gleitkommazahl ist.  Gibt einen Anfangswert von `Number.POSITIVE_INFINITY` zurück.  Negative Infinity \(\-unendlich\) ist kleiner als die kleinste Gleitkommazahl.  
  
## Syntax  
  
```  
Infinity   
```  
  
## Hinweise  
 Die Konstante `Infinity` ist ein Member des `Global`\-Objekts. Sie ist nach der Initialisierung des Skriptmoduls verfügbar.
