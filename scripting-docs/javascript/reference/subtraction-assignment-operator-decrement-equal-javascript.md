# Subtraktionszuweisungsoperator (-=) (JavaScript)
Subtrahiert den Wert eines Ausdrucks vom Wert einer Variablen und weist das Ergebnis der Variablen zu.  
  
## Syntax  
  
```  
  
result -= expression  
```  
  
## Parameter  
 *Ergebnis*  
 Beliebige numerische Variable.  
  
 *expression*  
 Beliebiger numerischer Ausdruck.  
  
## Hinweise  
 Die Verwendung des `-=`\-Operators entspricht genau folgendem Ablauf:  
  
```javascript

```
