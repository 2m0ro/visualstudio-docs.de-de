# <a name="subtraction-assignment-operator---javascript"></a>Subtraktionszuweisungsoperator (-=) (JavaScript)
Subtrahiert den Wert eines Ausdrucks vom Wert einer Variablen und weist das Ergebnis der Variablen zu.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>Parameter  
 *Ergebnis*  
 Beliebige numerische Variable.  
  
 *Ausdruck*  
 Ein beliebiger numerischer Ausdruck.  
  
## <a name="remarks"></a>Hinweise  
 Mithilfe der `-=` Operator ist identisch mit der folgenden Schritte:  
  
```JavaScript  
result = result - expression  
```
