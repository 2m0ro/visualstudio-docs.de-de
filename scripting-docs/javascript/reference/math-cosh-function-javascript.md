# Math.cosh-Funktion (JavaScript)
Gibt den Hyperbelkosinus einer Zahl zurück.  
  
## Syntax  
  
```  
  
Math.cosh(x)   
```  
  
## Hinweise  
 Der Rückgabewert ist identisch mit `(exp(x) + exp(-x))/2`.  Wenn `x` gleich `NaN` ist, lautet das Ergebnis `NaN`.
