# Math.log10-Funktion (JavaScript)
Gibt den Logarithmus einer Zahl zur Basis 10 zurück.  
  
## Syntax  
  
```  
Math.log10(x)   
```  
  
## Rückgabewert  
 Wenn `x` gleich `NaN` ist, gibt diese Funktion `NaN` zurück.  Wenn `x` kleiner als Null \(0\) ist, gibt diese Funktion `NaN` zurück.
