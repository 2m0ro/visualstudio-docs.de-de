# toString-Methode (Set) (JavaScript)
Gibt eine Zeichenfolgendarstellung eines Satzes zurück.  
  
## Syntax  
  
```javascript  
setObj.toString()  
```  
  
#### Parameter  
 `setObj`  
 Erforderlich.  Ein `Set`\-Objekt.  
  
## Eigenschaftswert\/Rückgabewert  
 Die Zeichenfolgendarstellung des Satzes.
