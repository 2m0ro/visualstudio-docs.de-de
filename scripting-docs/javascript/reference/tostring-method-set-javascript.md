# <a name="tostring-method-set-javascript"></a>toString-Methode (Set) (JavaScript)
Gibt eine Zeichenfolgendarstellung eines Satzes zurück.  
  
## <a name="syntax"></a>Syntax  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>Parameter  
 `setObj`  
 Erforderlich. Ein `Set`-Objekt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Die Zeichenfolgendarstellung des Satzes.
