# Math.tanh-Funktion (JavaScript)
Gibt den Hyperbeltangens einer Zahl zurück.  
  
## Syntax  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## Hinweise  
 Der Rückgabewert ist identisch mit `(exp(x) - exp(-x))/(exp(x) + exp(-x))`.  Wenn `x` gleich `NaN` ist, lautet das Ergebnis `NaN`.
