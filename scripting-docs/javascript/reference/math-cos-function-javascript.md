# <a name="mathcos-function-javascript"></a>Math.cos-Funktion (JavaScript)
Gibt den Kosinus einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche `radians` Argument ist ein numerischer Ausdruck, der einen im Bogenmaß angegebener Winkel enthält.  
  
 Der Rückgabewert ist der Kosinus des Arguments numerischen `radians`.
