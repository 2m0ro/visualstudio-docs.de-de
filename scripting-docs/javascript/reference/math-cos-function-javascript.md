# Math.cos-Funktion (JavaScript)
Gibt den Kosinus einer Zahl zurück.  
  
## Syntax  
  
```  
  
Math.cos(radians)   
```  
  
## Hinweise  
 Das erforderliche `radians`\-Argument ist ein numerischer Ausdruck, der einen im Bogenmaß gemessenen Winkel enthält.  
  
 Der Rückgabewert ist der Kosinus des numerischen Arguments `radians`.
