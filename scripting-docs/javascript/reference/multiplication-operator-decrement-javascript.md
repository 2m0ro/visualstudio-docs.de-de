# Multiplikationsoperator (*) (JavaScript)
Multipliziert den Wert zweier Ausdrücke.  
  
## Syntax  
  
```  
  
result = number1*number2  
```  
  
## Parameter  
 *Ergebnis*  
 Beliebige Variable.  
  
 `number1`  
 Ein beliebiger Ausdruck.  
  
 `number2`  
 Ein beliebiger Ausdruck.
