# <a name="void-operator-javascript"></a>void-Operator (JavaScript)
Verhindert, dass ein Ausdruck einen Wert zurückgibt.  
  
## <a name="syntax"></a>Syntax  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Hinweise  
 Die `expression` Argument ist ein beliebiger gültiger [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] Ausdruck.  
  
 Die `void` Operator wertet seine Ausdruck aus und gibt `undefined`. Es ist hilfreich in Situationen, in denen ein Ausdruck ausgewertet werden soll, jedoch nicht möchten, die Ergebnisse für den Rest des Skripts sichtbar.
