# <a name="mathexpm1-function-javascript"></a>Math.expm1-Funktion (JavaScript)
Gibt die potenzierte Konstante e (die Basis des natürlichen Logarithmus) minus 1 zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>Hinweise  
 Das erforderliche `number`-Argument ist ein numerischer Ausdruck, der die Potenz von e darstellt.  
  
 Der Rückgabewert ist eine Zahl. Die Konstante e ist die eulersche Zahl (entspricht ca. 2,71828), und `number` ist das bereitgestellte Argument.
