# Modulozuweisungsoperator (JavaScript)
Dividiert den Wert einer Variablen durch den Wert eines Ausdrucks und weist den Divisionsrest der Variablen zu.  
  
## Syntax  
  
```  
  
result %= expression  
```  
  
## Argumente  
 `result`  
 Beliebige Variable.  
  
 `expression`  
 Ein beliebiger numerischer Ausdruck.  
  
## Hinweise  
 Die Verwendung des `%=`\-Operators entspricht genau der folgenden Angabe:  
  
```  
result = result % expression  
```
