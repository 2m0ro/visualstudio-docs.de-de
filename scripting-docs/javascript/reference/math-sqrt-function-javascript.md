# Math.sqrt-Funktion (JavaScript)
Gibt die Quadratwurzel einer Zahl zurück.  
  
## Syntax  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## Hinweise  
 Das erforderliche `number`\-Argument ist ein numerischer Ausdruck.  
  
 Wenn `number` negativ ist, wird der Wert `NaN` zurückgegeben.
