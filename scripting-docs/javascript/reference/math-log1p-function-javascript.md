# <a name="mathlog1p-function-javascript"></a>Math.log1p-Funktion (JavaScript)
Gibt den natürlichen Logarithmus von 1 + einer Zahl zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn `x` gleich `NaN` ist, gibt diese Funktion `NaN` zurück. Wenn `x` kleiner als Null (0) ist, gibt diese Funktion `NaN` zurück.
