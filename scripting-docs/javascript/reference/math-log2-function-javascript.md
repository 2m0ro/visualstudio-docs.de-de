# <a name="mathlog2-function-javascript"></a>Math.log2-Funktion (JavaScript)
Gibt den Logarithmus einer Zahl zur Basis 2 zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
Math.log2(x)   
```  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn `x` gleich `NaN` ist, gibt diese Funktion `NaN` zurück. Wenn `x` kleiner als Null (0) ist, gibt diese Funktion `NaN` zurück.
