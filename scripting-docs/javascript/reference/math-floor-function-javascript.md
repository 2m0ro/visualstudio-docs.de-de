# <a name="mathfloor-function-javascript"></a>Math.floor-Funktion (JavaScript)
Gibt der größten ganze Zahl kleiner oder gleich dem numerischen Argument.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Hinweise  
 Die erforderliche *Anzahl* Argument ist ein numerischer Ausdruck.  
  
 Ein Ganzzahlwert, der die größte ganze Zahl gleich ist der Rückgabewert kleiner oder gleich dem numerischen Argument.
