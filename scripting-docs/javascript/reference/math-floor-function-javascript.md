# Math.floor-Funktion (JavaScript)
Gibt die größte ganze Zahl zurück, die kleiner oder gleich dem numerischen Argument ist.  
  
## Syntax  
  
```  
  
Math.floor(number)   
```  
  
## Hinweise  
 Das erforderliche *number*\-Argument ist ein numerischer Ausdruck.  
  
 Der Rückgabewert ist ein Wert mit ganzer Zahl gleich der größten ganzen Zahl, die kleiner oder gleich dem numerischen Argument ist.
