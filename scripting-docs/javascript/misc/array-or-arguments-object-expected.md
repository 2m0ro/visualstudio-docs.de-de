# Es wurde ein array- oder arguments-Objekt erwartet
Sie haben kein Array als Argument angegeben.  Dieser Fehler gilt nur für die **Function.prototype.apply**\-Methode.  Wenn es angegeben wird, muss das zweite Argument für diese Funktion entweder ein `Array`\-Objekt oder ein **Arguments**\-Objekt sein.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie entweder `Array`\-Objekt oder ein **Arguments**\-Objekt als zweites Argument an.
