# &quot;(&quot; erwartet (JavaScript)
Sie haben versucht, einen Ausdruck in einem Satz von Klammern einzuschließen, Sie haben jedoch die linke Klammer nicht einbezogen.  Einige Ausdrücke müssen in ein Klammernpaar eingeschlossen werden.  Beachten Sie die Verwendung von Klammern im folgenden Beispiel.  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie dem Auswertungsausdruck die linke Klammer hinzu.
