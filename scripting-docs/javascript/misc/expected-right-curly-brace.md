# &quot;}&quot; erwartet
Sie haben nicht die rechte geschweifte Klammer eingegeben, die das Ende des Funktionsrumpfes, der Schleife, des Codeblocks oder des Objektinitialisierers markiert.  Ein Beispiel für diesen Fehler wäre eine **for** \-Schleife, bei der nur die öffnende Klammer den Text der Schleife markiert.  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie die rechte geschweifte Klammer hinzu, die das Ende der Funktion, der Schleife, des Blocks oder des Objektinitialisierers markiert.
