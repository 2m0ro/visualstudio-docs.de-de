# Das Verwenden eines Schl&#252;sselworts f&#252;r einen Bezeichner ist ung&#252;ltig
1048: Das Verwenden eines Schlüsselworts für einen Bezeichner ist unzulässig  
  
 Sie können ein Schlüsselwort nicht als Variablennamen, als Funktion oder als anderen benutzerdefinierten Bezeichner verwenden.  
  
 Weitere Informationen über Schlüsselwörter finden Sie unter [Reservierte Wörter in JavaScript](../../javascript/reference/javascript-reserved-words.md) und [Für die zukünftige Verwendung reserviertes Wort \(JavaScript\)](../../javascript/reference/javascript-future-reserved-words.md).
