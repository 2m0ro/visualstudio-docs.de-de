# <a name="expected-39end39"></a>Erwartete &#39; @end&#39;
Sie haben versucht, einen bedingt kompilierten Code-Block zu erstellen, aber enthielt keinen der  **@end**  Anweisung am Ende. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]-Anweisungen können bedingt kompiliert werden, schließen Sie diese in eine  **@if/ @end**  Block.  
  
### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Hinzufügen der entsprechenden  **@end**  Anweisung.
