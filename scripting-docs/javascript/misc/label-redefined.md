# Die Bezeichnung wurde neu definiert
Sie haben eine neue Bezeichnung erstellt, ihr aber den Namen einer vorhandenen Bezeichnung gegeben.  Bezeichnungen können verwendet werden, um Codeblöcke zu markieren, sie müssen jedoch innerhalb eines gegebenen Bereichs eindeutig sein.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass alle Bezeichnungen, die Sie in Programmen verwenden, in ihrem jeweiligen Gültigkeitsbereich eindeutig sind.
