# &#39;/&#39; erwartet
Sie haben versucht, ein Literal für einen regulären Ausdruck erstellen, aber einen der Schrägstriche \(\/\) nicht angegeben.  So wie Zeichenfolgenliterale als in Anführungszeichen eingeschlossene Zeichen geschrieben werden, werden als Literale für reguläre Ausdrücke als in ein Paar Schrägstriche \(\/\) eingeschlossene Zeichen dargestellt.  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie einen Schrägstrich ein, um das Ende des regulären Ausdrucks zu markieren.
