# Zeichenfolge erwartet
Sie haben versucht, die **String.prototype.toString**\-Methode oder **String.prototype.valueOf**\-Methode für ein Objekt aufzurufen, das nicht vom Typ `String` ist.  Das Objekt, für das diese Art von Aufruf erfolgt, muss vom Typ `String` sein.  
  
### So beheben Sie diesen Fehler  
  
-   Rufen Sie für Objekte vom Typ `String` nur die **String.prototype.toString**\-Methode oder die **String.prototype.valueOf**\-Methode auf.
