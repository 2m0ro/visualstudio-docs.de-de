# <a name="expected-3939"></a>Erwartete &#39;] &#39;
Sie einen Verweis auf ein Arrayelement vorgenommen, sondern enthielt keinen die schließenden Klammer. Ein beliebiger Ausdruck, der auf ein Arrayelement verweist umfasst sowohl öffnende bzw. schließende Klammern.  
  
### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie der schließenden Klammer auf den Ausdruck, der auf das Arrayelement verweist.
