# &quot;]&quot; erwartet
Sie haben einen Verweis auf ein Arrayelement eingefügt, aber die rechte Klammer vergessen.  Jeder Ausdruck, der auf ein Arrayelement verweist, muss sowohl öffnende als auch schließende Klammer enthalten.  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie dem Ausdruck, der auf das Arrayelement verweist, die rechte Klammer hinzu.
