# Ung&#252;ltige Zuweisung (JavaScript)
Sie haben versucht, einem schreibgeschützten Bezeichner einen Wert zuzuweisen.  Einem schreibgeschützten Bezeichner kann kein Wert zugewiesen werden.  Beispielsweise vom Host definierte Objekte und externe COM\-Objekte.  
  
### So beheben Sie diesen Fehler  
  
-   Weisen Sie schreibgeschützten Bezeichnern keine Werte zu.
