# <a name="the-number-of-fractional-digits-is-out-of-range"></a>Die Anzahl der Dezimalstellen liegt außerhalb des gültigen Bereichs.
Sie haben versucht, ein ungültiges Argument an die Funktion übergeben **Number.prototype.toExponential**. Das Argument der Funktion **toExponential()** muss zwischen 0 und 20 (einschließlich) liegen.  
  
### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher das Argument für **toExponential()** nicht zu groß oder zu klein ist.
