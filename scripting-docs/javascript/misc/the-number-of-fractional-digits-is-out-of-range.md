# Die Anzahl von Dezimalstellen liegt au&#223;erhalb des definierten Bereichs
Sie haben versucht, ein ungültiges Argument an die **Number.prototype.toExponential**\-Funktion zu übergeben.  Das Argument für die Funktion **toExponential\(\)** muss zwischen 0 und 20 \(einschließlich\) liegen.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass das Argument für **toExponential** nicht zu groß oder zu klein ist.
