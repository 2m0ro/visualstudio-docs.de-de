# Zahl erwartet
Sie haben versucht, die **Number.prototype.toString**\-Methode oder die **Number.prototype.valueOf**\-Methode für ein Objekt aufzurufen, das einen anderen Typ als **Number** hat.  Das Objekt, für das diese Art von Aufruf erfolgt, muss vom Typ **Number** sein.  
  
### So beheben Sie diesen Fehler  
  
-   Rufen Sie die **Number.prototype.toString**\-Methode oder die **Number.prototype.valueOf**\-Methode nur für Objekte vom Typ **Number** auf.
