# &quot;=&quot; erwartet (JavaScript)
Sie haben versucht, eine Variable für die Verwendung mit bedingten Kompilierungsanweisungen zu erstellen, aber kein Gleichheitszeichen zwischen der Variablen und dem Wert, den Sie ihr zuweisen möchten, eingefügt.  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie ein Gleichheitszeichen hinzu.  Beispiel:  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
