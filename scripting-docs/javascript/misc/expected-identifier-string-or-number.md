# Es wurde ein Bezeichner, eine Zeichenfolge oder eine Zahl erwartet
Ein Objektliteral wurde mit einer falschen Literalsyntax deklariert.  Bei den Eigenschaften eines Objektliterals muss es sich um einen Bezeichner, eine Zeichenfolge oder eine Zahl handeln.  Ein Objektliteral \(auch als "Objektinitialisierung" bezeichnet\) besteht aus einer durch Trennzeichen getrennten Liste von Eigenschaft\-Wert\-Paaren, die alle in eckigen Klammern eingeschlossen sind.  Beispiel:  
  
```javascript  
var point = {x:1.2, y:-3.4};  
```  
  
### So beheben Sie diesen Fehler  
  
-   Es muss die richtige Literalsyntax verwendet werden.
