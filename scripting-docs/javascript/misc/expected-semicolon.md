# &#39;;&#39; erwartet
Sie haben versucht, mehr als eine Anweisung in einer Zeile zu speichern, sie jedoch nicht durch Semikolons getrennt.  Semikolons werden verwendet, um Anweisungen zu beenden.  Obwohl Sie mehrere Anweisungen in einer Zeile einfügen können, muss jede Anweisung von der darauf folgenden Anweisung mit einem Semikolon getrennt werden.  
  
### So beheben Sie diesen Fehler  
  
-   Markieren Sie das Ende jeder Anweisung mit einem Semikolon.  
  
-   Stellen Sie sicher, dass im Funktionsaufruf Klammern ordnungsgemäß verwendet werden.  
  
-   Stellen Sie sicher, dass die Semikolons innerhalb des Headers einer **for**\-Schleife nicht fehlen.
