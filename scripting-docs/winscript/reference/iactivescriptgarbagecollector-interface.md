# IActiveScriptGarbageCollector-Schnittstelle
Bietet eine Möglichkeit zum Anfangs\-GarbageCollection bereit.  Diese Schnittstelle sollte durch Active Script\-Module implementiert werden, die ihre Ressourcen bereinigen möchten.  
  
## Methoden  
  
|Methode|Description|  
|-------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Startet Garbage Collection.|
