# <a name="iremotedebugapplicationenumthreads"></a>IRemoteDebugApplication::EnumThreads
Listet alle Threads bekannt, dass der Anwendung zugeordnet sind.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
HRESULT EnumThreads(  
   IEnumRemoteDebugApplicationThreads**  pperdat  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pperdat`  
 [out] Ein Enumerator, der alle Threads bekannt, dass der Anwendung zugeordnet sind aufgelistet.  
  
## <a name="return-value"></a>Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück. Mögliches Werte (aber nicht die Einzigen) sind die in der folgenden Tabelle.  
  
|Wert|Beschreibung|  
|-----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## <a name="remarks"></a>Hinweise  
 Diese Methode listet alle Threads bekannt, dass der Anwendung zugeordnet sind. Neue Threads können jederzeit hinzugefügt werden.
