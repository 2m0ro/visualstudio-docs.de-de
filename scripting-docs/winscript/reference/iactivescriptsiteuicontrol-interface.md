|-------------|-----------------|  
|[IActiveScriptSiteUIControl::GetUIBehavior\-Methode](../../winscript/reference/iactivescriptsiteuicontrol-getuibehavior-method.md)|Ruft [SCRIPTUICHANDLING\-Enumeration](../../winscript/reference/scriptuichandling-enumeration.md) ab, das die Methode darstellt, die ein UI\-Steuerelement behandelt werden soll.|
