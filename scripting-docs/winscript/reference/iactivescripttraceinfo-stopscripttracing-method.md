# IActiveScriptTraceInfo::StopScriptTracing-Methode
Endskriptablaufverfolgung.  
  
## Syntax  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### Parameter  
 Diese Methode hat keine Parameter.  
  
## Rückgabewert  
 Die möglichen Rückgabewerte für diese Methode handelt, wie folgt:  
  
1.  S\_OK: Erfolg.  
  
2.  E\_NOTIMPL: Nicht implementiert.
