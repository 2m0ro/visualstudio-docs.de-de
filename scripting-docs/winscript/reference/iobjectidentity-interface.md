# IObjectIdentity-Schnittstelle

## Methoden  
  
|Methode|Description|  
|-------------|-----------------|  
|[IObjectIdentity::IsEqualObject](../../winscript/reference/iobjectidentity-isequalobject.md)|Bestimmt, ob ein Objekt gleich dem aktuellen Objekt ist.|
