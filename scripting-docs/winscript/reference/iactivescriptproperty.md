# IActiveScriptProperty
Die `IActiveScriptProperty`\-Schnittstelle wird verwendet, um Konfigurationseigenschaften abzurufen und festzulegen.  
  
## Methoden in Vtable\-Reihenfolge  
  
|Methode|Description|  
|-------------|-----------------|  
|[GetProperty](../../winscript/reference/iactivescriptproperty-getproperty.md)|Ruft die Eigenschaft ab, die vom Parameter angegeben wird.|  
|[SetProperty](../../winscript/reference/iactivescriptproperty-setproperty.md)|Legt die Eigenschaft fest, die vom Parameter angegeben wird.|
