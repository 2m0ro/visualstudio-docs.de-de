# <a name="iactivescriptproperty"></a>IActiveScriptProperty
Die `IActiveScriptProperty` Schnittstelle dient zum Abrufen und Festlegen von Konfigurationseigenschaften.  
  
## <a name="methods-in-vtable-order"></a>Methoden in Vtable-Reihenfolge  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[GetProperty](../../winscript/reference/iactivescriptproperty-getproperty.md)|Ruft die Eigenschaft, die durch den Parameter angegeben wird.|  
|[SetProperty](../../winscript/reference/iactivescriptproperty-setproperty.md)|Legt die Eigenschaft, die durch den Parameter angegeben wird.|
