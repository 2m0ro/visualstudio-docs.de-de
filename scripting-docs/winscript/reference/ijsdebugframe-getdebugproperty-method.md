# IJsDebugFrame::GetDebugProperty-Methode
Gibt einen Eigenschaftenbrowser für diesen Stapelrahmen zurück.  
  
## Syntax  
  
```  
HRESULT GetDebugProperty(  
   IJsDebugProperty **ppDebugProperty  
);  
```  
  
#### Parameter  
 `ppDebugProperty`  
 \[out\] Objekt, das den Eigenschaftenbrowser darstellt.
