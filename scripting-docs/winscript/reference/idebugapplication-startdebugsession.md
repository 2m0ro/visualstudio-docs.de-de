# <a name="idebugapplicationstartdebugsession"></a>IDebugApplication::StartDebugSession
Startet den Standard-Debugger integrierten Entwicklungsumgebung (IDE), und fügt eine Debugsitzung für diese Anwendung, wenn nicht bereits angefügt ist.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT StartDebugSession();  
```  
  
#### <a name="parameters"></a>Parameter  
 Diese Methode nimmt keine Parameter.  
  
## <a name="return-value"></a>Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück. Mögliches Werte (aber nicht die Einzigen) sind die in der folgenden Tabelle.  
  
|Wert|Beschreibung|  
|-----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## <a name="remarks"></a>Hinweise  
 Diese Methode wird verwendet, um Just-in-Time-Debuggen zu implementieren.
