# IJsDebugProcess::PerformAsyncBreak-Methode
Setzt das Skriptmodul in den Unterbrechungsmodus, der bei der nächsten Skriptanweisung eine Unterbrechung veranlasst.  
  
## Syntax  
  
```  
HRESULT PerformAsyncBreak(  
   DWORD threadId  
);  
```  
  
#### Parameter  
 `threadId`  
 \[in\] Die Thread\-ID.
