# IJsDebugProperty::GetPropertyInfo-Methode
Ruft Informationen für dieses Objekt ab.  
  
## Syntax  
  
```  
HRESULT GetPropertyInfo(  
   UINT nRadix,  
   JsDebugPropertyInfo *pPropertyInfo  
);  
```  
  
#### Parameter  
 `nRadix`  
 \[in\] Zu verwendende Basis.  
  
 `pPropertyInfo`  
 \[out\] Informationen über das Objekt.
