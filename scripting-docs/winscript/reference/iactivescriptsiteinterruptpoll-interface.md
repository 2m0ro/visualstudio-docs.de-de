# IActiveScriptSiteInterruptPoll-Schnittstelle
Die `IActiveScriptSiteInterruptPoll`\-Schnittstelle ermöglicht es einem Host, um anzugeben, dass ein Skript beendet werden soll.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IActiveScriptSiteInterruptPoll`\-Schnittstelle die folgende Methode verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|Ermöglicht einem Host, um anzugeben, dass ein Skript beendet werden soll.|
