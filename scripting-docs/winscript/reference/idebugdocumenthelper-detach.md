# IDebugDocumentHelper::Detach
Entfernt dieses Dokument von der Dokumentstruktur.  
  
## Syntax  
  
```  
HRESULT Detach();  
```  
  
#### Parameter  
 Diese Methode verwendet keine Parameter.  
  
## Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück.  Zu den möglichen Werten zählen, aber nicht zu, die in der folgenden Tabelle beschränkt.  
  
|Wert|Description|  
|----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## Hinweise  
 Diese Methode entfernt dieses Dokument von der Dokumentstruktur.
