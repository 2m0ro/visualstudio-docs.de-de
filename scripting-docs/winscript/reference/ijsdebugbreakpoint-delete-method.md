# IJsDebugBreakPoint::Delete-Methode
Löscht den Haltepunkt.  
  
## Syntax  
  
```  
HRESULT Delete(void);  
```  
  
## Rückgabewert  
  
## Hinweise  
 Gibt S\_FALSE zurück bei Aufruf an einem gelöschten Haltepunkt.
