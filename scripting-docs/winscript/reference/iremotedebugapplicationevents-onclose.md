# IRemoteDebugApplicationEvents::OnClose
Behandelt ein Anwendungsabschlussereignis.  
  
## Syntax  
  
```  
HRESULT OnClose();  
```  
  
#### Parameter  
 Diese Methode verwendet keine Parameter.  
  
## Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück.  Zu den möglichen Werten zählen, aber nicht zu, die in der folgenden Tabelle beschränkt.  
  
|Wert|Description|  
|----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## Hinweise  
 Diese Methode behandelt ein Anwendungsabschlussereignis.
