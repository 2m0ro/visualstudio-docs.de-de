# IActiveScriptAuthorProcedure-Schnittstelle
Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IScriptAuthorProcedure`\-Schnittstelle die folgende Methode verfügbar.  
  
## Methoden in Vtable\-Reihenfolge  
  
|Methode|Description|  
|-------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analysiert eine Kennzahlprozedur, fügt den Text der Kennzahlprozedur dem Skripterstellungsmodul hinzu und erstellt ein `IScriptEntry`\-Objekt, das der Kennzahlprozedur entspricht.|
