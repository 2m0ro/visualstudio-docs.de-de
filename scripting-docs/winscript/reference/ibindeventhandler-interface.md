# IBindEventHandler-Schnittstelle
Ermöglicht dem Aufrufer, um ein Objekt bereitstellen, das einen angegebenen Ereignishandler behandelt.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IBindEventHandler`\-Schnittstelle die folgenden Methoden verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Umschließt ein Ereignis zu einem Objekt.|
