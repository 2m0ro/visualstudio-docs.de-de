# IRemoteDebugApplicationThreadEx-Schnittstelle
Stellt einen Ausführungsthread innerhalb einer bestimmten Anwendung dar.  
  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IRemoteDebugApplicationThreadEx`\-Schnittstelle die folgenden Methoden verfügbar.  
  
## Methoden in Vtable\-Reihenfolge  
  
|Methode|Description|  
|-------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|Listet die globalen Ausdruckskontexte für alle Sprachen aufgeführt, die in diesen Thread ausgeführt werden.|
