# IDebugDocumentTextEvents::onDestroy
Gibt, an, dass das zugrunde liegende Dokument zerstört wurde und ist nicht mehr gültig.  
  
## Syntax  
  
```  
HRESULT onDestroy();  
```  
  
#### Parameter  
 Diese Methode verwendet keine Parameter.  
  
## Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück.  Zu den möglichen Werten zählen, aber nicht zu, die in der folgenden Tabelle beschränkt.  
  
|Wert|Description|  
|----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## Hinweise  
 Diese Methode gibt, an, dass das zugrunde liegende Dokument zerstört wurde und ist nicht mehr gültig.
