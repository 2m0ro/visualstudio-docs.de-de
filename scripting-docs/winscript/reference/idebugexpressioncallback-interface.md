# IDebugExpressionCallBack-Schnittstelle
Stellt die Statusereignisse bereit, die den Status einer `IDebugExpression` Auswertung verknüpft sind.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IDebugExpressionCallBack`\-Schnittstelle die folgenden Methoden verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|Gibt an, dass die Ausdrucksauswertung abgeschlossen ist.|
