# SCRIPT_E_PROPAGATE-Fehlercode
Ein Skriptfehler wird an den Aufrufer zurückgegeben, der sich in einem anderen Thread ist.  Der Host sollte den Fehlercode an den Aufrufer übergeben.  
  
## Syntax  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
