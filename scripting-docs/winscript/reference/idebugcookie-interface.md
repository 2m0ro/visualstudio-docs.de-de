# IDebugCookie-Schnittstelle
Ermöglicht die Debug\-, die mit der `IMachineDebugManagerCookie`\-Schnittstelle festgelegt werden Cookie.  Weitere Informationen finden Sie unter [IMachineDebugManagerCookie\-Schnittstelle](../../winscript/reference/imachinedebugmanagercookie-interface.md).  Diese Schnittstelle wird vom Prozessdebug\-Manager \(PDM\) implementiert und verwendet den von Skriptdebuggern.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IDebugCookie`\-Schnittstelle die folgenden Methoden verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[IDebugCookie::SetDebugCookie](../../winscript/reference/idebugcookie-setdebugcookie.md)|Legt das Debug\-Anwendungscookie fest.|
