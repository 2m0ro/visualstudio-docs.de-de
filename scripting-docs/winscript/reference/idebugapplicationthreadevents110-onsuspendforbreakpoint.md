# IDebugApplicationThreadEvents110::OnSuspendForBreakPoint
Bestimmt, ob der Thread ist vollständig für einen Haltepunkt unterbrochen und noch nicht normale erneut ausgeführt.  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110\-Schnittstelle](../../winscript/reference/idebugapplicationthreadevents110-interface.md) wird durch PDM v11.0 und höher implementiert.  Fundumgebung activdbg100.h.  
  
## Syntax  
  
```cpp  
HRESULT OnSuspendForBreakPoint( void );  
```  
  
#### Parameter  
 Diese Methode hat keine Parameter.
