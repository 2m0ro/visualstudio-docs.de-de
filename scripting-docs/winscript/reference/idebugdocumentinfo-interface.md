# IDebugDocumentInfo-Schnittstelle
Enthält Informationen zu einem Dokument bereit, das möglicherweise instanziiert wird.  
  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IDebugDocumentInfo`\-Schnittstelle die folgenden Methoden verfügbar.  
  
## Methoden in Vtable\-Reihenfolge  
  
|Methode|Description|  
|-------------|-----------------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|Gibt den angegebenen Dokumentnamen zurück.|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|Gibt eine CLSID zurück, das den Dokumenttyp identifiziert.|
