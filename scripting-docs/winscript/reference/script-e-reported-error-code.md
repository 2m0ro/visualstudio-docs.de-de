# SCRIPT_E_REPORTED-Fehlercode
Das Skriptmodul hat ein Ausnahme an den Host über [IActiveScriptSite::OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md) gemeldet.  Der Host kann diesen Fehler ignorieren.  
  
## Syntax  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
  
```
