# IDebugDocumentContext-Schnittstelle
Stellt eine abstrakte Darstellung eines Teils des Dokuments bereit, das gedebuggt wird.  Für diese Darstellung Textdokumente besteht aus einem Zeichenpositionsbereich.  
  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IDebugDocumentContext`\-Schnittstelle die folgenden Methoden verfügbar.  
  
## Methoden in Vtable\-Reihenfolge  
  
|Methode|Description|  
|-------------|-----------------|  
|[IDebugDocumentContext::GetDocument](../../winscript/reference/idebugdocumentcontext-getdocument.md)|Gibt das Dokument zurück, das diesen Kontext enthält.|  
|[IDebugDocumentContext::EnumCodeContexts](../../winscript/reference/idebugdocumentcontext-enumcodecontexts.md)|Listet die Codekontexte auf, die mit diesem Dokumentenkontext zugeordnet werden.|
