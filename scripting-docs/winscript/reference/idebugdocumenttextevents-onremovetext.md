# IDebugDocumentTextEvents::onRemoveText
Gibt an, dass Text aus dem Dokument entfernt wurde.  
  
## Syntax  
  
```  
HRESULT onRemoveText(  
   ULONG  cCharacterPosition,  
   ULONG  cNumToRemove  
);  
```  
  
#### Parameter  
 `cCharacterPosition`  
 \[in\] Die Zeichenposition des ersten Zeichens entfernt.  
  
 `cNumToRemove`  
 \[in\] Die Anzahl von Zeichen entfernt.  
  
## Rückgabewert  
 Die Methode gibt ein `HRESULT` zurück.  Zu den möglichen Werten zählen, aber nicht zu, die in der folgenden Tabelle beschränkt.  
  
|Wert|Description|  
|----------|-----------------|  
|`S_OK`|Die Methode war erfolgreich.|  
  
## Hinweise  
 Diese Methode gibt an, dass Text aus dem Dokument entfernt wurde.
