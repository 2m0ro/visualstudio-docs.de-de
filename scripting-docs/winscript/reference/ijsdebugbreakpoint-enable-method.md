# IJsDebugBreakPoint::Enable-Methode
Aktiviert den Haltepunkt.  
  
## Syntax  
  
```  
HRESULT Enable(void);  
```  
  
## Rückgabewert  
  
## Hinweise  
 Gibt E\_UNEXPECTED zurück, wenn der Aufruf an einem gelöschten Haltepunkt erfolgt.  
  
## Anforderungen  
 **Header:**  jscript9diag.h
