# IActiveScriptProfilerCallback3-Schnittstelle
Stellt eine Methode bereit, um den Profiler zu benachrichtigen, dass die profilierten Ausführungen im Kontext eines Internet\-Workers vorkommen.  
  
## Syntax  
  
```  
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## Methoden  
  
|Methode|Description|  
|-------------|-----------------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId\-Methode](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|Benachrichtigt den Profiler über die Worker ID, um für diese Profilerstellungssitzung zu verwenden.|
