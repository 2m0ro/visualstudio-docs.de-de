# IEnumDebugStackFrames-Schnittstelle
Listet die Stapelrahmen entsprechend einem Thread auf.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IEnumDebugStackFrames`\-Schnittstelle die folgenden Methoden verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[IEnumDebugStackFrames::Next](../../winscript/reference/ienumdebugstackframes-next.md)|Ruft eine angegebene Anzahl von Segmenten in der Enumerationssequenz ab.|  
|[IEnumDebugStackFrames::Skip](../../winscript/reference/ienumdebugstackframes-skip.md)|Überspringt eine angegebene Anzahl von Segmenten in einer Enumerationssequenz.|  
|[IEnumDebugStackFrames::Reset](../../winscript/reference/ienumdebugstackframes-reset.md)|Setzt eine Enumerationsfolge auf den Anfang zurück.|  
|[IEnumDebugStackFrames::Clone](../../winscript/reference/ienumdebugstackframes-clone.md)|Erstellt einen Enumerator, der den gleichen Zustand wie der aktuelle Enumerator enthält.|
