# IVariantChangeType-Schnittstelle
Ermöglicht dem Aufrufer, um einen VARIANTEN Wert von einem Typ in einen anderen zu konvertieren.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `IVariantChangeType`\-Schnittstelle die folgenden Methoden verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Verwendet einen varianten Wert und erstellt eine neue Variante mit einem angegebenen Typ.|
