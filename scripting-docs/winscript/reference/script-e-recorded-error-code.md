# SCRIPT_E_RECORDED-Fehlercode
Ein Fehler ist zwischen das Skriptmodul und Host übergeben wurde.  Die Hostanforderungen, den Fehlercode an den Aufrufer übergeben.  
  
## Syntax  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
