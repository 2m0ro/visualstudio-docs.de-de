# ICanHandleException-Schnittstelle
Ermöglicht dem Aufrufer eines Skriptmoduls zurück, um anzugeben, welche Ausnahmen der Aufrufer behandelt.  
  
## Methoden  
 Zusätzlich zu den von `IUnknown` geerbten Methoden macht die `ICanHandleException`\-Schnittstelle die folgenden Methoden verfügbar.  
  
|Methode|Description|  
|-------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Bestimmt, ob der Aufrufer des Skriptmoduls eine bestimmte Ausnahme behandeln kann.|
