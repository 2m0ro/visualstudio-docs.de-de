# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>Kaufen von Visual Studio-Cloudabonnements – Kündigen von Abonnements

Um ein Visual Studio-Cloudabonnement zu kündigen, melden Sie sich beim [Abonnementverwaltungsportal](https://manage.visualstudio.com/) von Visual Studio mit einem Konto an, das Zugriff auf das Azure-Abonnement hat, das zum ursprünglichen Kauf verwendet wurde.
1.  Überprüfen Sie in der oberen linken Ecke, dass Sie das richtige Azure-Abonnement ausgewählt haben.
2.  Wählen Sie im linken Navigationsbereich **Reduzieren** unter dem Abonnement, das Sie kündigen möchten.
3.  Es wird ein Fenster angezeigt, in dem Sie die Anzahl der Ihrem Konto zugeordneten Abonnements einstellen können.
4.  Verringern Sie die Anzahl von Abonnements auf die gewünschte Zahl – 0 (null) zum Kündigen – und klicken Sie auf **Fortfahren**.
5.  Überprüfen Sie Ihren Auftrag, klicken Sie auf **Bestätigen** – und das war‘s!
## <a name="faq"></a>FAQ:
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>F: Kann ich nach der Kündigung immer noch ein Abonnement verwenden?
A: Ja, Sie können während der verbleibenden Gültigkeitsdauer des Abonnements weiterhin auf Ihre Vorteile zugreifen.
