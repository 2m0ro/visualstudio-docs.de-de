# <a name="visual-studio-subscriptions"></a>Visual Studio-Abonnements
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>Entwicklertools, Clouddienste, Software, Support und Schulungen

Visual Studio-Abonnements ist die umfassendste Sammlung von Ressourcen zum Erstellen, Bereitstellen und Verwalten großartiger Anwendungen auf Ihren bevorzugten Plattformen und Geräten, einschließlich Windows, iOS, Android und Linux.
