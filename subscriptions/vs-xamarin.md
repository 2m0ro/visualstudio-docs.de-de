# <a name="xamarin-university-training-benefit-in-visual-studio-subscriptions"></a>Xamarin University-Trainingsvorteil in Visual Studio-Abonnements

Erstellen Sie anhand von aufgezeichneten Kursen und Gastvorlesungen iOS- und Android-Apps in C#.  
-  Das Xamarin University Mobile Training für Visual Studio Dev Essentials besteht aus zwölf Kursen und drei Gastvorlesungen. 
-  Das Xamarin University Mobile Training für Visual Studio Professional- und Visual Studio Enterprise-Abonnements enthält die gleichen Kurse und Vorträge wie das Angebot für Visual Studio Dev Essentials PLUS 18 weitere Kurse und sechs weitere Vorlesungen.

## <a name="activation-steps"></a>Aktivierungsschritte
1.  Melden Sie sich bei [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) an. 
2.  Suchen Sie im Abschnitt „Professionelle Entwicklung“ die Kachel „Xamarin University Training“, und klicken Sie im unteren Bereich der Kachel mit den Vorteilen auf den Link **Code abrufen**.   Sie erhalten eine Benachrichtigung, dass der Gutscheincode erfolgreich abgerufen wurde.  Klicken Sie zum Fortfahren auf **Activate** (Aktivieren).

    ![Kachel „Xamarin University-Trainingsvorteil“](_img\vs-xamarin\vs-xamarin-tile.png)

3.  Vervollständigen Sie auf der Registrierungsseite Ihre Kontaktdaten, klicken Sie auf das Kontrollkästchen, um die Nutzungsbedingungen und Datenschutzbestimmungen zu akzeptieren, und klicken Sie auf **Activate benefit** (Vorteil aktivieren). 

    ![Xamarin University-Trainingsvorteil – Registrierung](_img\vs-xamarin\vs-xamarin-registration-resized.png)
4.  Das ist auch schon alles.  Nun, da Sie Ihren Xamarin University-Trainingsvorteil aktiviert haben, werden Sie auf die Seite „Xamarin University-Schulung für mobile Technologien für Microsoft Visual Studio“ weitergeleitet und können Ihre Weiterbildung beginnen.  

    ![Xamarin University-Trainingsvorteil – Homepage](_img\vs-xamarin\vs-xamarin-home-resized.png)

## <a name="frequently-asked-questions"></a>Häufig gestellte Fragen
### <a name="q--what-topics-are-covered-in-the-classes-and-lectures"></a>F: Welche Themen werden in den Kursen und Vorlesungen behandelt?
A: Eine vollständige Liste der Kurse und Vorlesungen für beide Angebote finden Sie unter https://www.xamarin.com/university/mobile-training. 

### <a name="q-im-already-a-xamarin-university-subscriber-does-this-affect-my-xamarin-university-mobile-training-benefit"></a>Q: Ich habe bereits ein Xamarin University-Abonnement. Wirkt sich das auf meinen Xamarin University Mobile Training-Vorteil aus?
A: Nein. Vorhandene Xamarin University-Abonnenten haben vollständigen Zugriff auf ihr Xamarin University-Jahresabonnement mit allen Livekursen, Materialien und Gastvorlesungen. Melden Sie sich einfach bei „university.xamarin.com“ an.  
Kurse, die als Vorteil für das Visual Studio Dev Essentials-Programm und die Visual Studio-Abonnements angeboten werden, sind unabhängig von Ihrem Xamarin University-Abonnement. Der Zugriff erfolgt über Ihr Visual Studio Dev Essentials- oder Visual Studio-Portal, nicht über „university.xamarin.com“.  Die Xamarin University Mobile Training-Kursaufzeichnungen, die im Visual Studio Dev Essentials-Programm und den Visual Studio-Abonnements enthalten sind, sind ebenfalls in Ihrem Xamarin University-Jahresabonnement enthalten, bleiben aber verfügbar, solange die Programmmitgliedschaft oder das Visual Studio-Abonnement gültig sind. Sie können unabhängig vom Status Ihres Xamarin University-Abonnements weiterhin über Ihr Visual Studio Dev Essentials- oder Visual Studio-Portal auf diese Aufzeichnungen zugreifen.

### <a name="q--can-i-become-a-certified-xamarin-developer-with-the-xamarin-university-included-in-visual-studio-dev-essentials-what-about-through-visual-studio-subscriptions"></a>F: Kann ich mit den Kursen der Xamarin University, die in Studio Dev Essentials enthalten sind, zertifizierter Xamarin-Entwickler werden? Oder über Visual Studio-Abonnements?
A: Nein. Die Kurse und Materialien, die für Visual Studio Dev Essentials- und Visual Studio/MSDN-Abonnements in der Xamarin University angeboten werden, berechtigen nicht zur Zertifizierung. Materialien, Kurse und Aufzeichnungen, die für eine Xamarin-Zertifizierung erforderlich sind, sind nur im vollständigen Xamarin University-Abonnement enthalten.
