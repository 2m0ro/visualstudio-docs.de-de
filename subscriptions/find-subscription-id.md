# <a name="locating-your-subscription-id-for-visual-studio-subscriptions"></a>Suchen der Abonnement-ID für Visual Studio-Abonnements

## <a name="what-is-a-subscription-id"></a>Was ist eine Abonnement-ID?
Eine Abonnement-ID ist eine 32-stellige GUID, der dem Abonnement oder Programm eines Abonnenten zugeordnet ist.  In einigen Fällen kann das Supportteam einen Abonnenten dazu auffordern, seine Abonnenten-ID anzugeben, um mögliche Probleme mit einem bestimmten Abonnement eingrenzen zu können. 

## <a name="how-to-find-your-subscription-id"></a>Ermitteln der Abonnenten-ID
1. Wechseln Sie zu [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs).
2. Melden Sie sich bei Ihrem Abonnement an.  (Unterstützung zur [Anmeldung](/visualstudio/subscriptions/signing-in).)
3. Wenn Sie sich angemeldet haben, sehen Sie eine Liste der Abonnements, die der von Ihnen zur Anmeldung verwendeten E-Mail-Adresse zugeordnet sind.  Folgendes wird angezeigt:
    - Abonnement-/Programmtyp
    - Abonnement-ID 
    - Typ (Einzelhandel, NFR usw.)
    - Gültig bis-Datum  (Ablauf)
    - Links zu Verlängerungen, Supportinformationen und bei Visual Studio Dev Essentials zur Beendigung des Programms
