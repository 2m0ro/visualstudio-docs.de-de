# <a name="buying-visual-studio-cloud-subscriptions---buying-additional-subscriptions"></a>Kaufen von Visual Studio-Cloudabonnements – Kaufen zusätzlicher Abonnements

Um zusätzliche Visual Studio-Cloudabonnements zu kaufen, melden Sie sich beim [Abonnementverwaltungsportal](https://manage.visualstudio.com/) von Visual Studio mit einem Konto an, das Zugriff auf das Azure-Abonnement hat, das zum ursprünglichen Kauf verwendet wurde.

1.  Überprüfen Sie in der oberen linken Ecke, dass Sie das richtige Azure-Abonnement ausgewählt haben.
2.  Wählen Sie im linken Navigationsbereich **Mehr kaufen** unter dem Abonnement, das Sie kaufen möchten.
3.  Erhöhen Sie die Anzahl von Abonnements auf die gewünschte Zahl, und klicken Sie auf **Fortfahren**.
4.  Überprüfen Sie Ihre Bestellung, und klicken Sie auf **Bestätigen**.
5.  Bei der Rückkehr zum Abonnementverwaltungsportal sollten Sie einen Anstieg der Anzahl verfügbarer Abonnements feststellen.

Weitere Informationen zum Zuweisen und Verwalten von Abonnements finden Sie in unserer [Dokumentation](/visualstudio/subscriptions/).

## <a name="faq"></a>FAQ:
### <a name="q--how-long-do-i-have-to-wait-before-i-can-assign-subscriptions"></a>F: Wie lange muss ich warten, bis ich Abonnements zuweisen kann?
A: Ihre Abonnements stehen sofort zur Zuweisung im Abonnementverwaltungsportal zur Verfügung.
