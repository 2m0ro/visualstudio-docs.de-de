# <a name="buying-visual-studio-cloud-subscriptions---overview"></a>Kaufen von Visual Studio-Cloudabonnements – Übersicht

Mit unseren neuen Cloudabonnements können Sie Visual Studio, Visual Studio Team Services und die Abonnentenvorteile, die Sie benötigen, ohne langfristigen Vertrag mieten. Die Abrechnung erfolgt über Ihr Microsoft Azure-Abonnement, wobei die Zahlung per Kreditkarte unterstützt wird, Käufe erfolgen über den Enterprise Agreement-Vertrag* und [Cloudlösungsanbieter](https://www.microsoft.com/solution-providers/search)-Partner. Sie haben noch kein Azure-Abonnement? Kein Problem. Die Einrichtung ist kostenlos und während Ihres ersten Kaufs möglich.

Vergleichen Sie die [Vorteile und Preise der Cloudabonnements](https://www.visualstudio.com/vs/pricing/).

## <a name="before-you-start"></a>Vor der Installation
Befolgen Sie vor dem Kauf diese Schritte, damit eine erfolgreiche Transaktion möglich ist. • Zur Abrechnung Ihrer Käufe benötigen Sie ein [Azure-Abonnement](https://azure.microsoft.com/pricing/purchase-options/).  Sie können sich vor dem Kauf oder während des ersten Kaufs in Visual Studio Marketplace [anmelden](https://portal.azure.com/).  

•   Falls Sie für Ihre Organisation Käufe tätigen und in dieser Identitäten mithilfe von Azure Active Directory (Azure AD) verwaltet werden, müssen Sie ein [persönliches Microsoft-Konto](https://www.microsoft.com/account) oder ein [Geschäfts-, Schul- oder Unikonto](https://azure.microsoft.com/en-us/documentation/articles/sign-up-organization/) anlegen, um sich für ein Azure-Abonnement zu registrieren und Käufe im Visual Studio Marketplace tätigen zu können.
