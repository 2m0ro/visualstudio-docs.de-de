# <a name="legacy-designer-for-windows-workflow-foundation-ui-help"></a>Benutzeroberflächenhilfe von Visual Studio Designer für Windows Workflow Foundation (Vorgängerversion)

Dieser Abschnitt enthält die kontextbezogenen Hilfethemen für die älteren Windows-Workflow-Designer in Visual Studio 2010. Verwenden Sie den Designer den Vorgängerversion, wenn Sie .NET Framework, Version 3.5 oder die WinFX abzielen möchten.

Sie können diese Themen über die Benutzeroberfläche von Visual Studio zugreifen, durch Drücken von **F1** Wenn ein legacy-Workflow-Designer-Dialogfeld geöffnet ist.
