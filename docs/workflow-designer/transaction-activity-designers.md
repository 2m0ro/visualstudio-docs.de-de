# <a name="transaction-activity-designers"></a>Transaktionsaktivitätsdesigner

Transaktionsaktivitätsdesigner werden verwendet, um Transaktionsaktivitäten, z. B. das Kompensieren oder Bestätigen, zu erstellen und zu konfigurieren. Die Themen in diesem Abschnitt enthalten und in der folgenden Tabelle aufgeführten enthalten Anleitungen zum Workflow-Designer-Aktivität und den Vorlagen-Designer verwenden.
