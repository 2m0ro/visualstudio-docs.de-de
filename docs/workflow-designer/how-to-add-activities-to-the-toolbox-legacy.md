# Vorgehensweise: Hinzuf&#252;gen von Aktivit&#228;ten zur Toolbox (Vorg&#228;ngerversion)
Beim Erstellen einer Workflowprojektmappe mit der Vorgängerversion von [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)], die auf [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] oder [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)] abzielt, können benutzerdefinierte Aktivitäten für einfachen Zugriff dem Workflowprojekt und deren Designern hinzugefügt werden, die sich in der **Toolbox** befinden.Sie können auch Aktivitäten direkt aus einer Dynamic Link Library \(DLL\) zur **Toolbox** hinzufügen.  
  
### So fügen Sie eine Aktivität aus einer DLL zur Toolbox hinzu  
  
1.  Klicken Sie mit der rechten Maustaste unter **Windows Workflow** auf die Toolboxfensteroberfläche, und klicken Sie dann auf **Elemente auswählen**.  
  
2.  Klicken Sie im Dialogfeld **Toolboxelemente auswählen** auf die Registerkarte **System.Activities\-Komponenten**, und klicken Sie dann in der unteren rechten Ecke des Fensters auf **Durchsuchen**.  
  
3.  Wählen Sie aus dem Dateiverzeichnis die DLL aus, die die Implementierung der benutzerdefinierten Aktivität enthält, die der **Toolbox** hinzugefügt werden soll, und klicken Sie dann auf **Öffnen**.  
  
4.  Klicken Sie auf **OK**, um das Hinzufügen der Aktivität zur Toolbox abzuschließen.
