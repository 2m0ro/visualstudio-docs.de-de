# Flussdiagramm-Aktivit&#228;tsdesigner
[!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] umfasst eine Reihe von vom System bereitgestellten Aktivitäten, die Sie zum Erstellen von Workflows verwenden können.Dieser Abschnitt enthält die vom System bereitgestellten Aktivitäten, die zum Verwalten von Flussdiagrammen innerhalb eines Workflows dienen.In den folgenden Themen werden diese Aktivitäten beschrieben und Anleitungen zu ihrer Verwendung bereitgestellt.  
  
## In diesem Abschnitt  
 [Flussdiagramm](../workflow-designer/flowchart-activity-designer.md)  
 **Dient zum Erstellen von Workflows, die komplexe Flusssteuerungen definieren und verwalten.**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md)  
 Stellt eine Verzweigung für den Steuerungsverlauf in eine von zwei Alternativen bereit, deren Auswahl darauf basiert, ob eine angegebene Bedingung erfüllt wird.  
  
 [FlowSwitch\<T\>](../workflow-designer/flowswitch-t-activity-designer.md)  
 Stellt eine Verzweigung für den Steuerungsverlauf bereit, die auf der Übereinstimmung mit Kriterien basiert, wenn mehr als zwei alternative Verzweigungen erforderlich sind.
