# <a name="using-themes-in-workflows-legacy"></a>Verwenden von Designs in Workflows (Vorgängerversion)

Dieses Thema beschreibt, wie Designs in Windows Workflow Foundation (WF) bei Verwendung von älteren Windows Workflow-Designer mit der Zielversion .NET Framework, Version 3.5 oder die WinFX.

Sie können das Aussehen Ihrer Workflows mit dem legacy-Designer anpassen, indem Sie mithilfe von Windows Workflow Foundation-Designs. Ein Design definiert die Hintergrund- und Vordergrundfarben, Formate, Symbole und andere visuelle Elemente eines Workflows. Sie können Designs für die Wiederverwendung durch andere Workflows speichern.

Sie erstellen und bearbeiten Designs mit dem [Dialogfeld "Designkonfiguration" (Vorgängerversion)](../workflow-designer/theme-configuration-dialog-box-legacy.md). Wählen Sie zum Öffnen dieses Dialogfelds **neues Design erstellen** auf die **Workflow** Menüs oder mit der rechten Maustaste des workflowentwurfs auf Entwurfsoberfläche, und wählen **neues Design erstellen**.

Auszuwählender standardmäßige und benutzerdefinierte Designs gespeichert, mit der [Designs, Workflow-Designer, Optionen (Dialogfeld) (Legacy)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md). Um dieses Dialogfeld zu öffnen, klicken Sie auf **benutzerdefiniertes Design auswählen** auf die **Workflow** Menü (oder mit der rechten Maustaste in die Entwurfsoberfläche des Workflows, und klicken Sie auf **benutzerdefiniertes Design auswählen**) und wählen Sie dann die **Workflow-Designer** Option. Wählen Sie das Design, Sie möchten, und klicken Sie auf **OK**.
