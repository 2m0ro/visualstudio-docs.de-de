# <a name="integration-of-xml-literals-with-xml-schema-explorer"></a>Integration von XML-Literalen in XML-Schema-Explorer

Visual Basic unterstützt XML-Literale, was bedeutet, dass Sie XML-Fragmente direkt in Visual Basic-Code integrieren können. Weitere Informationen finden Sie unter [Übersicht über XML-Literale](http://go.microsoft.com/fwlink/?LinkId=140325).

## <a name="how-to"></a>Gewusst wie

Wenn eine XSD-Datei in Visual Basic-Projekt ein XML-Literal enthält, können Sie anzeigen, dass das XML-Schemaset in der **XML-Schema-Explorer**. Um das Schemaset eines XML-Literals zugeordneten anzuzeigen, mit der rechten Maustaste auf einen XML-Knoten in ein XML-Literal oder einem XML-Namespaceimport, und wählen Sie **im Schema-Explorer anzeigen**.

![XML-Literale in Visual Basic; XML-Schema-Explorer](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer1.gif)

Daraufhin wird die **XML-Schema-Explorer** parallel mit Visual Basic-Datei.

![XML-Literale in Visual Basic; XML-Schema-Explorer](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer2.gif)
