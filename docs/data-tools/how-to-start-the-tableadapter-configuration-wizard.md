# Gewusst wie: Starten des TableAdapter-Konfigurations-Assistenten
Der **TableAdapter\-Konfigurations\-Assistent** erstellt und bearbeitet TableAdapters in stark typisierten Datasets.  Der Assistent erstellt TableAdapters auf der Grundlage von dort eingegebenen SQL\-Anweisungen oder gespeicherten Prozeduren in der Datenbank.  Ausgehend von SQL\-Anweisungen, die Sie im Assistenten eingeben, kann der Assistent auch neue gespeicherte Prozeduren in der Datenbank erstellen.  
  
### Den TableAdapter\-Konfigurations\-Assistent zum Erstellen eines neuen TableAdapters starten  
  
1.  Öffnen Sie das Dataset im **DataSet\-Designer**.  Weitere Informationen finden Sie unter [Gewusst wie: Öffnen eines Datasets im DataSet\-Designer](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md).  
  
    > [!NOTE]
    >  Wenn Sie noch kein Dataset in Ihrem Projekt haben, finden Sie weitere Informationen unter [Gewusst wie: Erstellen eines typisierten Datasets](../data-tools/create-and-configure-datasets-in-visual-studio.md).  
  
2.  Wenn Sie einen neuen TableAdapter erstellen, ziehen Sie einen **TableAdapter** aus der Registerkarte **DataSet** des **Werkzeugkastens** auf den **DataSet\-Designer**.  
  
3.  Wählen Sie auf der Seite **Datenverbindung wählen** eine Datenverbindung aus der Liste der aktuell verfügbaren Verbindungen oder **Neue Verbindung**, um eine neue zu erstellen.  
  
### So starten Sie den TableAdapter\-Konfigurations\-Assistent zum Bearbeiten eines vorhandenen TableAdapters  
  
1.  Öffnen Sie das Dataset im **DataSet\-Designer**.  Weitere Informationen finden Sie unter [Gewusst wie: Öffnen eines Datasets im DataSet\-Designer](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md).  
  
2.  Klicken Sie mit der rechten Maustaste auf den TableAdapter im **DataSet\-Designer** und wählen Sie **Konfigurieren**.  Der Assistent öffnet die Seite **SQL\-Anweisungen generieren** oder **Befehle an vorhandene gespeicherte Prozeduren binden**, je nachdem, wie der TableAdapter ursprünglich konfiguriert wurde.  
  
3.  Durchlaufen Sie den Assistenten.
