# Gewusst wie: Erstellen eines typisierten Datasets
Ein typisiertes <xref:System.Data.DataSet> wird mit dem **Assistenten zum Konfigurieren von Datenquellen** oder dem [Erstellen und Bearbeiten von typisierten Datasets](../data-tools/creating-and-editing-typed-datasets.md) erstellt.  
  
> [!NOTE]
>  Informationen zum programmgesteuerten Erstellen von Datasets finden Sie unter [Erstellen eines DataSet](../Topic/Creating%20a%20DataSet.md).  
  
## Erstellen von typisierten Datasets mit dem Assistenten zum Konfigurieren von Datenquellen oder mit dem DataSet\-Designer  
  
#### So erstellen Sie ein Dataset mit dem Assistenten zum Konfigurieren von Datenquellen  
  
1.  Klicken Sie im Menü **Daten** auf **Neue Datenquelle hinzufügen**. Daraufhin wird der **Assistent zum Konfigurieren von Datenquellen** gestartet.  
  
2.  Wählen Sie auf der Seite **Datenquellentyp auswählen** die Option **Datenbank** aus.  
  
3.  Beenden Sie den Assistenten, und dem Projekt wird ein typisiertes Dataset hinzugefügt.  Weitere Informationen finden Sie unter [Assistent zum Konfigurieren von Datenquellen](../data-tools/media/data-source-configuration-wizard.png).  
  
#### So erstellen Sie ein Dataset mit dem DataSet\-Designer  
  
1.  Klicken Sie im Menü **Projekt** auf **Neues Element hinzufügen**.  
  
2.  Wählen Sie im Dialogfeld **Neues Element hinzufügen** die Option **DataSet** aus.  
  
3.  Geben Sie einen Namen für das Dataset ein.  
  
4.  Klicken Sie auf **Hinzufügen**.  
  
     Das Dataset wird dem Projekt hinzugefügt und im **DataSet\-Designer** geöffnet.  
  
5.  Ziehen Sie Elemente von der Registerkarte **DataSet** der **Toolbox** auf den Designer.  Weitere Informationen finden Sie unter [Gewusst wie: Bearbeiten eines Datasets](../Topic/How%20to:%20Edit%20a%20Dataset.md).  
  
     \- oder \-  
  
     Ziehen Sie Elemente aus einer aktiven Verbindung im **Server\-Explorer**\/**Datenbank\-Explorer** in den **DataSet\-Designer**.
