# Speichern von Entit&#228;tsdaten
Wenn Sie Entitäten erstellen, aktualisieren oder löschen, die über WCF Data Services eine Verbindung mit einer Datenbank herstellen, müssen sie die Änderungen explizit in der Datenbank speichern.  In den folgenden Themen wird beschrieben, wie Änderungen an Entitätsdaten in WCF\-Data Services\-Anwendungen gespeichert werden.
