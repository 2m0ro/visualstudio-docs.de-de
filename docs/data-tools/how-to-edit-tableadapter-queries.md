# Gewusst wie: Bearbeiten von TableAdapter-Abfragen
Sie bearbeiten TableAdapter\-Abfragen im **Dataset\-Designer** mit dem [TableAdapter\-Abfragekonfigurations\-Assistent](../data-tools/editing-tableadapters.md).  Eine TableAdapter\-Abfrage muss geändert werden, wenn sie nicht mehr den Anforderungen der Anwendung entspricht.  \(Alternativ hierzu können Sie auch zusätzliche Abfragen für den TableAdapter erstellen.\)  Weitere Informationen zum Hinzufügen neuer Abfragen finden Sie unter [Gewusst wie: Erstellen von TableAdapter\-Abfragen](../data-tools/how-to-create-tableadapter-queries.md).  
  
> [!NOTE]
>  Wenn anstelle des **Konfigurations\-Assistenten für TableAdapter\-Abfragen** der [TableAdapter\-Konfigurations\-Assistent](../Topic/TableAdapter%20Configuration%20Wizard.md) geöffnet wird, haben Sie möglicherweise die `Fill`\-Hauptabfrage für den TableAdapter und nicht eine der zusätzlichen Abfragen ausgewählt.  Informationen zum Bearbeiten der `Fill`\-Hauptabfrage für den TableAdapter finden Sie unter [Gewusst wie: Bearbeiten von TableAdapters](../Topic/How%20to:%20Edit%20TableAdapters.md).  
  
 ![TableAdapter mit mehreren Abfragen](../data-tools/media/tableadapter.gif "TableAdapter")  
  
### So bearbeiten Sie eine TableAdapter\-Abfrage  
  
1.  Öffnen Sie das Dataset im **Dataset\-Designer**.  Weitere Informationen finden Sie unter [Gewusst wie: Öffnen eines Datasets im DataSet\-Designer](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md).  
  
2.  Wählen Sie die zu bearbeitende TableAdapter\-Abfrage aus.  
  
3.  Klicken Sie mit der rechten Maustaste auf die TableAdapter\-Abfrage, und wählen Sie **Konfigurieren** aus.  
  
     Der **Konfigurations\-Assistent für TableAdapter\-Abfragen** wird geöffnet. Sie können jetzt die Abfrage oder die gespeicherte Prozedur für diese Abfrage bearbeiten.  
  
4.  Beenden Sie den **Konfigurations\-Assistenten für TableAdapter\-Abfragen** mit den gewünschten Änderungen.  Weitere Informationen finden Sie unter [TableAdapter\-Abfragekonfigurations\-Assistent](../data-tools/editing-tableadapters.md).
