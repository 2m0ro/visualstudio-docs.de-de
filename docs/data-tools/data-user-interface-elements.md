# Daten, Elemente der Benutzeroberfl&#228;che
Dieser Abschnitt enthält Informationen über alle Dialogfelder und Assistenten, die Sie beim Entwerfen des Datenzugriffs in [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)]\- oder Visual C\#\-Anwendungen verwenden.  
  
 [!INCLUDE[note_settings_general](../data-tools/includes/note_settings_general_md.md)]  
  
## In diesem Abschnitt  
 [Data Smart Tags](http://msdn.microsoft.com/de-de/1e0a848f-c57b-47ab-b884-eaaa40726f43)  
 Enthält Informationen über einige Smart Tag\-Befehle, die für Daten verfügbar sind.  
  
 [Dialogfeld "Dataset hinzufügen"](http://msdn.microsoft.com/de-de/0e03c0ff-212b-4bfa-ac51-3c2adb71ead0)  
 Bringt ein vorhandenes typisiertes oder ein neues nicht typisiertes Dataset \(eine Instanz der Klasse **System.Data.Dataset**\) auf ein Formular oder eine Komponente.  
  
 [Dialogfeld "Erweiterte Optionen für SQL\-Generierung"](http://msdn.microsoft.com/de-de/41420450-1ff4-4a1a-b85b-6f6901538fef)  
 Ermöglicht es zu steuern, wie SQL\-Anweisungen oder gespeicherte Prozeduren für einen Datenadapter erstellt werden. Dazu werden Optionen für Aktualisierungen, optimistische Nebenläufigkeit und das Aktualisieren des Datasets angegeben.  
  
 [Choose a Data Source Instance Dialog Box](http://msdn.microsoft.com/de-de/51c47f06-fdc5-453e-9178-0a5a2c5c9f34)  
 Ermöglicht das Auswählen des gewünschten Datasets von einer Liste von Datasets im Projekt.  
  
 [Choose a Data Source to Merge with Dialog Box](http://msdn.microsoft.com/de-de/accafff7-f6bd-481c-a121-fe8a76cd681d)  
 Damit kann ausgewählt werden, mit welcher Datenquelle zusammengeführt werden soll, wenn mehrere Datenquellen verfügbar sind.  
  
 [Choose Key Dialog Box](http://msdn.microsoft.com/de-de/4ddbfbb7-a80a-412a-b80d-291d86376ca3)  
 Ermöglicht es, einen Schlüssel auszuwählen, wenn eine Spalte an einer Einschränkung mehrerer Tasten teilnimmt.  
  
 [Auflistungs\-Editoren](../Topic/Collection%20Editors.md)  
 Liefert Links zu Themen über unterschiedliche Auflistungs\-Editoren, mit denen Sie individuelle Member einer Auflistung erstellen und bearbeiten können.  
  
 [Connections Missing](http://msdn.microsoft.com/de-de/bb9b2e12-7f76-4ee5-acbb-5d20116ee044)  
 Informiert Sie, wenn ein Dataset eine Referenz zu einer Verbindungszeichenfolge enthält, die sich nicht in Anwendungseinstellungen befindet.  
  
 [Dialogfeld "Datenadapter\-Konfigurationsfehler"](http://msdn.microsoft.com/de-de/9ce65cd2-0c7d-4f51-8685-d68be5f3009b)  
 Zeigt einen oder mehrere Fehler an, die aufgetreten sind, als Visual Studio versucht hat, eine Instanz eines Datenadapter zu erstellen und dessen Eigenschaften festzulegen.  
  
 [Konfigurationsassistent für Datenadapter](http://msdn.microsoft.com/de-de/efff90cb-0e4c-4eb3-87dc-65dd9d418809)  
 Konfiguriert SQL\-Befehle oder gespeicherte Prozeduren, die ein Datenadapter verwendet, um Daten aus einer Datenbank in ein Dataset einzulesen und wieder zurückzuschreiben.  Das Thema beschreibt, wie der Assistent ausgeführt wird und was zu tun ist, nachdem er fertig ist.  
  
 [Dialogfeld "Datenadapter\-Vorschau"](http://msdn.microsoft.com/de-de/1f614cd3-4530-457e-84af-00ccbaea08cc)  
 Ermöglicht es zu sehen, wie Daten durch einen Datenadapter in ein Dataset gefüllt werden. Das ist hilfreich, wenn Sie testen möchten, ob der Adapter die erwarteten Daten zurückgibt, ob Tabellenzuordnungen richtig funktionieren und wie sich unterschiedliche Parameterwerte auswirken.  
  
 [Dialogfeld "Datenquellenanmeldung"](http://msdn.microsoft.com/de-de/6f2d9a57-53c3-4841-bd37-a3643eb68d2e)  
 Ermöglicht es, den Zugriff auf eine Datenquelle \(typischerweise eine Datenbank\) anzufordern, für die Sie sich noch nicht authentifiziert haben.  
  
 [DataSet Tab, Toolbox](http://msdn.microsoft.com/de-de/fa5f2d6f-924d-4262-ba1b-e9e7f90e7764)  
 Zeigt die Objekte an, die Sie zu einem typisierten Dataset hinzufügen können.  
  
 [Do you want to include the password in the connection string Dialog Box](http://msdn.microsoft.com/de-de/193696a7-5213-4396-8328-05ac2df6ee94)  
 Ermöglicht zu steuern, ob ein Kennwort in die Verbindungszeichenfolge eingebettet ist.  
  
 [Edit Key Dialog Box](http://msdn.microsoft.com/de-de/f5c80e39-3a42-4284-b222-6ca009fd9675)  
 Ermöglicht es, Schlüssel zu definieren und zu bearbeiten.  
  
 [Dialogfeld "Fremdschlüsseleinschränkung"](http://msdn.microsoft.com/de-de/45d15629-1f4d-40a7-8708-c9ddfebedc1e)  
 Ermöglicht es, auf eine oder mehrere Spalten in einer Dataset\-Tabelle, die auf eine andere Tabelle bezogen ist, eine Fremdschlüsseleinschränkung zu platzieren.  
  
 [Dialogfeld "Dataset generieren"](http://msdn.microsoft.com/de-de/c0efdbaf-13b1-4ee8-ade6-f8a784126cdc)  
 Ermöglicht es, ein neues typisiertes Dataset aus Informationen zu generieren, die von einem oder mehreren Datenadapter bereitgestellt werden; außerdem können Tabellen zu einem vorhandenen Dataset hinzugefügt werden.  
  
 [Multiple BindingSources Dialog Box](http://msdn.microsoft.com/de-de/db76f70c-4fb5-479d-9b64-a67158d48f97)  
 Damit können Sie auswählen, welche <xref:System.Windows.Forms.BindingSource> verwendet werden soll, wenn mehr als eine <xref:System.Windows.Forms.BindingSource> verfügbar ist.  
  
 [Preview Data Dialog Box](http://msdn.microsoft.com/de-de/aa4f0d04-2695-4bb8-946d-54a97ae7287f)  
 Ermöglicht eine Vorschau der Daten, die von Abfragen im Projekt zurückgegeben werden.  
  
 [Dialogfeld "SQL\-Skriptvorschau"](http://msdn.microsoft.com/de-de/e9571e8b-821c-492d-9bc8-b44eba898bdd)  
 Wird als Teil des [Konfigurationsassistenten für Datenadapter](http://msdn.microsoft.com/de-de/efff90cb-0e4c-4eb3-87dc-65dd9d418809) angezeigt und ermöglicht es, das SQL\-Skript zu sehen, dass der Assistent zum Erstellen gespeicherter Prozeduren für das Lesen und Schreiben von Daten verwendet.  
  
 [Dialogfeld "Beziehung"](http://msdn.microsoft.com/de-de/ab8f4b0e-af4c-4725-a550-e2b2ebe43a02)  
 Ermöglicht das Erstellen einer Beziehung \(eines **DataRelation**\-Objekts\), das Informationen über übergeordnete und untergeordnete Datensätze in zwei Datentabellen eines Datasets enthält.  
  
 [Dialogfeld "Suchkriterien\-Generator"](../Topic/Search%20Criteria%20Builder%20Dialog%20Box.md)  
 Ermöglicht das Erstellen einer parametrisieren Abfrage auf einem datengebundenen Windows Form und fügt automatisch die Steuerelemente hinzu, die zum Ausführen der Abfrage erforderlich sind.  
  
 [Dialogfeld "Tabellenzuordnungen"](http://msdn.microsoft.com/de-de/fb4cec1e-f3c8-4773-b409-c2de15293fea)  
 Damit können Sie angeben, welche Spalten in einer Datenbanktabelle oder anderen Datenquelle den Spalten in der Dataset\-Tabelle entsprechen.  
  
 [Dialogfeld "Unique\-Einschränkung"](http://msdn.microsoft.com/de-de/e71a60d7-fae2-4bd0-a1e8-43aae351707d)  
 Ermöglicht es, auf eine oder mehrere Spalten in einer Tabelle eines nicht typisierten Dataset eine Unique\-Einschränkung zu platzieren.
