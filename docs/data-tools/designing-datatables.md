# Entwerfen von DataTables
[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] bietet Entwurfszeittools zum Erstellen und Bearbeiten von Datentabellen \(**DataTable**\).  Eine Übersicht über die Merkmale einer **DataTable** finden Sie unter [DataTables](../Topic/DataTables.md).  
  
 In den folgenden Themen wird erläutert, wie Datentabellen erstellt werden, wie sie typisierten Datasets mit dem **DataSet\-Designer** hinzugefügt werden und wie die Datenspalten \(**DataColumn**\), die eine Datentabelle definieren, erstellt und bearbeitet werden.  
  
> [!NOTE]
>  Je nach den aktiven Einstellungen oder der Version unterscheiden sich die Dialogfelder und Menübefehle auf Ihrem Bildschirm möglicherweise von den in der Hilfe beschriebenen.  Wählen Sie im Menü **Extras** die Option **Einstellungen importieren und exportieren** aus, um die Einstellungen zu ändern.  Weitere Informationen finden Sie unter [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/de-de/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## Referenz  
 <xref:System.Data.DataSet>  
 Stellt einen speicherinternen Datencache dar.  
  
 <xref:System.Data.DataTable>  
 Stellt eine Tabelle mit speicherinternen Daten dar.  
  
 <xref:System.Data.DataColumn>  
 Stellt das Schema einer Spalte in einer Datentabelle \(<xref:System.Data.DataTable>\) dar.  
  
 <xref:System.Data.DataRelation>  
 Stellt die hierarchische Beziehung zwischen zwei <xref:System.Data.DataTable>\-Objekten dar.
