# Gewusst wie: &#214;ffnen des Datenquellenfensters
Im **Datenquellenfenster** werden die Datenquellen Ihres Projekts angezeigt.  Sie verwenden das **Datenquellenfenster**, um datengebundene Benutzeroberflächen zu erstellen. Dazu ziehen Sie Elemente aus dem Fenster auf die Formulare des Projekts.  Siehe [Binden von Windows Forms\-Steuerelementen an Daten in Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Sie können das Fenster **Datenquellen** auffüllen, indem Sie den **Assistenten zum Konfigurieren von Datenquellen** ausführen.  Weitere Informationen finden Sie unter [Assistent zum Konfigurieren von Datenquellen](../data-tools/media/data-source-configuration-wizard.png).  
  
### So öffnen Sie das Datenquellenfenster  
  
-   Klicken Sie in der Menüleiste auf **Ansicht**, **Weitere Fenster**, **Datenquellen** \(oder drücken Sie die Tasten UMSCHALT\+ALT\+D\).  
  
    > [!IMPORTANT]
    >  Sie können eine Datenquelle einem Projekt nur hinzufügen, wenn es das Erstellen von Datenquellen und Arbeiten mit Datenquellen unterstützt.  Beispielsweise können Sie das Fenster **Datenquellen** in einem Projekt für eine Windows Store\-App nicht öffnen.
