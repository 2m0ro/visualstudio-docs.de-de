# <a name="the-connection-property-in-the-application-settings-file-is-missing-or-incorrect"></a>Die Verbindungseigenschaft in der Application Settings-Datei fehlt oder ist fehlerhaft.

Die Verbindungseigenschaft fehlt in der Datei mit den Anwendungseinstellungen, oder sie ist falsch. Die Verbindungszeichenfolge aus der *dbml* an seiner Stelle Datei verwendet wurde.

Die *dbml* Datei enthält einen Verweis auf eine Verbindungszeichenfolge in der Datei mit den Anwendungseinstellungen, die nicht gefunden werden kann. Diese Meldung dient nur zu Informationszwecken; Einstellung der Verbindungszeichenfolge wird erstellt, wenn **OK** geklickt wird.

Wählen Sie auf diese Meldung reagieren, **OK**. Die Verbindungsinformationen, die in enthalten ist das *dbml* Anwendungseinstellungen Datei hinzugefügt wird.
