# <a name="extensibility-in-visual-studio"></a>Erweiterbarkeit in Visual Studio
Visual Studio bietet Ihnen viele Optionen, Erweiterbarkeit. Sie können erstellen eigene SDKs, verwenden Sie die Windows, Windows Phone und Azure-SDKs (die als Teil von Visual Studio installiert sind) und Visual Studio selbst erweitern.  
  
## <a name="extend-visual-studio"></a>Erweitern von Visual Studio  
 Sie können das Visual Studio SDK jedes Teil von Visual Studio erweitern: Befehle, Menüs, Windows, Editoren und Projekte. Um weitere Informationen finden Sie unter [Visual Studio SDK](../extensibility/visual-studio-sdk.md).  
  
## <a name="create-your-own-sdks"></a>Erstellen Sie eigene SDKs  
 Weitere Informationen zum Erstellen, Verpacken und Bereitstellen Ihrer eigenen Plattform und Erweiterung-SDKs: [Erstellen eines Software Development Kit](../extensibility/creating-a-software-development-kit.md).  
  
## <a name="roslyn-extensibility"></a>Roslyn-Erweiterbarkeit  
 Erfahren Sie, wie Roslyn für Erweiterbarkeit verwendet: [.NET Compiler Platform ("Roslyn") Erweiterbarkeit](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).
