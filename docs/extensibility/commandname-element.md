# CommandName-Element
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Die `CommandName` Elements gibt den Text, der angezeigt wird, in der Kategorie Tastatur die **Optionen** im Dialogfeld und in der **Befehle** Liste der **Anpassen** \(Dialogfeld\).  
  
## Syntax  
  
```  
<CommandName>MyCommand</CommandName>  
```  
  
## Attribute und Elemente  
 In den folgenden Abschnitten werden Attribute sowie untergeordnete und übergeordnete Elemente beschrieben.  
  
### Attribute  
 Keine  
  
### Untergeordnete Elemente  
 Keine  
  
### Übergeordnete Elemente  
  
|Element|Beschreibung|  
|-------------|------------------|  
|[Zeichenfolgen\-Element](../extensibility/strings-element.md)|Text\-Elemente, wie z. B. Gruppen `ButtonText` und `CommandName`.|
