Methoden werden unter jede Schnittstelle in der Tabelle der Inhalte und auf der Seite "Schnittstelle" in Vtable-Reihenfolge in alphabetischer Reihenfolge aufgeführt.

 [Wichtige Schnittstellen](../../../extensibility/debugger/reference/core-interfaces.md)

 [Schnittstellen für die Ausdrucksauswertung](../../../extensibility/debugger/reference/expression-evaluation-interfaces.md)  
 Beschreibt die Schnittstellen, die für das Auswerten von Ausdrücken in einer Aufrufliste im Unterbrechungsmodus.

 Beschreibt die Schnittstellen, die ein Symbol-Anbieter implementiert werden, um Typen zu IDs zuzuordnen.
