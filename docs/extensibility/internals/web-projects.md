# Webprojekte
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Ein Webprojekt ist ein System, das Projektsystem Projects erstellt.  Webprojekte erstellen wiederum Webanwendungen.  Sie können Programm eine Webseite mit den traditionellen Programmiersprachen, z. B. [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] , oder [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]und Verarbeiten von Informationen zu einem Benutzer sammeln, speichern es in einer Datenbank usw.  Webseiten mit zugeordnetem Code werden die intelligente Webseiten aufgerufen.

 Bietet eine Übersicht über [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] Websiteprojekten dar.  Ein Websiteprojekt generiert eine ausführbare Datei für jede intelligente Webseite.  Zusätzliche ausführbare Dateien werden von Quellcodedateien im Ordner \/App\_Code\- generiert.
