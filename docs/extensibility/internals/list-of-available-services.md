# Liste der verf&#252;gbaren Dienste
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] und das Visual Studio SDK unterstützt die folgenden Dienste. Einige Pakete bieten eigene Dienste, die hier nicht aufgeführt sind, Sprachdienste verfügen z. B. keinen einzelnen Dienst\-GUID. Sie müssen den Namen der Sprache verwenden, um die GUID des Sprachdiensts in der Registrierung gefunden.  
  
 Verwenden Sie den Dienst\-GUIDs, die hier aufgeführten oder von einer anderen Quelle \(z. B. Language Services\) erhalten hat, um die primäre Schnittstelle oder Schnittstellen, die zusammen mit jedem Dienst zu erhalten.
