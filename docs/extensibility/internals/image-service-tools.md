# <a name="image-service-tools"></a>Image-Service-Tools
Das VS-SDK enthält mehrere Tools, mit denen Extension-Generatoren, die den Image-Dienst von Visual Studio verwenden.  
  
-   Die [Manifest aus Ressourcen](../../extensibility/internals/manifest-from-resources.md) Tool kann eine Liste von Bildressourcen (PNG oder XAML-Dateien) und generiert eine Bild-Manifestdatei.  
  
-   Die [Code Manifest](../../extensibility/internals/manifest-to-code.md) Tool akzeptiert eine Bild-Manifestdatei und generiert eine Wrapperdatei in C++, c#, VB oder VSCT-Dateien verwendet.  
  
-   Die [Bildbibliotheks-Viewer](../../extensibility/internals/image-library-viewer.md) laden, bearbeiten und bildmanifesten suchen können.
