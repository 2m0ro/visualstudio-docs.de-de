# Image-Service-Tools
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Das Visual Studio SDK enthält mehrere Tools zum Extension\-Generatoren, die den Image\-Dienst von Visual Studio verwenden.  
  
-   Die [Manifest aus Ressourcen](../../extensibility/internals/manifest-from-resources.md) Tool nimmt eine Liste von Bildressourcen \(PNG oder XAML\-Dateien\) und generiert eine Manifestdatei Bild.  
  
-   Die [Manifest für Code](../../extensibility/internals/manifest-to-code.md) Tool nimmt eine Image\-manifest\-Datei und generiert eine Wrapperdatei in C\+\+, c\#, VB oder VSCT\-Dateien.  
  
-   Die [Bildanzeige\-Bibliothek](../../extensibility/internals/image-library-viewer.md) laden, bearbeiten und Image Manifeste suchen können.
