# <a name="objects"></a>erzwingen
Die folgenden Verweisinformationen erläutert Objekte, die zum Erstellen einer Instanz von der Kern-Text-Editor verwendet werden.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [VSCodeWindow-Objekt](../extensibility/vscodewindow-object.md)  
 Beschreibt das Code-Fenster-Objekt, einem spezialisierten Dokumentfenster angezeigt, die einer oder mehreren Textansichten einschließen kann.  
  
 [VSCodeWindowManager-Objekt](../extensibility/vscodewindowmanager-object.md)  
 Beschreibt das Code-Fenster-Manager-Objekt, das von der Sprachdienst implementiert wird, und ist zuständig für das Verwalten von randsteuerelementen an.  
  
 [VSTextBuffer-Objekt](../extensibility/vstextbuffer-object.md)  
 Beschreibt die TextBuffer-Objekt, das einen Stream von Unicode-Textdatei, die Daten für eine Datei darstellt.  
  
 [VSTextView-Objekt](../extensibility/vstextview-object.md)  
 Beschreibt die Textansicht ein Fenster, das ermöglicht Benutzern das Anzeigen und Bearbeiten von Unicode-Text des Textpuffers.
