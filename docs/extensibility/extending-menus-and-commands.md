# Erweitern von Men&#252;s und Befehlen
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Befehle werden Sie Aktionen und Prozesse zu Visual Studio hinzufügen. In den meisten Fällen werden die Befehle in Menüs oder Symbolleisten angezeigt. VSPackage\-Projektvorlage veranschaulicht einen einfachen Befehl implementieren. Eine etwas länger, aber immer noch grundlegenden Implementierung finden Sie unter [Erstellen eine Erweiterung mit einem Menübefehl](../extensibility/creating-an-extension-with-a-menu-command.md).  
  
 Weitere Informationen zu Visual Studio\-Befehle, Menüs und Symbolleisten finden Sie unter [Befehle, Menüs und Symbolleisten](../extensibility/internals/commands-menus-and-toolbars.md).  
  
 Befehle, Menüs und Symbolleisten werden in der VSCT\-Datei definiert, die Teil eines VSPackage\-Projekten. Sie finden Informationen zu Visual Studio\-IDE und die VSCT\-Datei im [Wie VSPackages Benutzeroberflächenelemente hinzufügen](../extensibility/internals/how-vspackages-add-user-interface-elements.md).  
  
 In den folgenden Themen wird erläutert, wie verschiedene Arten von Befehle, Menüs und Symbolleisten hinzufügen.
