# Erweitern von L&#246;sungen
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Eine Lösung ist eine Gruppierung von einem oder mehreren Projekten, die zum Erstellen einer Anwendung zusammenarbeiten.  
  
## In diesem Abschnitt  
 Die folgenden exemplarischen Vorgehensweisen zeigen einige Methoden in denen Sie Lösungen erweitern können.  
  
1.  [Erweitern Sie den Projektmappen\-Explorer\-Filter](../extensibility/extending-the-solution-explorer-filter.md) Erläutert das Filtern, das im Projektmappen\-Explorer angezeigt.  
  
2.  [Verwalten von Projekt laden in einer Projektmappe](../extensibility/managing-project-loading-in-a-solution.md) Erläutert das Projekt laden, um die Leistung zu verbessern, wenn ein Benutzer eine Lösung lädt verwalten.
