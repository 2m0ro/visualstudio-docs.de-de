# Visuellen Sprachw&#246;rterbuch f&#252;r Visual Studio
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Visual Studio verwendet eine gemeinsame visuelle Sprache zum Veranschaulichen der Konzepte, die die Bausteine der ikonographie und andere Bilder sind. Wenn Sie neue Symbole zu entwerfen, verwenden dieses Wörterbuch die Konzepte zu identifizieren, die Sie in Ihr eigenes Bild vermitteln möchten.  
  
 **Allgemeine Richtlinien**  
  
-   Diese visual Konzepte müssen einheitlich mit ihrer Wörterbuchdefinition verwendet werden. Andernfalls riskieren Sie Benutzer, die gemeinsame Elemente auf konsistente bedeuten erwarten verwirrend.  
  
-   Wählen Sie nur die meisten relevanten oder wichtige Konzepte für die Verwendung in Ihrer Symbole, sodass die Bilder gestochen und leicht zu unterscheiden ist.  
  
-   Verwenden Sie nur ein oder zwei Konzepte in jedem Bild.  
  
-   Weitere Strategien zum Entwerfen von Symbolen, finden Sie unter [Symbol Entwurf](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign).
