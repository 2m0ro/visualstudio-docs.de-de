# LocationFieldMRUPrefix-Element (Visual Studio-Vorlagen)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Gibt die zuletzt verwendeten \(MRU\-\)Pfade im Dialogfeld **Neues Projekt** und **Neues Element hinzufügen** an.  
  
## Syntax  
  
```  
<LocationFieldMRUPrefix> ... </LocationFieldMRUPrefix>  
```  
  
## Attribute und Elemente  
 In den folgenden Abschnitten werden Attribute sowie untergeordnete und übergeordnete Elemente beschrieben.  
  
### Attribute  
 Keine.  
  
### Untergeordnete Elemente  
 Keine.  
  
### Übergeordnete Elemente  
  
|Element|Description|  
|-------------|-----------------|  
|[TemplateData](../extensibility/templatedata-element-visual-studio-templates.md)|Kategorisiert die Vorlage und definiert, wie diese in den Dialogfeldern **Neues Projekt** oder **Neues Element hinzufügen** angezeigt wird.|  
  
## Hinweise  
 Dieses Element sollte nur in den über [!INCLUDE[vsipprvsip](../extensibility/includes/vsipprvsip_md.md)] erstellten Vorlagen verwendet werden.
