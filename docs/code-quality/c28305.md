# <a name="c28305"></a>C28305
Warnung C28305: Fehler beim Analysieren der \<token > ermittelt wurde.

 Diese Warnung wird ausgegeben, wenn der Ausdruck, der mit dem angegebenen Token nicht ordnungsgemäß formatiert ist.
