# C1256
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C1256: PCH muss geschrieben und gelesen.  
  
 Das Codeanalysetool meldet diese Warnung, wenn einem internen Fehler im Tool gibt, nicht im Code, die analysiert wird.
