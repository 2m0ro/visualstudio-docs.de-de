# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28227: Anmerkung kann nicht zusätzlich ein PrimOp sein \(siehe vorherige Deklaration\).  
  
 Diese Warnung gibt an, dass, die benannte Anmerkung als gewöhnliche Anmerkung deklariert wird, und auch wurde zuvor als PrimOp deklariert.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
