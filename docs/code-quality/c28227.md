# <a name="c28227"></a>C28227
Warnung C28227: Anmerkung kann nicht zusätzlich ein primop sein. (Siehe vorherige Deklaration)  
  
 Diese Warnung gibt an, dass die benannten Anmerkung als gewöhnliche Anmerkung deklariert wird, und auch wurde zuvor als zusätzlich ein PrimOp deklariert. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
