# <a name="code-analysis-rule-set-reference"></a>Codeanalyse-Regelsatzreferenz

Wenn Sie die Codeanalyse für Projekte mit verwaltetem Code in Visual Studio konfigurieren, können Sie auswählen, aus einer Liste integrierter *-Regelsätze*. Sie können entweder eine der folgenden integrierten Regelsätze verwenden, oder Sie können [Anpassen eines Regelsatzes](../code-quality/how-to-create-a-custom-rule-set.md) an Ihre projektanforderungen anpassen.

Die Themen in diesem Abschnitt wird beschrieben, die integrierte Regel legt fest, und die Regeln (oder Warnungen), die sie enthalten.
