# C6704
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C6704: Unerwarteter Ausdrucksfehler der Anmerkung: \<Anmerkungen\> \[\<warum\>\]  
  
 Diese Warnung wird ausgegeben, wenn ein Fehler in den Anmerkungen gibt.
