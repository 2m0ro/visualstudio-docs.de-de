# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28236: Die Anmerkung kann nicht in einem Ausdruck verwendet werden.  
  
 Diese Warnung gibt an, dass versucht, ein Feature zu verwenden ausgeführt wurde, die deklariert hat, eine Anmerkung in einem Ausdruckskontext zu sein.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
