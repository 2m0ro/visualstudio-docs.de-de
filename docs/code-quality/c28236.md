# <a name="c28236"></a>C28236
Warnung C28236: Die Anmerkung kann nicht in einem Ausdruck verwendet werden.

 Diese Warnung gibt an, dass versucht wurde, eine Funktion deklariert, um eine Anmerkung im Ausdruckskontext eines zu verwenden. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
