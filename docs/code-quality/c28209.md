# <a name="c28209"></a>C28209
Warnung C28209: Die Deklaration für das Symbol weist einen Konflikt auf.  
  
 Diese Warnung gibt an, die Deklaration eines nicht ordnungsgemäß konstruierten Anmerkung. Diese Warnung sollte niemals während normaler Verwendung auftreten.
