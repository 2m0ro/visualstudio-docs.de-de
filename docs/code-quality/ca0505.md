# <a name="ca0505"></a>CA0505
Die veraltete Eigenschaft wird ignoriert, da die ersetzende Eigenschaft definiert ist.

 Warnung CA505 wird generiert, wenn die angegebene veraltete sowohl ersetzende Codeanalyseeigenschaften enthalten Werte in einem Projekt oder die Konfigurationsdatei zu erstellen. Um dieses Problem zu beheben, entfernen Sie die veraltete Eigenschaft.
