# <a name="ca0505"></a>CA0505

Die als veraltet markierte Eigenschaft wird ignoriert, da die ersetzende Eigenschaft definiert ist.

Warnung CA505 wird generiert, wenn sowohl die veralteten und ablösenden Code Analysis-Eigenschaften Werte in einem Projekt oder erstellen die Konfigurationsdatei. Um dieses Problem zu beheben, entfernen Sie die als veraltet markierte Eigenschaft ein.
