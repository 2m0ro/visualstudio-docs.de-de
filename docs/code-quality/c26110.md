# C26110
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C26110: Fehler des Aufrufers beim Aufrechterhalten der Sperre \<Sperre\>, bevor die Funktion \<func\> aufgerufen wird.  
  
 Wenn eine Sperre erforderlich ist, stellen Sie sicher, zu erl채utern, dass die Funktion selbst oder sein Aufrufer die Sperre erhalten sollen.  Warnung C26110 wird ausgegeben, wenn eine Verletzung der Anmerkung `_Requires_lock_held_` gibt.
