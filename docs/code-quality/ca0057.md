# <a name="ca0057"></a>CA0057
Verstöße gegen darf nicht den aktuellen Satz von Zielen und Regeln zugeordnet werden  
  
 Dieser Fehler tritt auf, wenn FxCopCmd.exe manuell mit einer Projektdatei aufgerufen wird oder einen importierten XML-Bericht mit Verletzungen, die den aktuellen Satz von Zielen und Regeln zugeordnet werden kann.  
  
 Erfolgt nur, wenn Analysis erfolgt mit der Befehlszeilenversion des Tools.
