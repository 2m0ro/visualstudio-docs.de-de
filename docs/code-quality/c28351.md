# <a name="c28351"></a>C28351
Warnung C28351: die Anmerkung \<Annotation > beschreibt, wo ein dynamischer Wert (Variable) in der Bedingung verwendet werden darf.  
  
 Diese Warnung wird ausgegeben, wenn eine Anmerkung angewendet wird, in denen die C-/C++-Typ überprüft wird.  
  
 Der Ausdruck in der `_When_` sollte auf eine Konstante ausgewertet. Die `_When_` wird ignoriert.
