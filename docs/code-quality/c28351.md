# C28351
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28351: Die Anmerkung \<Anmerkung\> beschreibt, wo ein dynamischer Wert \(eine Variable\) in der Bedingung nicht verwendet werden darf.  
  
 Diese Warnung wird ausgegeben, wenn eine Anmerkung angewendet wird, wobei der C\/C\+\+\-Typ überprüft wird.  
  
 Der Ausdruck in `_When_` sollte als Konstante auswerten.  Die `_When_` wird ignoriert.
