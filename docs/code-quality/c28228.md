# <a name="c28228"></a>C28228
Warnung C28228: Anmerkungsparameter: Typ kann nicht in Anmerkungen verwendet werden.  
  
 Diese Warnung gibt an, dass ein Parameter des Typs ist, die nicht unterstützt wird. Anmerkungen können nur eine begrenzte Anzahl von Typen als Parameter verwenden. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
