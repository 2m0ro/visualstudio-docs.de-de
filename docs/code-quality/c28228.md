# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28228: Anmerkungsparameter: Typ kann nicht in Anmerkungen verwendet werden.  
  
 Diese Warnung gibt an, dass ein Parameter vom Typ ist, der nicht unterstützt wird.  Anmerkungen können einen eingeschränkten Satz von Typen als Parameter nur verwenden.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
