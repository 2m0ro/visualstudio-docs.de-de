# C28216
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28216: Die \_Check\_return\_ Anmerkung gilt nur für Nachbedingungen für Funktionsparameter bestimmten zu.  
  
 `_Check_return_` Die Anmerkung ist in einem anderen Kontext als `__post` angewendet wurde; sie möglicherweise einen Modifizierer `__post` \(oder `__drv_out`\), oder sie wird falsch platziert werden.
