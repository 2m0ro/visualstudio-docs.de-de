# <a name="c28216"></a>C28216
Warnung C28216: die _Check_return\_ Anmerkung gilt nur für nach der Bedingungen für den bestimmten Funktionsparameter.

 Die `_Check_return_` Anmerkung anders als in einem Kontext angewendet wurde `__post`; muss möglicherweise neu eine `__post` (oder `__drv_out`)-Modifizierer, oder es kann falsch platziert werden.
