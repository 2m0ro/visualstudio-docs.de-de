# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28232: \_Pre\_, \_Post\_ oder \_Deref\_ angewendet kein Konfigurationsformat Anmerkung  
  
 Diese Warnung gibt an, dass `_Pre_`, `_Post_` oder den Operator `_Deref_` in einem Anmerkungsausdruck ohne eine funktionale folgenden Anmerkung wird; der Modifizierer wurde ignoriert, aber dieser gibt eine falsch codierten Anmerkung besteht.
