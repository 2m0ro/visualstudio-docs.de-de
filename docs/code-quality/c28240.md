# <a name="c28240"></a>C28240
Warnung C28240: die Anmerkung zum Parameter verfügt über param2, jedoch nicht über param1.  
  
 Interner Fehler in der Modelldatei PREfast. Diese Warnung sollte in der Regel nicht auftreten.
