# <a name="ca0053"></a>CA0053
Regelassembly konnte nicht geladen werden.

 Dies gibt an, dass eine Regelassembly nicht mittels Reflektion geladen werden konnte. Dies kann auftreten, wenn der Benutzer benutzerdefinierte Regeln verfügt, die nicht ordnungsgemäß erstellt wurden, oder das Analysetool nicht die Assembly aus anderen Gründen laden, z. B. der Dateisystemzugriff verweigert wurde.
