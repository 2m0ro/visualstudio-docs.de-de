# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28225: In der Anmerkung wurde nicht die richtige Anzahl an Parametern gefunden  
  
 Diese Warnung gibt an, dass die benannte Anmerkung mit der falschen Anzahl von Parametern verwendet wird.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
