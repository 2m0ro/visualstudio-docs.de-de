Unsichere Deserialisierer sind anfällig, wenn nicht vertrauenswürdige Daten zu deserialisieren. Ein Angreifer könnte die serialisierten Daten unerwartete die Typen mit schädlichen Nebeneffekten ändern. Ein Angriff auf eine unsichere Deserialisierungsprogramm kann z. B. Befehle ausführen, auf das zugrunde liegende Betriebssystem, über das Netzwerk kommunizieren, Dateien, oder löschen.
