# C28241
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28241: Die Anmerkung für die Funktion des Parameters wird nicht erkannt.  
  
 Ein nicht erkannter Anmerkungsname wurde verwendet.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
