# <a name="c28241"></a>C28241
Warnung C28241: Die Anmerkung für die Funktion des Parameters wird nicht erkannt.  
  
 Es wurde eine unbekannte Anmerkungsname verwendet. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
