# <a name="ca0501"></a>CA0501
Kann nicht gelesen werden Ausgabebericht  
  
 Dies tritt auf, wenn Ausgabebericht aus einem der folgenden Gründe nicht geschrieben werden kann:  
  
-   Der Datenträger ist voll  
  
-   Keine Berechtigungen zum Schreiben im Verzeichnis  
  
-   Verzeichnis ist nicht vorhanden.
