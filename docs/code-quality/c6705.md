# C6705
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C6705: Anmerkungsfehler hat \<erwartete\_Zahl\> Argumente für Anmerkung \<Parameter\> erwartet, \<tatsächliche\_Zahl\> wurde gefunden.  
  
 Diese Warnung wird ausgegeben, wenn ein Fehler in den Anmerkungen gibt.
