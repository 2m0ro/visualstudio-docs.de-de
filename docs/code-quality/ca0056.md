# <a name="ca0056"></a>CA0056
Eine Projektdatei hat eine falsche Version von Analysis-Tool.  
  
 Diese Warnung tritt auf, wenn FxCopCmd.exe manuell mit einer Projektdatei aufgerufen wird, die eine falsche Code Analysis-Toolversion vorhanden ist.
