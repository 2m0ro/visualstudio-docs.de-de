# <a name="ca0056"></a>CA0056
Eine Projektdatei hat eine falsche Version des Analysetools.

 Diese Warnung tritt auf, wenn FxCopCmd.exe manuell mit der Projektdatei aufgerufen wird, die über einen falschen Code Analysis Tool-Version verfügt.
