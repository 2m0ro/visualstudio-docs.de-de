# C6323
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C6323: Verwendung eines arithmetischen Operators auf Boolean\-Typ\(en\)  
  
 Diese Warnung wird ausgegeben, wenn arithmetische Operatoren auf booleschen Datentypen verwendet werden.  Die Verwendung eines falschen Operators kann zu falschen Ergebnissen f√ºhren.  Zudem weist sie darauf hin, dass die Absicht des Programmierers im Code nicht wiedergegeben wird.  
  
## Beispiel  
 Der folgende Code generiert diese Warnung:  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 Um diese Warnung zu korrigieren, verwenden Sie den richtigen Datentyp und Operator.  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
