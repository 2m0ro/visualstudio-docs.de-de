# Messen von Komplexit&#228;t und Verwaltbarkeit verwalteten Codes
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Die gesteigerte Komplexität moderner Softwareanwendungen erschwert gleichzeitig auch die Erstellung zuverlässigen und verwaltbaren Codes.  In den vergangenen Jahren wurden viele neue Softwaremaßstäbe entwickelt. Anhand dieser Codemetrik können Entwickler leichter verstehen, an welchen Stellen ihr Code umprogrammiert oder verstärkt getestet werden muss.  
  
 Entwickler können mit [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] Codemetrikdaten generieren, durch die die Komplexität und Verwaltbarkeit ihres verwalteten Codes gemessen wird.  Codemetrikdaten können für eine ganze Projektmappe oder für ein einzelnes Projekt generiert werden.  
  
 In diesem Abschnitt wird beschrieben, wie Sie Codemetrikdaten in [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] generieren und verwenden.
