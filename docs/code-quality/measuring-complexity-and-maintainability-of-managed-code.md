# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Messen von Komplexität und Verwaltbarkeit verwalteten Codes
Zudem erhöht die zunehmenden Komplexität der modernen softwareanwendungen schwierigkeiten versteht man den Code, zuverlässig und verwaltbar. In den letzten Jahren viele Softwaremaßstäben, Codemetrik genannt wurden entwickelt, ermöglicht es Entwicklern, die zu verstehen, in denen ihre auszugehen Überarbeitungen und erhöhte testen können.  
  
 Entwickler können [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] zum Generieren von Codemetrikdaten, die die Komplexität und verwaltbarkeit verwalteten Codeprobleme zu messen. Codemetrikdaten können für eine gesamte Projektmappe oder ein einzelnes Projekt generiert werden.  
  
 In diesem Abschnitt wird beschrieben, wie zum Generieren und Verwenden von Codemetrik in [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
