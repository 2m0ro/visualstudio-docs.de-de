# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28289: Eine Funktion: ReadableTo oder WritableTo haben eine GrenzeSpezifikation nicht als Parameter  
  
 Das Codeanalysetool meldet diese Warnung, wenn die Funktion\/Parameteranmerkung miscoded, wie angegeben.
