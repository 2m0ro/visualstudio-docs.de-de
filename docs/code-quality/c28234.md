# C28234
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28234: \_At\_ Ausdruck gilt nicht auf aktuelle Funktion zu  
  
 Diese Warnung gibt an, dass der Wert eines Ausdrucks `_At_` kein Objekt, auf das zugegriffen werden kann, identifiziert.
