# <a name="c6995"></a>C6995
Warnung C6995: Fehler beim Speichern der XML-Protokolldatei

 Diese Warnung gibt an, dass Code Analysis-Tool das Protokoll Fehler erstellen kann, das die Ausgabe die Codeanalyse ist.

 Dieser Fehler kann auf ein Datenträgerfehler hin oder bedeutet, dass Sie nicht über die Berechtigung zum Erstellen einer Datei im angegebenen Verzeichnis verfügen.
