# C28288
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28288: Eine Funktion Syntaxfehler in \_At\_\(\) Anmerkung \(Ungültige Parametername\)  
  
 Das Codeanalysetool meldet diese Warnung bei die Anmerkung `SAL_at` \(`__drv_at`\) wird verwendet und der Parameterausdruck kann im aktuellen Kontext nicht interpretiert werden.  Dazu gehören mit einem falsch geschriebenen Parameters oder eines Membernamens ein, oder einen Orthographiefehler "return" oder "this" Schlüsselwörter.
