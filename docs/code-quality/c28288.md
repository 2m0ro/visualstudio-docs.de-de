# <a name="c28288"></a>C28288
Warnung C28288: für Funktion Syntaxfehler in _gleich\_()-Anmerkung (Ungültiger Parametername)  
  
 Der Code Analysis-Tool meldet diese Warnung, wenn die `SAL_at` (`__drv_at`)-Anmerkung und der Ausdruck im aktuellen Kontext nicht interpretiert werden kann. Dazu gehören beispielsweise einen falsch geschriebenen Parameter oder Elementnamen oder eine falsche Schreibweise für "return" oder "this" Schlüsselwörter verwenden.
