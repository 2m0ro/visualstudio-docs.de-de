# C28284
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28284: Für das Symbol werden Prädikate für Nicht\-Funktionssymbole momentan nicht unterstützt.  
  
 Diese Warnung gibt an, dass eine bedingte Anmerkung \(Prädikat, `__drv_when`\) auf eine andere als eine Funktion gefunden wurde.
