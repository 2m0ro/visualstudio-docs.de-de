# <a name="c28284"></a>C28284
Warnung C28284: Für das Symbol werden Prädikate für Nicht-Funktionssymbole momentan nicht unterstützt.  
  
 Diese Warnung gibt an, die einer bedingten Anmerkung (Prädikats `__drv_when`) auf einen anderen Wert als eine Funktion gefunden wurde.
