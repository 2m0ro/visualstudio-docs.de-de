# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C1252: Zirkuläre oder fehlende Abhängigkeit zwischen Plug\-Ins: erfordert GUID  
  
 Das Codeanalysetool meldet diese Warnung, wenn einem internen Fehler mit der Abhängigkeiten gibt, nicht im Code, die analysiert wird.
