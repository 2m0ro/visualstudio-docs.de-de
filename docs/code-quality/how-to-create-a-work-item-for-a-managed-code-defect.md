# <a name="how-to-create-a-work-item-for-a-managed-code-defect"></a>Gewusst wie: Erstellen einer Arbeitsaufgabe für einen Fehler in verwaltetem Code
Können Sie die Arbeitsaufgabe adressnachverfolgungsfeature Protokoll Arbeitsaufgabe innerhalb [!INCLUDE[vsPreLong](../code-quality/includes/vsprelong_md.md)]. Um dieses Feature verwenden zu können, muss das Projekt Teil eines Teamprojekts in [!INCLUDE[esprfound](../code-quality/includes/esprfound_md.md)].  
  
### <a name="to-create-a-work-item-for-managed-code-defect"></a>So erstellen eine Arbeitsaufgabe für Fehler in verwaltetem code  
  
1.  In der **Codeanalyse** Fenster, wählen Sie die Warnung.  
  
2.  Wählen Sie **Aktionen**, wählen Sie dann **Arbeitsaufgabe erstellen** , und wählen Sie den Typ des zu erstellenden Arbeitsaufgabentyp.  
  
     Zur Angabe der Fehler-Informationen wird eine neue Arbeitsaufgabe erstellt.  
  
### <a name="to-create-a-work-item-for-multiple-managed-code-defects"></a>So erstellen eine Arbeitsaufgabe für mehrere Fehlern in verwaltetem code  
  
1.  In der **Fehlerliste**, mehrere Warnungen auswählen und dann auf die Warnungen.  
  
2.  Zeigen Sie auf **Arbeitsaufgabe erstellen** , und klicken Sie auf den Typ des zu erstellenden Arbeitsaufgabentyp.  
  
     Ein einzelnes Arbeitselement wird für die ausgewählten Warnungen zur Angabe der Fehlerinformationen erstellt.
