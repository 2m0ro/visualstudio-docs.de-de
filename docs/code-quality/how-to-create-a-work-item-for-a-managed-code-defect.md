# Gewusst wie: Erstellen einer Arbeitsaufgabe f&#252;r einen Fehler in verwaltetem Code
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Mithilfe des Features für die Arbeitsaufgabenverfolgung können Sie in [!INCLUDE[vsPreLong](../code-quality/includes/vsprelong_md.md)] Arbeitsaufgaben protokollieren.  Um diese Funktion verwenden zu können, muss das Projekt Teil eines Teamprojekts in [!INCLUDE[esprfound](../code-quality/includes/esprfound_md.md)] sein.  
  
### So erstellen Sie eine Arbeitsaufgabe für einen Fehler in verwaltetem Code  
  
1.  Im Fenster **Codeanalyse** wählen Sie die Warnung.  
  
2.  Wählen Sie, und wählen Sie und wählen Sie den Typ der Arbeitsaufgabe **Aktionen**, um zu erstellen.  
  
     Eine neue Arbeitsaufgabe wird erstellt, in der Sie Informationen zum Fehler angeben können.  
  
### So erstellen Sie eine Arbeitsaufgabe für mehrere Fehler in verwaltetem Code  
  
1.  Wählen Sie in der **Fehlerliste** mehrere Warnungen aus, und klicken Sie dann mit der rechten Maustaste auf die Warnungen.  
  
2.  Zeigen Sie auf **Arbeitsaufgabe erstellen**, und klicken Sie auf den Typ der zu erstellenden Arbeitsaufgabe.  
  
     Für alle ausgewählten Warnungen wird eine einzige Arbeitsaufgabe erstellt, in der Sie Probleminformationen angeben können.
