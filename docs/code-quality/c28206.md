# C28206
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28206: \<Ausdruck\>: linker Operand zeigt auf eine Struktur, `->` verwenden.  
  
 Diese Warnung wird ausgegeben, wenn der Struct\-Zeiger Notation `->` wurde erwartet.
