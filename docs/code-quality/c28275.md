# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28275: Der Parameter für \_Macro\_value\_ ist ungültig  
  
 Diese Warnung gibt an, dass es einen internen Fehler in der Modelldatei gibt, nicht im Code, die analysiert wird.  Die *macroValue*\-Funktion wurde ohne einen Parameter aufgerufen.
