# <a name="c28226"></a>C28226
Warnung C28226: Anmerkung kann nicht zusätzlich ein primop sein (in der aktuellen Deklaration).  
  
 Diese Warnung gibt an, dass die benannten Anmerkung als zusätzlich ein PrimOp deklariert wird, und auch wurde zuvor als eine normale Anmerkung deklariert. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, während normaler Verwendung betrachtet werden.
