# C28226
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28226: Anmerkung kann nicht zusätzlich ein PrimOp sein \(in der aktuellen Deklaration\).  
  
 Diese Warnung gibt an, dass die benannte Anmerkung, als PrimOp deklariert wird, und auch wurde zuvor als normale Anmerkung deklariert.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, während normaler Verwendung gesehen werden erwartet.
