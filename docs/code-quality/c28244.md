# C28244
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28244: Die Anmerkung für die Funktion verfügt über einen nicht analysierbaren Parameter\/eine externe Anmerkung.  
  
 Dies sollte nicht nur möglich sein, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
