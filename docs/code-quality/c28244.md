# <a name="c28244"></a>C28244
Warnung C28244: Die Anmerkung für die Funktion verfügt über einen nicht analysierbaren Parameter/eine externe Anmerkung.

 Dies sollte derzeit nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden. Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
