# <a name="c28246"></a>C28246
Warnung C28246: die Anmerkung für Funktion "\<Name >"-Parameter "\<Parameter >" entspricht nicht dem Typ des Parameters

 Ein __deref-Operator wurde in einen Zeigertyp angewendet, für die Erstellung eine Anmerkung.
