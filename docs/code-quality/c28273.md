# <a name="c28273"></a>C28273
Warnung C28273: Für Funktion sind die Hinweise inkonsistent mit der Funktionsdeklaration.

 Diese Warnung gibt an, ein Fehler in den Anmerkungen, nicht in den Code, der analysiert wird. Die Anmerkungen, die auf einer Funktionsdefinition angezeigt werden, sind inkonsistent mit denen auf eine Deklaration angezeigt werden. Die beiden Anmerkungen sollten entsprechend aufgelöst werden.
