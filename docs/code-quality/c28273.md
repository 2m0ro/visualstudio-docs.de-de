# <a name="c28273"></a>C28273
Warnung C28273: Für die Funktion ist der Hinweis inkonsistent mit der Funktionsdeklaration.

 Diese Warnung gibt an, ein Fehler in den Anmerkungen, nicht in den Code, der analysiert wird. Die Anmerkungen, die auf einer Funktionsdefinition angezeigt wurden, sind inkonsistent mit denen auf eine Deklaration angezeigt. Die beiden Anmerkungen sollten entsprechend aufgelöst werden.
