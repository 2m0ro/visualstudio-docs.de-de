# C28273
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28273: Für die Funktion ist der Hinweis inkonsistent mit der Funktionsdeklaration.  
  
 Diese Warnung gibt einen Fehler in den Anmerkungen, nicht im Code, die analysiert wird.  Mit Anmerkungen, die auf eine Funktionsdefinition werden, sind mit dem überein, die auf einer Deklaration werden.  Die beiden Anmerkungen sollten behoben werden, damit sie übereinstimmen.
