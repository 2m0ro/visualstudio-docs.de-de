# <a name="c28204"></a>C28204
Warnung C28204: \<Funktion >: einer der Werte dieser Überladung und das Schema bei \<Filename > (\<Zeile >) sind mit Anmerkungen versehen, für \<Paramname >: beide oder keines von beiden muss als Anmerkung.  
  
 Diese Warnung wird ausgegeben, wenn ein Fehler, in den Anmerkungen vorliegt.
