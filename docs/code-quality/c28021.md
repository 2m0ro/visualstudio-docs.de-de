# <a name="c28021"></a>C28021
Warnung C28021: der Parameter \<Param > versehende mit \<Anno > muss ein Zeiger sein  
  
 Diese Warnung wird ausgegeben, wenn das versehende Objekt nicht um ein Zeigertyp ist. Diese Anmerkung kann nicht verwendet werden, mit `void` oder ganzzahligen Typen.
