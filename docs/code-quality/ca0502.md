# <a name="ca0502"></a>CA0502
Nicht unterstützte Sprache

 Dieser Fehler tritt auf, wenn der Benutzer versucht, ein ASP.NET-Projekt zu analysieren, enthält mindestens eine weitere Seiten oder Klassen, die in einer anderen Sprache als C#- oder Visual Basic geschrieben wurde. Um diesen Fehler zu beheben, entfernen Sie die Datei aus dem Projekt, oder deaktivieren Sie der Codeanalyse.
