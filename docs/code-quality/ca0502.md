# <a name="ca0502"></a>CA0502
Nicht unterstützte Sprache

 Dieser Fehler tritt auf, wenn der Benutzer versucht, ein ASP.NET-Projekt zu analysieren, mindestens Seiten und/oder einer anderen Sprache als C#- oder Visual Basic-Klassen enthält. Um diesen Fehler zu beheben, entfernen Sie die Datei aus dem Projekt, oder deaktivieren Sie die Codeanalyse.
