# <a name="c28214"></a>C28214
Warnung C28214: Attributparameternamen müssen p1...p9 sein.  
  
 Diese Warnung gibt an, dass beim Erstellen einer Anmerkung Deklaration die Parameternamen beschränkt auf p1... p9. Diese Warnung sollte niemals während normaler Verwendung auftreten.
