# C28214
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28214: Attributparameternamen müssen p1...p9 sein.  
  
 Diese Warnung gibt an, dass, wenn Sie eine Anmerkungsdeklaration erstellen, die die Parameternamen in p1... p9 beschränkt werden.  Diese Warnung sollte nie in normaler Verwendung auftreten.
