# C28350
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28350: Die Anmerkung \<Anmerkung\> beschreibt eine Situation, die nicht bedingt anwendbar ist.  
  
 Normalerweise wird diese Warnung generiert, wenn eine Anmerkung angewendet wird, wobei der C\/C\+\+\-Typ überprüft wird.
