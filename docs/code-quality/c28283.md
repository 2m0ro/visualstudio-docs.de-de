# <a name="c28283"></a>C28283
Warnung C28283: Für das Symbol wird die festgelegte Größenangabe noch nicht unterstützt.  
  
 Die Warnung gibt an, dass die Anmerkung Größe-Spezifikation "Sentinel" einen anderen Wert als 0 (null) empfangen. Der Aufrufer versucht im Wesentlichen wird argumentiert, dass die Zeichenfolge durch ein anderes Zeichen als binäre 0 (null) beendet wird.
