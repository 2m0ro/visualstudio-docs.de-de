# C28283
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28283: Für das Symbol wird die festgelegte Größenangabe noch nicht unterstützt.  
  
 Die Warnung gibt an, dass die Größenspezifikations" Sentinel" Anmerkung einen anderen Wert als null hat.  Im Wesentlichen der Aufrufer versucht, zu identifizieren, dass die Zeichenfolge durch ein anderes Zeichen als Binärdatei null beendet wird.
