# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28230: Der Parametertyp weist keinen Member auf.  
  
 Diese Warnung gibt an, dass ein Argument einer Anmerkung versucht, auf `struct` zuzugreifen, `class` oder `union` und der benannte Member nicht vorhanden ist.
