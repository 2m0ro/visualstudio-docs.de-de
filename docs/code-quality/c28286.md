# C28286
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28286: Syntaxfehler am Ende der Funktion  
  
 Das Codeanalysetool meldet diese Warnung, wenn ein Fehler mit höherer Wahrscheinlichkeit in der Modelldatei erreicht wird.  Einige Quelldateifehler können derartige Fehler ebenfalls verursachen.
