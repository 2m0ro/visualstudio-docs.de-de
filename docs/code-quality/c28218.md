# C28218
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28218: Bei dem Funktionsparameter stimmt der Parameter für die Anmerkung nicht mit dem bei der Datei gefundenen Parameter überein.  
  
 Mit Anmerkungen auf der aktuellen Zeile und in der Zeile in der Meldung sind inkonsistent.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, während normaler Verwendung gesehen werden erwartet.
