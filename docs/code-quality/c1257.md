# <a name="c1257"></a>C1257
1257 Warnung: Fehler bei der Plug-In-Initialisierung.

 Der Code Analysis-Tool meldet diese Warnung aus, wenn es ist ein interner Fehler in das Plug-in, nicht in den Code analysiert wird.
