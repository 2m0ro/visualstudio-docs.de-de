# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28229: Anmerkung unterstützt keine Parameter.  
  
 Diese Warnung gibt an, dass eine Anmerkung mit einem Parameter verwendet wurde, wenn der Anmerkung ohne Parameter deklariert wird.  Dies sollte nicht möglich, wenn die Standardmakros für Anmerkungen verwendet werden; wird diese Warnung nicht, in der typischen Verwendung gesehen werden erwartet.
