# <a name="c28229"></a>C28229
Warnung C28229: Anmerkung unterstützt keine Parameter.  
  
 Diese Warnung gibt an, dass eine Anmerkung mit einem Parameter verwendet wurde, wenn die Anmerkung ohne Parameter deklariert wurde. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
