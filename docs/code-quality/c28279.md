# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28279: Für das Symbol wurde ein 'begin' ohne zugehöriges 'end' gefunden.  
  
 Die Anmerkungssprachunterstützung ein Alle und ein Ende \(`{` und `}` \) im C Konstrukt und Zuordnen hat unausgeglichenes abgerufen.  Dies kann vermieden werden, wenn die Makros verwendet werden.
