Warnung C28239: die Anmerkung zum Parameter wurden, sowohl Wert, StringValue oder LongValue und Paramn = Xxx. Verwenden Sie nur paramn=xxx.

 Interner Fehler in der Modelldatei PREfast. Diese Warnung sollte in der Regel nicht auftreten.
