# C28109
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28109: Variable kann nicht zum Zeitpunkt des Funktionsaufrufs enthalten sein.  
  
 Die Programmausf√ºhrung wird eine Ressource an, die nicht unterbrochen werden soll, wenn diese Funktion aufruft.  In der Regel gibt es an, dass die Ressource nicht zweimal abgerufen wurde.  Das Codeanalysetool meldet diese Warnung, wenn Ressourcen abgerufen und freigegeben in der gleichen Funktion Sie.  
  
## Beispiel  
 Im folgenden Code wird diese Warnung generiert:  
  
```

...

```
