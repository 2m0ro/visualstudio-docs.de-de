# C1251
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C1251: Die an Lastmodellen.  
  
 Das Codeanalysetool meldet diese Warnung, wenn einem internen Fehler in der Modelldatei gibt, nicht im Code, die analysiert wird.
