# <a name="c26464-useconstpointerforelements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  Die Werte, die auf Elemente des Arrays "% Array %" zeigt nur einmal aus, markieren Sie die Elemente als Zeiger auf zugewiesen `const`. Finden Sie unter [C++ Core Richtlinien con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
