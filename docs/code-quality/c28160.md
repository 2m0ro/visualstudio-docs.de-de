# <a name="c28160"></a>C28160
Warnung C28160: Fehleranmerkung  
  
 Diese Warnung wird ausgegeben, wenn eine `__drv_error` Anmerkung wurde gefunden. Diese Anmerkung wird verwendet, um Codierungstechniken zu kennzeichnen, die korrigiert werden soll, und kann verwendet werden, mit einem `__drv_when` -Anmerkung an, bestimmte Kombinationen von Parametern.
