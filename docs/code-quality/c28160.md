# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28160: Fehleranmerkung  
  
 Diese Warnung wird ausgegeben, wenn eine `__drv_error` Anmerkung aufgetreten ist.  Diese Anmerkungen Programmierpraktiken wird verwendet, um zu kennzeichnen, die korrigiert werden soll, und mit einem `__drv_when` Anmerkung verwendet werden, um bestimmte Kombinationen von Parametern anzugeben.
