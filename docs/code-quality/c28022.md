# <a name="c28022"></a>C28022
Warnung C28022: die Funktion Klasse(n) \<classlist1 > für diese Funktion entsprechen nicht die Funktion Klasse(n) \<classlist2 > auf die Typedef verwendet, um es zu definieren.

 Diese Warnung wird ausgegeben, wenn ein Fehler, in den Anmerkungen vorliegt. Die Typedef und die Funktion selbst verfügen `_Function_class_` Anmerkungen, aber sie stimmen nicht überein. Wenn beide verwendet werden, müssen sie übereinstimmen.
