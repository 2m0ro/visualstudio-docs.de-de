# <a name="c28224"></a>C28224
Warnung C28224: Anmerkung erfordert Parameter

 Diese Warnung gibt an, dass die benannten Anmerkung ohne Parameter verwendet wird und mindestens ein Parameter erforderlich ist. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, während normaler Verwendung betrachtet werden.
