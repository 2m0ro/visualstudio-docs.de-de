# <a name="ca0055"></a>CA0055
Datei konnte nicht geladen werden:\<Pfad >

 Das Codeanalysetool konnte die Zieldatei, die angegeben wurde für die Analyse geladen. Dies tritt auf, wenn die Datei nicht gefunden wurde oder der Zugriff auf die Datei wurde verweigert. In der Regel tritt dies auf, wenn der Benutzer möchte zum Überschreiben der Eigenschaft Eingabeassembly in [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], oder fxcopcmd.exe manuell aufgerufen.
