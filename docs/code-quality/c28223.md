# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28223: Hat nicht erwartet Token\/Bezeichner für Anmerkungen, Parameter  
  
 Diese Warnung gibt an, dass ein Parameter mit einer Anmerkung erwartet wird, um einen Bezeichner zu sein, und ein anderes Symbol wurde beendet.  Dies gibt normalerweise einem falsch codierten Anmerkungsmakro an.  Die Verwendung mit C oder C\+\+\-Schlüsselworten an dieser Position zu diesem Fehler.
