# <a name="c28223"></a>C28223
Warnung C28223: Fanden nicht Erwartetes Token/Bezeichner für Anmerkung, Parameter  
  
 Diese Warnung gibt an, dass ein Parameter in eine Anmerkung ein Bezeichner sein muss und einige andere Symbol wurde gefunden. Dies wird normalerweise ein Makro falsch codierten Anmerkung angegeben. Dieser Fehler wird die Verwendung von C oder C++-Schlüsselwörter in dieser Position verursacht werden.
