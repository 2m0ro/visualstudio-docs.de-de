# <a name="c6317"></a>C6317
Warnung C6317: Falscher Operator: Der logische Operator NOT (!) ist nicht mit dem Operator Einerkomplement (~) auswechselbar.

 Diese Warnung gibt an, dass ein logisches Not (`!`) eine Konstante, die wahrscheinlich ein Bitflag angewendet wird. Das Ergebnis der logischen Not-ist boolescher Wert; ist falsch, der bitweise anwenden â€“ und (`&`) Operator, um einen booleschen Wert. Verwenden Sie das Einerkomplement (`~`) Operator, um Flags zu kippen.

## <a name="example"></a>Beispiel
 Der folgende Code generiert diese Warnung:

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & !FLAGS) // warning
  {
    // code
  }
}
```

 So korrigieren Sie die Warnung unter Verwendung des folgenden Codes

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & ~FLAGS)
  {
    // code
  }
}
```
