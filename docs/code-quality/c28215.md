# <a name="c28215"></a>C28215
Warnung C28215: Der Typefix kann nicht auf einen Parameter angewendet werden, der bereits über einen Typefix verfügt.  
  
 Anwenden einer `__typefix` Anmerkung an einen Parameter, die bereits für diese Anmerkung ist ein Fehler. Die `__typefix` Anmerkungen werden nur in einigen besonderen Fällen verwendet, und diese Warnung ist nicht dazu gedacht, während normaler Verwendung angezeigt werden.
