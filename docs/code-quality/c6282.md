# C6282
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C6282: Falscher Operator: Zuweisung einer Konstanten im Boolean\-Kontext.  Verwenden Sie stattdessen "\=\=".  
  
 Diese Warnung gibt an, dass in einem Testkontext die Zuweisung einer Konstanten zu einer Variablen entdeckt wurde.  Die Zuweisung einer Konstanten zu einer Variablen in einem Testkontext ist fast immer ung√ºltig.  Ersetzen Sie `=` durch `==`, oder entfernen Sie die Zuweisung aus dem Testkontext, um diese Warnung zu vermeiden.  
  
## Beispiel  
 Der folgende Code generiert diese Warnung:  
  
```  
void f( int i )  
{  
   while (i = 5)  
   {  
   // code    
   }  
}  
```  
  
 So korrigieren Sie die Warnung unter Verwendung des folgenden Codes  
  
```  
void f( int i )  
{  
   while (i == 5)  
   {  
   // code    
   }  
}  
```
