# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28290: Die Anmerkung der Funktion enthält mehr Externe als die tatsächliche Anzahl von Parametern.  
  
 Das Codeanalysetool meldet diese Warnung, wenn die Anmerkung für die Funktion mehr als Externals die tatsächliche Anzahl von Parametern enthält.
