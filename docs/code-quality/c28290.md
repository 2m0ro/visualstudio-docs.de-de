# <a name="c28290"></a>C28290
Warnung C28290: Die Anmerkung der Funktion enthält mehr Externe als die tatsächliche Anzahl von Parametern.

 Der Code Analysis-Tool meldet diese Warnung aus, wenn die Anmerkung für die Funktion mehr externe als die tatsächliche Anzahl der Parameter enthält.
