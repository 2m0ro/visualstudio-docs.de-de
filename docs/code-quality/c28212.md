# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28212: Zeigerausdruck für Anmerkung erwartet  
  
 Diese Warnung gibt an, dass der Parameter zu einer nummerierten Anmerkung \(nicht die Funktion, die hinzugefügt wird\) erwartet wird, um einen Zeiger oder ein Arraytyp sein, sondern ein anderer Typ wurde beendet.  Die Anmerkung muss behoben werden.
