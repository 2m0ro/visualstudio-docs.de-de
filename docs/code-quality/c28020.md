# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28020: Der Ausdruck \<expr\> ist bei diesem Aufruf nicht "True".  
  
 Diese Warnung wird ausgegeben, wenn der \_Satisfies\_ Ausdruck, der aufgeführt ist, nicht erfüllt ist.  Häufig gibt dieses einem falschen Parameter an.  
  
 Wenn dies auf eine Funktionsdeklaration auftritt, die die Anmerkungen einer unmöglichen Zustand an.
