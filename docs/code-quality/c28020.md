# <a name="c28020"></a>C28020
Warnung C28020: der Ausdruck \<Expr > ist nicht an diesen Aufruf "true"

 Diese Warnung wird ausgegeben, wenn die _Satisfies\_ aufgeführten Ausdruck ist nicht "true". Häufig bedeutet dies einen falschen Parameter.

 In diesem Fall in einer Funktionsdeklaration Geben Sie die Anmerkungen eine unmöglich Bedingung an.
