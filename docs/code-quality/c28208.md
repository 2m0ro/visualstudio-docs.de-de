# C28208
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28208: Funktion \<Funktion\> wurde zuvor mit einer anderen Parameterliste bei \<Datei\>\(\<Zeile\>\) definiert.  Manche Analysetools können falsche Ergebnisse liefern.  
  
 Diese Warnung wird ausgegeben, wenn die bekannte Definition einer Funktion kein anderes Vorkommen übereinstimmt.
