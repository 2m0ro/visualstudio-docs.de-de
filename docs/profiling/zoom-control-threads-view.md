# Zoomsteuerelement (Threadansicht)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Bei dem Zoomsteuerelement handelt es sich um einen Schieberegler, mit dem Sie eine Zeitachse vergrößern und verkleinern und so Bereiche von besonderem Interesse genauer betrachten können.  Da dieses Steuerelement den Mittelpunkt der Zeitachsenansicht vergrößert, sollten Sie den jeweils relevanten Bereich zentrieren, bevor Sie die Ansicht vergrößern.  
  
## Vergrößern der Ansicht durch Ziehen in der Zeitachsenansicht  
 Wenn Sie die Ansicht durch Ziehen in der Zeitachsenansicht vergrößern, wird ein Bereich erstellt, der Gelb hervorgehoben wird.  Wenn Sie die Maustaste loslassen, wird die Zeitachsenansicht mit dem ausgewählten Bereich vergrößert.  
  
## Vergrößern und Verkleinern der Ansicht mit dem Mausrad  
 Klicken Sie auf einen beliebigen Punkt auf der Zeitachse, um sicherzustellen, dass der Mausfokus auf diesen Punkt festgelegt ist. Drücken Sie anschließend STRG, und drehen Sie das Mausrad \(vorwärts: Ansicht wird vergrößert; rückwärts: Ansicht wird verkleinert\).
