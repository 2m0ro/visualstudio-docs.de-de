# Zoomsteuerelement (Auslastungsansicht)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das Zoomsteuerelement ermöglicht, auf das Vergrößern zu vergrößern, um auf Bereiche von besonderem Interesse zu konzentrieren.  Dieses Steuerelement vergrößert die Mitte der Ansicht.  Daher sollten Sie den gewünschten Bereich zentrieren, bevor Sie die Ansicht vergrößern.  
  
 Sie können den Mauszeiger im CPU\-Auslastungsdiagramm oder im GPU\-Handlungsübersicht ziehen, um einen markierten Bereich zu erstellen.  Wenn die Maustaste losgelassen wird, vergrößert die Ansicht in den ausgewählten Bereich.
