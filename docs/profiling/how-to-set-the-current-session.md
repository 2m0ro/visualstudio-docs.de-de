# Gewusst wie: Festlegen der aktuellen Profilerstellungssitzung
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie können im **Leistungs\-Explorer** mehrere Leistungssitzungen öffnen und dann auswählen, dass eine der sichtbaren Leistungssitzungen als aktuelle Sitzung festgelegt wird.  
  
 **Voraussetzungen**  
  
-   [!INCLUDE[vsUltLong](../code-quality/includes/vsultlong_md.md)], [!INCLUDE[vsPreLong](../code-quality/includes/vsprelong_md.md)], [!INCLUDE[vsPro](../code-quality/includes/vspro_md.md)]  
  
 Nachdem Sie eine Sitzung als aktuelle Sitzung festgelegt haben, wird durch Klicken auf die Schaltfläche **Starten** auf der Symbolleiste **Leistungs\-Explorer** die Profilerstellung für die aktuelle Sitzung gestartet.  
  
### So legen Sie die aktuelle Leistungssitzung fest  
  
-   Klicken Sie im **Leistungs\-Explorer** mit der rechten Maustaste auf die Leistungssitzung, und klicken Sie dann auf **Als aktuelle Sitzung festlegen**.  
  
     Der Sitzungsname wird fett formatiert angezeigt.
