# Es sind mehrere Kategorien vorhanden
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Es wird ein Abschnitt ausgewählt, der mehr als einen Aktivitätstyp für einen bestimmten Zeitraum enthält.  Beachten Sie, dass zwei Farben vertikal gestapelt angezeigt werden.  Diese Farben stellen nur die zwei häufigsten Aktivitäten im jeweiligen Zeitraum dar.  Die Häufigkeiten dieser zwei Aktivitäten werden relativ zueinander als Drittel oder Hälften des senkrechten Balkens angezeigt.  Wenn Sie mehr Details anzeigen möchten, vergrößern Sie diesen Abschnitt der Threadansicht.
