# Zusammenfassungsbericht pro Thread
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Dieses Balkendiagramm zeigt den Anteil der Zeit, die jeder eingeblendeten Threads in jeder Aktivitätskategorie während des derzeit sichtbaren Zeitraum aufwendet. "Ausführung" bedeutet, dass der Thread ausführt; Alle anderen Kategorien bedeuten, dass der Thread auf etwas wartet.
