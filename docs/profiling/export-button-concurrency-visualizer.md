# Exportieren (Schaltfl&#228;che in Parallelit&#228;tsschnellansicht)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Die Schaltfläche **Exportieren** ermöglicht es Ihnen, die Aufrufliste als CSV\-Datei für Ihre eigenen Datensätze oder zur Verwendung in einem anderen Tool \(beispielsweise Microsoft Excel\) zu exportieren.
