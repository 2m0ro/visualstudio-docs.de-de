# Es wird keine Datentr&#228;geraktivit&#228;t angezeigt (Threadansicht)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Die Datenträgeraktivität für den aktuellen Zeitraum in den Datenträger\-E\/A\-Channels kann nicht angezeigt werden.  Die Datenträgeraktivität wird möglicherweise angezeigt, wenn Sie die Ansicht verkleinern oder einen Bildlauf durchführen.  Stellen Sie außerdem sicher, dass keine Datenträgerchannels ausgeblendet wurden.
