# Sichtbares Zeitachsenprofil
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das sichtbaren Zeitachsenprofils für die Threadblockierungsansicht stellt statistische Informationen und Links zu Berichten.  Wenn Sie die Ansicht vergrößern oder verkleinern, einen horizontalen Bildlauf durchführen oder Channels ausblenden oder anzeigen, ändern sich die Zahlen in der aktiven Legende entsprechend den gegenwärtig angezeigten Elementen.  Um in der Legende einen Bericht zu einem Element anzuzeigen, klicken Sie auf das Element.
