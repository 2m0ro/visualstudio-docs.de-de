# Meldungsmarker
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Ein Meldungsmarker stellt Protokollausgabe dar.  Eine Meldung ist eine Zeichenfolge, die durch einen bestimmten Thread zu einem bestimmten Zeitpunkt angezeigt wird.  Sie können Meldungen in eine Textdatei mit anderen Tools exportiert.  Sie können den Mauszeiger auf eine Meldung in der Parallelitätsschnellansicht zeigen, um die Meldungszeichenfolge anzuzeigen.  Außerdem können Sie alle Meldungsmarker im [Markerbericht](../profiling/markers-report.md) anzeigen.  Die folgende Abbildung zeigt einen Meldungsmarker an.  
  
## Meldungs\-Aggregations\-Marker  
 Manchmal mehrere Nachrichten treten auf, das Sie schließen beieinander in der Parallelitätsschnellansicht, dass sie nicht einzeln gezeichnet werden können.  Wenn dies auftritt, wird ein *Meldungsaggregationsmarker*, der die zugrunde liegenden Meldungen darstellt, angezeigt.  Wenn Sie den Mauszeiger auf einem dieser Symbole zeigen, Anzeigen einer QuickInfo die Anzahl der Nachrichten zugrunde liegenden, die dargestellt werden.  Um die Nachrichten anzeigen, Vergrößern Sie. Beim Einchecken ähnlich Vergrößern und weiterhin einen Aggregationsmarker erhalten, können Sie die zugrunde liegenden Meldungen im [Markerbericht](../profiling/markers-report.md) anzeigen.
