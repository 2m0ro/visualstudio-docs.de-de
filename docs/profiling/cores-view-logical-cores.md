# Logische Kerne in der Kernansicht
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Jeder Channelname stellt einen der im profilierten System vorhandenen logischen Kerne dar.  Bei der Benennung wird von 0 bis zur Gesamtzahl der logischen Kerne gezählt.  In einem Vierkernsystem wird z. B. vom logischem Kern 0 bis zum logischem Kern 3 gezählt.
