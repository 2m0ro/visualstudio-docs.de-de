# Referenz zu Profiler-APIs in Visual&#160;Studio (systemeigen)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Über die Profiler\-APIs in Visual Studio können Sie programmgesteuert die erfasste Datenmenge steuern und während der Profilerstellung sowohl einen Timestampwert als auch Profilmarkierungen einfügen.  Zur Verwendung systemeigener APIs nehmen Sie die Headerdatei VSPerf.h in das Projekt auf und fügen die Bibliothek VSPerf.lib hinzu.  
  
> [!NOTE]
>  Standardmäßig befinden sich VSPerf.h und VSPerf.lib in \<Laufwerk\>:\\Programme\\Microsoft Visual Studio 9\\Team Tools\\Performance Tools\\PerfSDK.
