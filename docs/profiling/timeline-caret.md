# Zeitachsen-Caretzeichen
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Wenn Sie einen Punkt auf der Zeitachse eines ausgeführten Threadsegments auswählen, wird eine Zeitachseneinfügemarke Berechtigungsebene.  Auf der aktuellen Stapelregisterkarte wird die Aufrufliste angezeigt, die dem Klicken auf das Segment zeitlich am nächsten liegt.  Die Einfügemarke wird verwendet, um die Aufrufliste, die unter der Registerkarte **Aktuell** angezeigt wird, dem Zeitpunkt zuzuordnen, als sie nachweislich wurde.  Die Einfügemarke wird die genaue Position der Aufrufliste angezeigt, die die größte Aufrufliste zum Speicherort ist, den der Benutzer ausgewählt hat.
