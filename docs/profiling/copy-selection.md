# Auswahl kopieren
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Um die ganze Aufrufliste von der Berichtsregisterkarte zu kopieren, klicken Sie auf **Kopieren**.  Sie können die Aufrufliste dann in jedes Programm einfügen, das diese Aktion unterstützt.
