# Zoomtool in der Kernansicht
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Bei dem Zoom\-Steuerelement handelt es sich um einen Schieberegler, mit dem Sie eine Zeitachse vergrößern und verkleinern und so Bereiche von besonderem Interesse genauer betrachten können.  Dieses Steuerelement zoomt auf dem Mittelpunkt der aktuellen Ansicht, passen Sie daher vor dem Zoomen die Ansicht an.  
  
## Andere Möglichkeiten für das Zoomen  
  
-   Vergrößern der Ansicht: Markieren Sie einen Bereich in der Zeitachsenansicht durch Ziehen.  Wenn Sie die Maustaste loslassen, wird die Zeitachsenansicht mit dem ausgewählten Bereich vergrößert.  
  
-   Vergrößern und Verkleinern der Ansicht: Zeigen Sie auf die Zeitachse, und drücken Sie dann die STRG\-TASTE, während Sie das Mausrad bewegen.
