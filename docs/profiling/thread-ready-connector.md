# Threadbereiter Konnektor
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Wenn Sie auf ein blockierendes Segment klicken, um eine Aufrufliste und den zugehörigen die Blockierung aufhebenden Stapel anzuzeigen, wird der threadbereite Connector möglicherweise auch angezeigt.  Wenn das Ereignis, das die Blockierung aufhebt, in einem anderen Thread im aktuellen Prozess eingetreten ist, identifiziert der threadbereite Connector visuell den Thread und das Ausführungssegment, die es dem blockierten Thread ermöglicht haben, die Ausführung fortzusetzen.
