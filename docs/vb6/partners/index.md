# <a name="visual-basic-60-offers-from-partners"></a>Visual Basic 6.0-Angebote von Partnern

In diesem Thema finden Sie Informationen zu kostenlosen und kostenpflichtigen Tools und Lösungen unserer Partner, die Sie bei einer umfassenderen Migration von Visual Basic 6.0 zu Visual Basic .NET unterstützen.

<table>
  <tr>
    <td><img src="media/vbuc.png" alt="Mobilize.Net's Visual Basic Upgrade Companion (VBUC)" width="100" /> </td>  
    <td><strong><a href="mobilize-net.md">Visual Basic Upgrade Companion (VBUC) von Mobilize.Net</a></strong><p> 
Der Visual Basic Upgrade Companion (VBUC) von Mobilize.Net ist ein branchenführendes Tool für die Migration von VB6 zu .NET. Mit einer nachweislichen Erfolgsbilanz von Millionen von Zeilen, die zu C# und VB.NET migriert wurden, hilft Ihnen der VBUC dabei, dass Ihre geschäftskritischen Anwendungen nicht veralten, und zwar ohne Bindung an Abhängigkeiten von Drittanbietern. Eine Version dieses Tools steht Ihnen kostenlos zur Verfügung.</td>
  </tr>
  <tr>
    <td><img src="media/gmstudio.png" alt="Great Migrations Studio (gmStudio)" width="100" /> </td>
    <td><strong><a href="gmstudio.md">Great Migrations Studio (gmStudio)</a></strong></p> 
gmStudio ist eine außergewöhnliche Systemoptimierungsumgebung für die Entwicklung skalierbarer, kundenspezifischer Lösungen für das Upgrade von VB6/ASP/COM auf. NET.   Als umfassende Softwareoptimierungsplattform unterstützt gmStudio Sie bei der Planung, Anpassung, Verbesserung, Überprüfung und Nachverfolgung Ihres Upgradeprozesses und erfüllt die besonderen Anforderungen Ihres Quellcodes und Ihrer .NET-Standards.  Wenn Sie Ihre VB6-/ASP-/COM-Ressourcen erhalten und verbessern möchten, ist gmStudio die fortschrittlichste und wirtschaftlichste Lösung auf dem Markt. </td> 
  </tr>

    <td><img src="media/migrationvb.jpg" alt="VB Migration Partner tool" width="100" /></td>
    <td><strong><a href="http://vbmigration.com">VB Migration Partner-Tool</a></strong></p>VB Migration Partner ist eine innovative VB6-Konvertierungssoftware, die von einem Team renommierter VB- und. NET-Experten unter Leitung von VB-Guru Francesco Balena entwickelt wurde und Ihnen hilft, VB6-Anwendungen zu VB .NET oder C# zu portieren. Das Tool unterstützt praktisch sämtliche VB6-Funktionen und -Steuerelemente und generiert .NET-Anwendungen mit weniger Fehlern und in kürzerer Zeit als andere Konvertierungstools. VB Migration Partner setzt auf ein ausgeklügeltes Konvertierungsmodul, das mit Konstrukten zurechtkommt, für die es in der .NET Framework-Welt kein direktes Äquivalent gibt. Das Codegenerierungsmodul und leistungsfähige Refactoringmodul geben .NET-Code aus, der mit von erfahrenen Entwicklern geschriebenem Code qualitativ mithalten kann. Darüber hinaus können Entwickler den generierten Code anpassen und das Verhalten der konvertierten Anwendung verbessern, während sie ihn dank der Zyklusmethode „Konvertieren-Testen-Korrigieren“ mit dem ursprünglichen VB6-Projekt synchron halten.</td>
