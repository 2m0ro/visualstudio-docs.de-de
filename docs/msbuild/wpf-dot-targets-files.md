# WPF .Targets Files
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

[!INCLUDE[TLA#tla_winclient](../misc/includes/tlasharptla_winclient_md.md)] erweitert [!INCLUDE[TLA#tla_msbuild](../msbuild/includes/tlasharptla_msbuild_md.md)], indem ein Satz [!INCLUDE[TLA2#tla_wpf](../msbuild/includes/tla2sharptla_wpf_md.md)]\-spezifischer Aufgaben hinzugefügt wird. Diese werden in einer speziellen TARGETS\-Datei, **Microsoft.WinFX.targets**, kombiniert.  In dieser Datei wird der Satz der [!INCLUDE[TLA2#tla_msbuild](../msbuild/includes/tla2sharptla_msbuild_md.md)]\-Aufgaben zusammengefasst, die zum Erstellen eines [!INCLUDE[TLA2#tla_msbuild](../msbuild/includes/tla2sharptla_msbuild_md.md)]\-Projekts in [!INCLUDE[TLA#tla_winclient](../misc/includes/tlasharptla_winclient_md.md)] erforderlich sind.
