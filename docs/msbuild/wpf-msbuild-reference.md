# WPF MSBuild Reference
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Windows Presentation Foundation \(WPF\) erweitert das Microsoft\-Buildmodul \(MSBuild\) um zusätzliche, in diesem Abschnitt dokumentierte Buildunterstützung.  
  
## In diesem Abschnitt  
 [.Targets Files](../msbuild/wpf-dot-targets-files.md)  
 Beschreibt WPF\-TARGETS\-Dateien.  
  
 [Task Reference](../msbuild/wpf-msbuild-task-reference.md)  
 Listet die verfügbaren WPF\-Buildaufgaben auf.  
  
 <xref:Microsoft.Build.Tasks>  
 Eine Buildaufgabenassembly.  
  
 <xref:Microsoft.Build.Tasks.Deployment.Bootstrapper>  
 Eine Bereitstellungs\-Bootstrapper\-Assembly für eine Buildaufgabe.  
  
 <xref:Microsoft.Build.Tasks.Deployment.ManifestUtilities>  
 Eine Bereitstellungsmanifest\-Dienstprogrammassembly für eine Buildaufgabe.  
  
 <xref:Microsoft.Build.Tasks.Hosting>  
 Eine Buildaufgabenhostingassembly.  
  
 <xref:Microsoft.Build.Tasks.Windows>  
 Eine Buildaufgabenfensterassembly.
