# Using the Visual C# Development Environment
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Die integrierte Entwicklungsumgebung \(Integrated Development Environment, IDE\) von Visual C\# setzt sich aus einer Reihe von Entwicklungstools zusammen, die über eine gemeinsame Benutzeroberfläche verfügbar gemacht werden.  Einige der Tools werden auch von anderen [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]\-Programmiersprachen verwendet. Andere, z. B. den C\#\-Compiler, gibt es nur in Visual C\#.  Dieses Thema enthält Links zu den wichtigsten Visual C\#\-Tools.
