# Befehle im Men&#252; &quot;Fenster&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das Menü **Fenster** enthält Befehle zum Verwalten von Spy\+\+\-Fenstern.  
  
 **Überlappend**  
 Ordnet alle Fenster überlappend an.  
  
 **Untereinander**  
 Ordnet alle Fenster untereinander an.  
  
 **Nebeneinander**  
 Ordnet alle Fenster nebeneinander an.  
  
 **Symbole anordnen**  
 Ordnet die Ansichtssymbole neu an, wenn Fenster minimiert wurden.  
  
 **Schließen\/Alle schließen**  
 Schließt ein Fenster oder alle Fenster in Spy\+\+.  
  
 **Aktualisieren**  
 Aktualisiert die gegenwärtig ausgewählte Ansicht.  
  
 **Fenster**  
 Ermöglicht die Auswahl eines bestimmten Fensters.
