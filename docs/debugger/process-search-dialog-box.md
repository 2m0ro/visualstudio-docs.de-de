# Dialogfeld &quot;Prozesssuche&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Verwenden Sie dieses Dialogfeld, um den Knoten für einen bestimmten Prozess in der [Prozessansicht](../debugger/processes-view.md) zu suchen und auszuwählen.  Um dieses Dialogfeld anzuzeigen, verschieben Sie den Fokus auf ein **Prozessansichtsfenster**.  Klicken Sie dann im Menü **Suchen** auf **Prozess suchen**.  
  
 Die folgenden Einstellungen sind verfügbar:  
  
 **Prozess**  
 Die ID des zu suchenden Prozesses.  
  
 **Modul**  
 Die zu suchende Modulzeichenfolge.  
  
 **Suchrichtung: Nach oben\/Nach unten**  
 Die anfängliche Richtung der Suche.
