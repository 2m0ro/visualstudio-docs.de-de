# <a name="how-to-display-window-properties"></a>Gewusst wie: Anzeigen von Fenstereigenschaften
So ermitteln Sie weitere Informationen zu einem Fenster aufgelistet, die [Fensteransicht](../debugger/windows-view.md), verwenden Sie die [Fenstereigenschaften (Dialogfeld)](../debugger/window-properties-dialog-box.md).  
  
### <a name="to-open-the-window-properties-dialog-box"></a>Öffnen Sie das Dialogfeld "Fenstereigenschaften"  
  
1.  Wählen Sie das Element aus, und wählen Sie dann **Eigenschaften** aus der **Ansicht** Menü.  
  
 Eigenschaftendialogfelder sind nicht modal. Wenn Sie im Fenster "ein" von Element zu Element klicken, wird das Dialogfeld mit Informationen zum jeweils ausgewählten Element automatisch aktualisiert.
