# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset
Setzt eine Enumerationsfolge auf den Anfang zurück.  
  
## <a name="syntax"></a>Syntax  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Gibt `S_OK`zurück.
