# IDiaEnumFrameData::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Setzt die Enumerationsfolge auf den Anfang zurück.  
  
## Syntax  
  
```  
HRESULT Reset();  
```  
  
## Rückgabewert  
 Gibt `S_OK` zurück.
