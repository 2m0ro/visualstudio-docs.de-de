# <a name="idiasymbolgetconstructor"></a>IDiaSymbol::get_constructor
Ruft ein Flag, das angibt, ob der benutzerdefinierte Datentyp einen Konstruktor oder Destruktor verfügt.  
  
## <a name="syntax"></a>Syntax  
  
```C++  
HRESULT get_constructor (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pRetVal`  
 [out] Gibt `TRUE` , wenn der benutzerdefinierte Datentyp einen Konstruktor oder Destruktor; besitzt, andernfalls `FALSE`.  
  
## <a name="return-value"></a>Rückgabewert  
 Im Erfolgsfall gibt `S_OK`ist, andernfalls gibt `S_FALSE` oder ein Fehlercode.  
  
> [!NOTE]
>  Ein Rückgabewert von `S_FALSE` bedeutet, dass die Eigenschaft nicht für das Symbol verfügbar ist.
