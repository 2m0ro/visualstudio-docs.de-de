# <a name="idiasymbolgetdatabytes"></a>IDiaSymbol::get_dataBytes
Ruft die Datenbytes, die von einem OEM-Symbol ab.  
  
## <a name="syntax"></a>Syntax  
  
```C++  
HRESULT get_dataBytes (   
   DWORD  cbData,  
   DWORD* pcbData,  
   BYTE   data[]  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `cbData`  
 [in] Die Größe des Puffers zum Speichern der Daten.  
  
 `pcbData`  
 [out] Gibt die Anzahl der Bytes, die geschrieben wird, oder, wenn die `data` Parameter ist `NULL`, gibt die Anzahl der verfügbaren Bytes.  
  
 `data[]`  
 [out] Ein Puffer, der mit der Datenbytes ausgefüllt ist.  
  
## <a name="return-value"></a>Rückgabewert  
 Im Erfolgsfall gibt `S_OK`ist, andernfalls gibt `S_FALSE` oder ein Fehlercode.  
  
> [!NOTE]
>  Ein Rückgabewert von `S_FALSE` bedeutet, dass die Eigenschaft nicht für das Symbol verfügbar ist.
