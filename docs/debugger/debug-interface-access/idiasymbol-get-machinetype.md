```C++  
   DWORD* pRetVal

```

 [out] Gibt einen Wert aus der [CV_CPU_TYPE_e-Enumeration](../../debugger/debug-interface-access/cv-cpu-type-e.md) -Enumeration, die das Ziel-CPU-Typ angibt.
