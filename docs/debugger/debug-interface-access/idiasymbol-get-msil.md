# <a name="idiasymbolgetmsil"></a>IDiaSymbol::get_msil

## <a name="syntax"></a>Syntax  
  
```C++  
HRESULT get_msil (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pRetVal`  
 [out] Gibt `TRUE` , wenn das Symbol auf MSIL-Code verweist, andernfalls `FALSE`.
