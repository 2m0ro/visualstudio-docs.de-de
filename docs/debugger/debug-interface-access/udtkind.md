# <a name="udtkind"></a>UdtKind
Beschreibt die unterschiedlichen den benutzerdefinierten Typ (UDT).  
  
## <a name="syntax"></a>Syntax  
  
```C++  
enum UdtKind {   
   UdtStruct,  
   UdtClass,  
   UdtUnion,  
   UdtInterface  
};  
```  
  
## <a name="elements"></a>Elements  
 UdtStruct  
 UDT ist eine Struktur.  
  
 UdtClass  
 UDT ist eine Klasse.  
  
 UdtUnion  
 UDT ist eine Union.  
  
 UdtInterface  
 UDT ist eine Schnittstelle.
