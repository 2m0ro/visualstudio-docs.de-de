# Befehle im Men&#252; &quot;Struktur&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das Menü **Struktur** stellt Befehle zum Erweitern und Reduzieren von Ansichten bereit.  
  
 **Eine Ebene erweitern**  
 Erweitert das gegenwärtig ausgewählte Element in einer Systemansicht, indem die Ebene unter dem Element angezeigt wird.  
  
 **Untergeordnete Ebenen erweitern**  
 Erweitert alle Ebenen unter dem gegenwärtig ausgewählten Element.  
  
 **Alle erweitern**  
 Erweitert alle Ebenen in einer Systemansicht und zeigt diese an.  
  
 **Reduzieren**  
 Reduziert alle Ebenen unter dem gegenwärtig ausgewählten Element.
