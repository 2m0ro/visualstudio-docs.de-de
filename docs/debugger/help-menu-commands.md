# Befehle im Men&#252; &quot;Hilfe&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Im Menü **Hilfe** finden Sie Befehle zum Öffnen der Hilfedateien.  
  
 **Index**  
 Öffnet das Inhaltsfenster der Spy\+\+\-Hilfe.  
  
 **Verwenden der Hilfe**  
 Öffnet eine Hilfedatei, in der die Verwendung der Hilfe beschrieben ist.  
  
 **Informationen zu Spy\+\+**  
 Zeigt die Softwareversion und die Copyrightinformationen für Spy\+\+ an.
