# How to: Expand and Collapse Spy++ Trees
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie können die Fenster\-, Prozess\- und Threadansichten mit zwei Verfahren erweitern und reduzieren: durch Klicken auf die Symbole im Fenster oder mithilfe des Menüs **Struktur**.  Die Symbole \+ und \- in der Struktur weisen die gleichen Funktionen wie im Visual C\+\+\-Projektfenster auf.  
  
 Das Menü **Struktur** enthält vier Befehle:  
  
|Menübefehl|Beschreibung|  
|----------------|------------------|  
|**Eine Ebene erweitern**|Erweitert das gegenwärtig ausgewählte Element auf die nächste Ebene.|  
|**Untergeordnete Ebenen erweitern**|Erweitert das gegenwärtig ausgewählte Element vollständig.|  
|**Alle erweitern**|Erweitert alle Elemente im Fenster vollständig.|  
|**Reduzieren**|Reduziert das gegenwärtig ausgewählte Element vollständig.|  
  
> [!TIP]
>  Wenn Sie einen Prozess erweitern, werden alle Threads angezeigt, die im Besitz des Prozesses sind.  Wenn Sie einen Thread erweitern, wird eine Liste aller Fenster angezeigt, die im Besitz des Threads sind.  
  
### So erweitern oder reduzieren Sie Spy\+\+\-Strukturen  
  
1.  Heben Sie eines der Elemente in der Fenster\-, Prozess\- oder Threadansicht hervor.  
  
2.  Wählen Sie im Menü **Struktur** den Befehl zum Erweitern oder Reduzieren aus.
