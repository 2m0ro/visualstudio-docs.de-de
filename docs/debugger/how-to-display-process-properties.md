# How to: Display Process Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Verwenden Sie das Dialogfeld [Prozesseigenschaften](../debugger/process-properties-dialog-box.md), um Informationen über einen in der [Prozessansicht](../debugger/processes-view.md) aufgeführten Prozess zu erhalten.  
  
### So öffnen Sie das Dialogfeld "Prozesseigenschaften"  
  
1.  Wählen Sie in der Struktur, die in der **Prozessansicht** angezeigt wird, einen Prozess aus.  
  
2.  Klicken Sie im Menü **Ansicht** auf **Eigenschaften**.  
  
 Eigenschaftendialogfelder sind nicht modal, sodass Sie in einem Ansichtsfenster auf ein anderes Element klicken können und im Dialogfeld Informationen zum ausgewählten Element angezeigt werden.
