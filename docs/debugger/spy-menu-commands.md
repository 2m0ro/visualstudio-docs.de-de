# Befehle im Men&#252; &quot;Spy&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das Menü **Spy** stellt Befehle zum Arbeiten mit Spy\+\+\-Fenstern und \-Ansichten bereit.  
  
 **Windows**  
 Öffnet eine neue Fensteransicht.  
  
 **Prozesse**  
 Öffnet eine neue Prozessansicht.  
  
 **Threads**  
 Öffnet eine neue Threadansicht.  
  
 **Meldungen protokollieren**  
 Öffnet eine neue Meldungsansicht und öffnet das Dialogfeld [Meldungsoptionen](../debugger/message-options-dialog-box.md) für die Ansicht.  
  
 **Fenster suchen**  
 Zeigt das Dialogfeld **Fenster suchen** an, um ein Fenster mit der Maus auszuwählen.  
  
 **Exit**  
 Schließt das Hilfsprogramm Spy\+\+.
