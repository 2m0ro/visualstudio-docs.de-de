# Befehle im Men&#252; &quot;Suchen&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das Menü **Suchen** stellt Befehle zum Suchen nach bestimmten Fenstern bereit.  
  
 **Fenster\/Prozess\/Thread\/Meldung suchen**  
 Öffnet das Dialogfeld **Fenster**\/**Prozess**\/**Thread**\/**Meldungssuche**, in dem Sie das **Suchtool** verwenden oder Suchkriterien angeben können.  
  
 **Weitersuchen**  
 Sucht in der aktuellen Ansicht das nächste entsprechende Fenster, den nächsten entsprechenden Prozess, den nächsten entsprechenden Thread bzw. die nächste entsprechende Meldung.  Dieses Menüelement \(und die zugehörige Symbolleistenschaltfläche\) ist nur verfügbar, wenn ein gültiges Suchergebnis nicht eindeutig ist.  
  
 **Vorheriges suchen**  
 Sucht in der aktuellen Ansicht das vorherige entsprechende Fenster, den vorherigen entsprechenden Prozess, den vorherigen entsprechenden Thread bzw. die vorherige entsprechende Meldung.  Dieses Menüelement \(und die zugehörige Symbolleistenschaltfläche\) ist nur verfügbar, wenn ein gültiges Suchergebnis nicht eindeutig ist.
