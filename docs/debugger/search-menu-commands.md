# <a name="search-menu-commands"></a>Befehle im Menü "Suchen"
Die **Suche** Menü enthält Befehle zum Suchen nach bestimmten Fenstern.  
  
 **Fenster/Prozess/Thread/Meldung**  
 Öffnet die **Fenster**/**Prozess**/**Thread**/**Meldungssuche** (Dialogfeld), dort können Sie verwenden die **Suchtool** , oder geben Sie die Suchkriterien.  
  
 **Weitersuchen**  
 Durchsucht die aktuelle Ansicht für den nächsten übereinstimmenden Fenster, Prozess, Thread oder Nachricht. Dieses Menüelement (und die zugehörige Symbolleistenschaltfläche) ist verfügbar, nur, wenn ein gültiges Suchergebnis, das nicht eindeutig ist.  
  
 **Vorheriges suchen**  
 Durchsucht die aktuelle Ansicht für die vorherigen übereinstimmenden Fenster, Prozess, Thread oder Nachricht. Dieses Menüelement (und die zugehörige Symbolleistenschaltfläche) ist verfügbar, nur, wenn ein gültiges Suchergebnis, das nicht eindeutig ist.
