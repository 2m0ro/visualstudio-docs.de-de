# How to: Start Spy++
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie können Spy\+\+ über Visual Studio oder eine Eingabeaufforderung starten.  
  
 Wenn Sie Spy\+\+ starten und eine Meldung angezeigt wird, die eine Berechtigung zum Durchführen der Änderung anfordert, klicken Sie auf **Ja**.  
  
> [!NOTE]
>  Sie können nur eine einzige Instanz von Spy\+\+ ausführen.  Wenn Sie versuchen, eine andere Instanz auszuführen, wird dadurch lediglich der aktuellen Instanz der Fokus zugewiesen.  
  
### So starten Sie Spy\+\+ über Visual Studio  
  
-   Klicken Sie im Menü **Extras** auf **Spy\+\+**.  
  
     Da Spy\+\+ unabhängig von Visual Studio ausgeführt wird, können Sie Visual Studio schließen.  
  
    > [!NOTE]
    >  Wenn Sie Nachrichten mit Spy\+\+ protokollieren, kann dies dazu führen, dass das Betriebssystem langsamer ausgeführt wird.  
  
### So starten Sie Spy\+\+ über eine Eingabeaufforderung  
  
1.  Ändern Sie in einem Eingabeaufforderungsfenster das Verzeichnis in den Ordner mit spyxx.exe.  In der Regel befindet sich der Ordner unter dem folgenden Pfad:  \\*Installationsordner von Visual Studio*\\Common7\\Tools\\.  
  
2.  Geben Sie **spyxx.exe** ein, und drücken Sie dann die EINGABETASTE.
