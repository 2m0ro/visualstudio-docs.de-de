# Using Spy++
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ \(SPYXX.EXE\) ist ein Win32\-basiertes Hilfsprogramm, das Ihnen eine grafische Ansicht der Prozesse, Threads, Fenster und Fenstermeldungen des Systems bietet.  Dieser Abschnitt der Spy\+\+\-Hilfe enthält die folgenden Hilfethemen:  
  
## In diesem Abschnitt  
 [Einführung in Spy\+\+](../debugger/introducing-spy-increment.md)  
 Erklärt die Typen von Informationen, die Sie mit dem Tool Spy\+\+ anzeigen können.  
  
 [How to: Start Spy\+\+](../debugger/how-to-start-spy-increment.md)  
 Erklärt, wie Spy\+\+ im Menü "Extras" gestartet wird.  
  
 [Spy\+\+ Toolbar](../debugger/spy-increment-toolbar.md)  
 Beschreibt die auf der Symbolleiste von Spy\+\+ verfügbaren Befehle.  
  
 [How to: Display Spy\+\+ Views](../debugger/how-to-display-spy-increment-views.md)  
 Beschreibt, wie Fensteransicht, Meldungsansicht, Prozessansicht und Threadansicht angezeigt werden.  
  
 [How to: Refresh the View](../debugger/how-to-refresh-the-view.md)  
 Erläutert den Befehl "Aktualisieren".  
  
 [How to: Change Fonts](../debugger/how-to-change-fonts.md)  
 Erklärt, wie die Anzeige einer Ansicht geändert wird.  
  
 [How to: Expand and Collapse Spy\+\+ Trees](../debugger/how-to-expand-and-collapse-spy-increment-trees.md)  
 Erklärt, wie eine Ansicht bearbeitet wird.  
  
## Verwandte Abschnitte  
 [Spy\+\+\-Ansichten](../debugger/spy-increment-views.md)  
 Erklärt die Spy\+\+\-Strukturansichten von Fenstern, Meldungen, Prozessen und Threads.  
  
 [Spy\+\+\-Referenz](../debugger/spy-increment-reference.md)  
 Enthält Abschnitte mit Beschreibungen der einzelnen Spy\+\+\-Menüs und \-Dialogfelder.
