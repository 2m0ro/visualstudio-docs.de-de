# How to: Display Message Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Verwenden Sie das Dialogfeld [Meldungseigenschaften](../debugger/message-properties-dialog-box.md), um Informationen über eine in der [Meldungsansicht](../debugger/messages-view.md) aufgeführte Meldung zu erhalten.  
  
### So öffnen Sie das Dialogfeld "Meldungseigenschaften"  
  
1.  Wählen Sie in der Struktur, die in der Meldungsansicht angezeigt wird, eine Meldung aus.  
  
2.  Klicken Sie im Menü **Ansicht** auf **Eigenschaften**.  
  
 Eigenschaftendialogfelder sind nicht modal, sodass Sie in einem Ansichtsfenster auf ein anderes Element klicken können und im Dialogfeld Informationen zum ausgewählten Element angezeigt werden.
