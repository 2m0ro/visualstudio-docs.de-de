# How to: Display Thread Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Verwenden Sie das Dialogfeld [Threadeigenschaften](../debugger/thread-properties-dialog-box.md), um weitere Informationen über einen in der [Threadansicht](../debugger/threads-view.md) aufgeführten Thread zu erhalten.  
  
### So öffnen Sie das Dialogfeld "Threadeigenschaften"  
  
1.  Wählen Sie in der Struktur, die in der [Threadansicht](../debugger/threads-view.md) angezeigt wird, einen Thread aus.  
  
2.  Klicken Sie im Menü **Ansicht** auf **Eigenschaften**.  
  
 Eigenschaftendialogfelder sind nicht modal, sodass Sie in einem Ansichtsfenster auf ein anderes Element klicken können und im Dialogfeld Informationen zum ausgewählten Element angezeigt werden.
