# Dialogfeld &quot;Threadsuche&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Verwenden Sie dieses Dialogfeld, um den Knoten für einen bestimmten Thread in der [Threadansicht](../debugger/threads-view.md) zu suchen und auszuwählen.  Verschieben Sie den Fokus auf ein **Threadansichtsfenster**, um dieses Dialogfeld anzuzeigen.  Klicken Sie dann im Menü **Suchen** auf **Thread suchen**.  
  
 Die folgenden Einstellungen sind verfügbar:  
  
 **Thread**  
 Die ID des zu suchenden Threads.  
  
 **Modul**  
 Die zu suchende Modulzeichenfolge.  
  
 **Suchrichtung: Nach oben\/Nach unten**  
 Die anfängliche Richtung der Suche.
