# How to: Display Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ wird in der [Fensteransicht](../debugger/windows-view.md) geöffnet, einer Strukturansicht aller auf dem Computer verfügbaren Fenster und Steuerelemente.  Die Beschriftung dieses Stammfensters lautet "Fenster 1". Es sind außerdem drei weitere Spy\+\+\-Ansichten verfügbar: [Meldungsansicht](../debugger/messages-view.md), [Prozessansicht](../debugger/processes-view.md) und [Threadansicht](../debugger/threads-view.md).  Sie können die gewünschte Ansicht über das Menü **Spy** anzeigen.  Weitere Informationen finden Sie unter [Spy\+\+\-Ansichten](../debugger/spy-increment-views.md).
