# Dialogfeld &quot;Ausnahmen-Assistent&quot;
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Das Dialogfeld **Ausnahmen\-Assistent** wird angezeigt, wenn eine Laufzeitausnahme ausgelöst wird.  Der **Ausnahmen\-Assistent** zeigt den Typ der Ausnahme an und stellt zusätzliche Informationen und Links zu Hinweisen für die Fehlerbehebung bereit. Außerdem besteht die Möglichkeit, online nach weiteren Informationen zu suchen, und der Benutzer kann bestimmte Aktionen ausführen, z. B. das Anzeigen von Einzelheiten der Ausnahme.  
  
## Aufgabenliste  
 Klicken Sie auf einen der Hinweise im Bereich **Tipps zur Problembehandlung**, um ein Thema zur Fehlerbehebung der aufgetretenen Ausnahmeart anzuzeigen.  
  
 Um die der Ausnahme zugeordneten Aktionen auszuführen, klicken Sie auf eine der im Aktionsbereich angezeigten Aktionen.  
  
 Informationen zum Aktivieren und Deaktivieren des Ausnahmen\-Assistenten finden Sie unter [Allgemein, Debuggen, Dialogfeld "Optionen"](../debugger/general-debugging-options-dialog-box.md).  
  
## Liste der Benutzeroberflächenelemente  
 **Ausnahmetyp**  
 Zeigt den Typ der ausgelösten Ausnahme an.  
  
 **Zusätzliche Informationen**  
 Zeigt zusätzliche Informationen zur Ausnahme an.  
  
 **Hinweise zur Fehlerbehebung**  
 Zeigt Links zu Hinweisen für die Fehlerbehebung an, die Ihnen unter Umständen dabei helfen, die Quelle der Ausnahme zu ermitteln.  
  
 **Aktionen**  
 Führt Aktionen auf, die ausgeführt werden können, z. B. das Anzeigen von weiteren Informationen über das Ausnahmeobjekt.  
  
 **Onlinehilfe erhalten**  
 Ermöglicht es Ihnen, online nach zusätzlicher Hilfe zu suchen.
