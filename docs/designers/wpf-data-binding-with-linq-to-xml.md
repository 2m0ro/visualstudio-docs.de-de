# <a name="wpf-data-binding-with-linq-to-xml"></a>WPF-Datenbindung mit LINQ to XML
In diesem Abschnitt erfahren Sie, wie Sie LINQ to XML als Datenquelle für die Datenbindung in WPF-Anwendungen (Windows Presentation Foundation) verwenden können. Dieses Szenario basiert auf speziellen *dynamischen Eigenschaften* von <xref:System.Xml.Linq.XAttribute?displayProperty=fullName> und <xref:System.Xml.Linq.XElement?displayProperty=fullName>. Diese dynamischen Eigenschaften werden ebenfalls in diesem Abschnitt erläutert.

|[Dynamische Eigenschaften in LINQ to XML](../designers/linq-to-xml-dynamic-properties.md)|Bietet Referenzinformationen zu den dynamischen Eigenschaften, die in den Klassen <xref:System.Xml.Linq.XAttribute> und <xref:System.Xml.Linq.XElement> verfügbar gemacht werden.|
