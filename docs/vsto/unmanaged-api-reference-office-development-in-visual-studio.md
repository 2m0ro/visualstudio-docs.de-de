# Referenz zur nicht verwalteten API (Office-Entwicklung in Visual&#160;Studio)
  Ab 2007 Microsoft Office System verwenden Office\-Anwendungen die [IManagedAddin-Schnittstelle](../vsto/imanagedaddin-interface.md)\-Schnittstelle, um Aufrufe in einer VSTO\-Add\-in\-Ladekomponente auszuführen, die in der [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)] enthalten ist. Mithilfe dieser Komponente wird das Laden verwalteter VSTO\-Add\-Ins unterstützt. Sie können eine eigene VSTO\-Add\-In\-Ladekomponente erstellen, indem Sie diese Schnittstelle implementiert.  
  
## In diesem Abschnitt  
 [IManagedAddin-Schnittstelle](../vsto/imanagedaddin-interface.md)  
 Eine COM\-Schnittstelle, die Sie implementieren können, um verwaltete VSTO\-Add\-Ins in Office\-Anwendungen zu laden und zu entladen.
