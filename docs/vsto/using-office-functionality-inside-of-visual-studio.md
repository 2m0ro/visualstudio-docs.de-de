# Verwenden von Office-Funktionen in Visual Studio
  Wenn Sie ein Projekt auf Dokumentebene erstellen, werden das Dokument und die zugehörigen Anwendungen in Visual Studio gehostet, sodass Sie das Dokument direkt entwerfen und mit ihm arbeiten können.  Wenn Sie eine Microsoft Office\-Anwendung in Visual Studio geöffnet haben, funktioniert sie im Allgemeinen wie erwartet.  Einige der Anwendungsfunktionen sind jedoch anders, oder es kann nicht auf sie zugegriffen werden.  
  
 [!INCLUDE[appliesto_alldoc](../vsto/includes/appliesto-alldoc-md.md)]  
  
## Dokumentschutz  
 Microsoft Office Word und Microsoft Office Excel bieten Funktionen zum Dokumentschutz, die Sie in den Projekten verwenden können.  Wenn der Dokumentschutz jedoch aktiviert ist, während das Dokument in Visual Studio geöffnet ist, können Sie möglicherweise einige Entwurfsänderungen nicht durchführen.  Weitere Informationen finden Sie unter [Dokumentschutz in Projektmappen auf Dokumentebene](../vsto/document-protection-in-document-level-solutions.md).  
  
## Verwaltung von Informationsrechten  
 Die Verwaltung von Informationsrechten \(IRM, Information Rights Management\) ist in Microsoft Office Word und Microsoft Office Excel verfügbar.  Mit IRM können Sie verhindern, dass nicht autorisierte Personen Ihre vertraulichen Informationen anzeigen oder ändern.  IRM kann jedoch auch verhindern, dass der Code ausgeführt wird.  Weitere Informationen finden Sie unter [Übersicht über Information Rights Management und Erweiterungen durch verwalteten Code](../vsto/information-rights-management-and-managed-code-extensions-overview.md).  
  
## Kennwortschutz  
 Microsoft Office Word\-Dokumente und Microsoft Office Excel\-Arbeitsmappen können so eingerichtet werden, dass sie von einem Benutzer, der das Kennwort nicht kennt, nicht geöffnet werden können.  Der Kennwortschutz wird in Word und Excel unterschiedlich behandelt und kann den Entwicklungsprozess beeinflussen.  Weitere Informationen finden Sie unter [Kennwortschutz für Office-Dokumente](../vsto/password-protection-on-office-documents.md).
