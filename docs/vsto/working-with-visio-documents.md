# Arbeiten mit Visio-Dokumenten
  Die Themen in diesem Abschnitt bieten Schritt\-für\-Schritt\-Anweisungen und Codebeispiele dafür, wie das Objektmodell von Microsoft Office Visio mit Dokumenten in Office\-Projekten verwendet wird.  
  
|Aufgabe|Verfahren|  
|-------------|---------------|  
|Erstellen eines leeren Visio\-Dokuments oder eines neues Dokuments auf der Grundlage einer vorhandenen Visio\-Vorlage, \-Schablone oder eines vorhandenen Visio\-Dokuments|[Gewusst wie: Programmgesteuertes Erstellen neuer Visio-Dokumente](../vsto/how-to-programmatically-create-new-visio-documents.md)|  
|Öffnen eines vorhandenen Visio\-Dokuments und Angeben von Optionen für Lese\-\/Schreibzugriff, Schreibgeschützt und Angedockt|[Gewusst wie: Programmgesteuertes Öffnen von Visio-Dokumenten](../vsto/how-to-programmatically-open-visio-documents.md)|  
|Schließen des aktiven Visio\-Dokuments|[Gewusst wie: Programmgesteuertes Schließen von Visio-Dokumenten](../vsto/how-to-programmatically-close-visio-documents.md)|  
|Speichern eines Dokuments mit dem gleichen Namen oder einem neuen Namen|[Gewusst wie: Programmgesteuertes Speichern von Visio-Dokumenten](../vsto/how-to-programmatically-save-visio-documents.md)|  
|Drucken eines ganzen Dokuments oder einer angegebenen Seite|[Gewusst wie: Programmgesteuertes Drucken von Visio-Dokumenten](../vsto/how-to-programmatically-print-visio-documents.md)|
