## <a name="prerequisites"></a>Erforderliche Komponenten

* [Visual Studio 2017](https://visualstudio.microsoft.com/downloads/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=button+cta&utm_content=download+vs2017) mit den folgenden zu der von Ihnen bevorzugten Sprache passenden Workloads:
  * ASP.NET: **ASP.NET und Webentwicklung**
  * Node.js: **Node.js-Entwicklung**

* Ein Azure-Abonnement. Wenn Sie noch nicht über eins verfügen, [registrieren Sie sich kostenlos](https://azure.microsoft.com/free/dotnet/). Dabei werden Ihnen 200 USD für 30 Tage und 12 Monate für beliebte kostenlose Dienste gutgeschrieben.

* Ein ASP.NET Core-, .NET Core- oder Node.js-Projekt. Wenn Sie noch nicht über ein Projekt verfügen, wählen Sie eine der nachfolgenden Optionen aus:
  * ASP.NET: Führen Sie die unter [Schnellstart: Verwenden von Visual Studio zum Erstellen Ihrer ersten ASP.NET Core-Web-App](../../ide/quickstart-aspnet-core.md) beschriebenen Schritte aus, oder klicken Sie auf **Datei** > **Neues Projekt** > **Visual C#** > **.NET Core** > **ASP.NET Core-Webanwendung**. Wenn Sie dazu aufgefordert werden, klicken Sie auf die Vorlage **Webanwendung (Model View Controller)**, vergewissern Sie sich, dass **No Authentication** (Keine Authentifizierung) ausgewählt ist, und klicken Sie auf **OK**.
  * Node.js: Führen Sie die unter [Schnellstart: Erstellen einer ersten Node.js-App mit Visual Studio](../../ide/quickstart-nodejs.md) beschriebenen Schritte aus, oder klicken Sie auf **Datei** > **Neues Projekt** > **JavaScript****Leere Node.js Webanwendung**.

* Erstellen Sie das Projekt über den Menübefehl **Erstellen > Projektmappe erstellen**, bevor Sie mit der Bereitstellung beginnen.
