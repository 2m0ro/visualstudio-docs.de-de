# <a name="vspackage-essentials"></a>Grundlegendes zu VSPackages
VSPackages sind Softwaremodule, die Visual Studio um Benutzeroberflächenelemente, Dienste, Projekte, Editoren und Designer erweitern. VSPackages sind die zentrale architektonische Einheit von [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]. [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] selbst wird hauptsächlich in Form einer Auflistung von VSPackages geschrieben.  
  
 Zum Erstellen einer [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] -erweiterungsanwendung Sie zuerst erstellen Sie ein VSPackage. Sie können dann hinzufügen, Menübefehle, Toolfenster, Dokumentfenster oder anderen Code zum Erweitern der [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] IDE.  
  
 Um VSPackages verwenden zu können, müssen Sie das Visual Studio SDK installieren. Um diese exemplarische Vorgehensweise befolgen zu können, müssen Sie das Visual Studio SDK installieren. Weitere Informationen finden Sie unter [Visual Studio SDK](../extensibility/visual-studio-sdk.md).
