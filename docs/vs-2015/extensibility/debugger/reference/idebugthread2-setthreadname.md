# <a name="idebugthread2setthreadname"></a>IDebugThread2::SetThreadName
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [IDebugThread2::SetThreadName](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugthread2-setthreadname).  
  
Legt den Namen des Threads.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT SetThreadName (   
   LPCOLESTR pszName  
);  
```  
  
```csharp  
int SetThreadName (   
   string pszName  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pszName`  
 [in] Der Name des Threads.  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn erfolgreich, wird `S_OK`ist, andernfalls ein Fehlercode zurückgegeben.  
  
## <a name="remarks"></a>Hinweise  
 Um den Threadnamen abzurufen, rufen die [GetName](../../../extensibility/debugger/reference/idebugthread2-getname.md) Methode.
