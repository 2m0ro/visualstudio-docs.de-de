# <a name="metadataaddressfield"></a>METADATA_ADDRESS_FIELD
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [METADATA_ADDRESS_FIELD](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/metadata-address-field).  
  
Diese Struktur stellt die Adresse eines Felds, einer Klasse oder Struktur dar.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
typedef struct _tagMETADATA_ADDRESS_FIELD {  
   _mdToken tokField;  
} METADATA_ADDRESS_FIELD  
```  
  
```csharp  
public struct METADATA_ADDRESS_FIELD {  
   public int tokField;  
}  
```  
  
## <a name="terms"></a>Begriffe  
 tokField  
 Die ID der dem Feld-Token.
