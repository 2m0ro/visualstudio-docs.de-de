# <a name="idebugdefaultport2queryislocal"></a>IDebugDefaultPort2::QueryIsLocal
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [IDebugDefaultPort2::QueryIsLocal](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugdefaultport2-queryislocal).  
  
Diese Methode bestimmt, ob dieser Port auf dem lokalen Computer ist.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT QueryIsLocal(  
   void  
);  
```  
  
```csharp  
int QueryIsLocal();  
```

 Gibt `S_OK` ist dieser Port lokaler (auf dem gleichen Computer wie der Aufrufer) oder `S_FALSE` Wenn Port auf einem anderen Computer verwendet wird.
