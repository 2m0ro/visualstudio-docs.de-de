# <a name="idebugcoreserver2getportsupplier"></a>IDebugCoreServer2::GetPortSupplier
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [IDebugCoreServer2::GetPortSupplier](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugcoreserver2-getportsupplier).  
  
Ruft einen bestimmten Port Lieferanten ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT GetPortSupplier(   
   REFGUID               guidPortSupplier,  
   IDebugPortSupplier2** ppPortSupplier  
);  
```  
  
```csharp  
int GetPortSupplier(   
   ref Guid                guidPortSupplier,  
   out IDebugPortSupplier2 ppPortSupplier  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `guidPortSupplier`  
 [in] Die GUID des portbereitstellers abgerufen werden sollen.  
  
 `ppPortSupplier`  
 [out] Gibt eine [IDebugPortSupplier2](../../../extensibility/debugger/reference/idebugportsupplier2.md) Objekt, das den gewünschten Port Lieferanten darstellt.  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn erfolgreich, wird `S_OK`ist, andernfalls ein Fehlercode zurückgegeben.
