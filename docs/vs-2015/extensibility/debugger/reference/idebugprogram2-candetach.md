# <a name="idebugprogram2candetach"></a>IDebugProgram2::CanDetach
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [IDebugProgram2::CanDetach](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugprogram2-candetach).  
  
Bestimmt, ob die Anwendung eine Debug-Engine (DE) trennen kann.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT CanDetach(  
   void  
);  
```  
  
```csharp  
int CanDetach();  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn können zu trennen, gibt `S_OK`ist, andernfalls ein Fehlercode zurückgegeben. Gibt `S_FALSE` , wenn das Programm die DE trennen kann nicht.
