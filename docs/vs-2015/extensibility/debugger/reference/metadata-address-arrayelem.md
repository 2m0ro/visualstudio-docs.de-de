# <a name="metadataaddressarrayelem"></a>METADATA_ADDRESS_ARRAYELEM

Die neueste Version dieses Themas finden Sie unter [METADATA_ADDRESS_ARRAYELEM](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/metadata-address-arrayelem).  
  
Diese Struktur stellt ein Arrayelement in einem Array dar.

```cpp  
typedef struct _tagMETADATA_ADDRESS_ARRAYELEM {
```  

