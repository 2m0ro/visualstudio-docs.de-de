# <a name="c28241"></a>C28241
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28241](https://docs.microsoft.com/visualstudio/code-quality/c28241).  
  
Warnung C28241: Die Anmerkung für die Funktion des Parameters wird nicht erkannt.  
  
 Es wurde eine unbekannte anmerkungsnamen verwendet. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
