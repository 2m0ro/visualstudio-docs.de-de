# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28218](https://docs.microsoft.com/visualstudio/code-quality/c28218).  
  
Warnung C28218: Bei dem Funktionsparameter stimmt der Parameter für die Anmerkung nicht mit dem bei der Datei gefundenen Parameter überein.  
  
 Die Anmerkungen für die aktuelle Zeile und in der Zeile in der Nachricht sind inkonsistent. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet bei normaler Nutzung angezeigt werden.
