# <a name="ca0057"></a>CA0057

Verstöße gegen können nicht den aktuellen Satz von Zielen und Regeln zugeordnet werden.

Dieser Fehler tritt auf, wenn FxCopCmd.exe manuell, mit der Projektdatei aufgerufen wird oder einen importierten XML-Bericht mit Verletzungen, die den aktuellen Satz von Zielen und Regeln zugeordnet werden kann.

Tritt auf, nur wenn Analyse erfolgt mit der Befehlszeilenversion des Tools.
