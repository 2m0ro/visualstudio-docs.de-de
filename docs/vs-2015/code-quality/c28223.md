# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28223: Erwartetes Token/Bezeichner für Anmerkung, wurde nicht gefunden werden Parameter  
  
 Diese Warnung gibt an, dass die Parameter für eine Anmerkung muss ein Bezeichner sein und einige andere Symbol wurde gefunden. Dies bedeutet in der Regel ein falsch codierten Anmerkung-Makro. Die Verwendung von C oder C++-Schlüsselwörter in dieser Position verursacht diesen Fehler.
