# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28226: Anmerkung kann nicht zusätzlich ein primop sein (in der aktuellen Deklaration).  
  
 Diese Warnung gibt an, dass die benannten Anmerkung als zusätzlich ein PrimOp deklariert wird ist, und auch wurde zuvor als eine normale Anmerkung deklariert. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet bei normaler Nutzung angezeigt werden.
