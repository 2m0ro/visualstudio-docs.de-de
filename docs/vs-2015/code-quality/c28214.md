# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28214](https://docs.microsoft.com/visualstudio/code-quality/c28214).  
  
Warnung C28214: Attributparameternamen müssen p1...p9 sein.  
  
 Diese Warnung gibt an, dass wenn Sie eine Deklaration für die Anmerkung erstellen, die Parameternamen auf p1... p9. Diese Warnung sollte niemals während normaler Verwendung auftreten.
