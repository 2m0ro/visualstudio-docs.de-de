# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28236](https://docs.microsoft.com/visualstudio/code-quality/c28236).  
  
Warnung C28236: Die Anmerkung kann nicht in einem Ausdruck verwendet werden.  
  
 Diese Warnung gibt an, dass versucht wurde, eine Funktion deklariert, um eine Anmerkung in einem Ausdruckskontext verwenden. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
