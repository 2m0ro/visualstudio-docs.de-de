# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28227: Anmerkung kann nicht zusätzlich ein primop sein. (Siehe vorherige Deklaration)  
  
 Diese Warnung gibt an, dass die benannten Anmerkung als gewöhnliche Anmerkung deklariert wird, und auch wurde zuvor als zusätzlich ein PrimOp deklariert. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
