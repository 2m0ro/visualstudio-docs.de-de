# <a name="ca0505"></a>CA0505

Die als veraltet markierte Eigenschaft wird ignoriert, da die ersetzende Eigenschaft definiert ist.

Warnung CA505 wird generiert, wenn die angegebene veraltete sowohl ersetzende Codeanalyseeigenschaften Werte in einem Projekt enthalten ist, oder erstellen Sie die Konfigurationsdatei. Um dieses Problem zu beheben, entfernen Sie die als veraltet markierte Eigenschaft ein.
