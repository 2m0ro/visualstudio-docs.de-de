# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28289: für Funktion: ReadableTo oder WritableTo keine Limit-Spezifikation als Parameter  
  
 Das Codeanalysetool meldet diese Warnung aus, wenn die Anmerkung Funktionsparameter/miscoded ist, wie bereits erwähnt.
