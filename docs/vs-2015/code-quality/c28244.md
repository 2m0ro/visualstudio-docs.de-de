# <a name="c28244"></a>C28244
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28244](https://docs.microsoft.com/visualstudio/code-quality/c28244).  
  
Warnung C28244: Die Anmerkung für die Funktion verfügt über einen nicht analysierbaren Parameter/eine externe Anmerkung.  
  
 Dies sollte derzeit nicht möglich sein, wenn die standard-Makros zur Anmerkungen verwendet werden. Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
