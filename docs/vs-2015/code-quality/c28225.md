# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28225](https://docs.microsoft.com/visualstudio/code-quality/c28225).  
  
Warnung C28225: In der Anmerkung wurde nicht die richtige Anzahl an Parametern gefunden  
  
 Diese Warnung gibt an, dass die benannten Anmerkung mit der falschen Anzahl von Parametern verwendet wird. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
