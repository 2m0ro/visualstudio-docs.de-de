# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28283](https://docs.microsoft.com/visualstudio/code-quality/c28283).  
  
Warnung C28283: Für das Symbol wird die festgelegte Größenangabe noch nicht unterstützt.  
  
 Die Warnung gibt an, dass die Anmerkung Größe-Spezifikation "Sentinel" einen anderen Wert als 0 (null) empfangen. Im Wesentlichen versucht sich der Aufrufer zu sagen, dass die Zeichenfolge durch ein anderes Zeichen als binäre 0 (null) beendet wird.
