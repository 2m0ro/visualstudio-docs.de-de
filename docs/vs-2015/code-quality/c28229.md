# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28229](https://docs.microsoft.com/visualstudio/code-quality/c28229).  
  
Warnung C28229: Anmerkung unterstützt keine Parameter.  
  
 Diese Warnung gibt an, dass eine Anmerkung mit einem Parameter verwendet wurde, wenn die Anmerkung ohne Parameter deklariert wird. Dies sollte nicht möglich, wenn die standard-Makros für Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, normalen datenbanknutzung angesehen werden.
