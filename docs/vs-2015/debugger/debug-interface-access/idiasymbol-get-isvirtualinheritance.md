# <a name="idiasymbolgetisvirtualinheritance"></a>IDiaSymbol::get_isVirtualInheritance
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [IDiaSymbol::get_isVirtualInheritance](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-isvirtualinheritance).  
  
Gibt an, ob die `this` Zeiger verweist auf einen Datenmember mit virtueller Vererbung.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT get_isVirtualInheritance(   
   BOOL* pRetVal);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pRetVal`  
 [out] Ein Zeiger auf eine `BOOL` , der angibt, ob die `this` Zeiger verweist auf einen Datenmember mit virtueller Vererbung.
