## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT get_notReached(  
   BOOL *pFlag  
);
```  

#### <a name="parameters"></a>Parameter

 [out] Gibt `TRUE` , wenn die Funktion oder Bezeichnung nie erreicht wird; andernfalls `FALSE`.
