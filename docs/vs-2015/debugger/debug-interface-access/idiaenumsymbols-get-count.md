# <a name="idiaenumsymbolsgetcount"></a>IDiaEnumSymbols::get_Count
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [idiaenumsymbols:: Get_count](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumsymbols-get-count).  
  
Ruft die Anzahl der Symbole ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 pRetVal  
 [out] Gibt die Anzahl der Symbole zurück.  
  
## <a name="return-value"></a>Rückgabewert  
 Wenn erfolgreich, wird `S_OK`ist, andernfalls ein Fehlercode zurückgegeben.
