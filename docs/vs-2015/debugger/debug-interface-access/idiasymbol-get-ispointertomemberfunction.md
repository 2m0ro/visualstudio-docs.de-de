# <a name="idiasymbolgetispointertomemberfunction"></a>IDiaSymbol::get_isPointerToMemberFunction
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [IDiaSymbol::get_isPointerToMemberFunction](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-ispointertomemberfunction).  
  
Gibt an, ob dieses Symbol, ein Zeiger auf eine Memberfunktion ist.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT get_isPointerToMemberFunction(   
   BOOL* pRetVal);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pRetVal`  
 [out] Ein Zeiger auf eine `BOOL` , der angibt, ob dieses Symbol, ein Zeiger auf eine Memberfunktion ist.
