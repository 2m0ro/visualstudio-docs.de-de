# <a name="idiaenumlinenumbersreset"></a>IDiaEnumLineNumbers::Reset

Die neueste Version dieses Themas finden Sie unter [idiaenumlinenumbers:: Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumlinenumbers-reset).  
  
Setzt eine Enumerationsfolge auf den Anfang zurück.  
  
## <a name="syntax"></a>Syntax  
  
```cpp#  
HRESULT Reset();  
```
