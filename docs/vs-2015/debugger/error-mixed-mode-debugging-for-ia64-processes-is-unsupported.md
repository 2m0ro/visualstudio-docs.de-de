# <a name="error-mixed-mode-debugging-for-ia64-processes-is-unsupported"></a>Fehler: Debuggen im gemischten Modus wird für IA64-Prozesse nicht unterstützt
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [Fehler: Debuggen im gemischten Modus für IA64-Prozesse wird nicht unterstützt](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-for-ia64-processes-is-unsupported).  
  
Der [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]-Debugger unterstützt das Debuggen von gemischtem systemeigenem und verwaltetem Code in Itanium-basierten Prozessen nicht.  
  
### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Erstellen Sie eine 32-Bit-Version der Anwendung zum Debuggen.
