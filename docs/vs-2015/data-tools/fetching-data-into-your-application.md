# <a name="fetching-data-into-your-application"></a>Abrufen von Daten für die Anwendung
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Laden von Daten in Ihre Anwendung beschreibt den Prozess der Abrufen von Daten aus einer Datenquelle, z. B. eine Datenbank oder einen Dienst, und füllen die Objekte im Datenmodell einer Anwendung. In den folgenden Themen wird beschrieben, wie Sie Daten abrufen und Laden von Daten in den verfügbaren Datenmodellen.
