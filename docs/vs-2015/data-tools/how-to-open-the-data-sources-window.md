# <a name="how-to-open-the-data-sources-window"></a>Gewusst wie: Öffnen des Datenquellenfensters
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die **Datenquellen** Fenster werden die Datenquellen in Ihrem Projekt angezeigt. Sie verwenden die **Datenquellen** Fenster, das eine Benutzeroberfläche datengebundene Steuerelemente durch Ziehen von Elementen aus dem Fenster auf Formulare im Projekt erstellt. Finden Sie unter [Binden von Windows Forms-Steuerelementen an Daten in Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Sie füllen können die **Datenquellen** Fenster durch Ausführen der **Datenquellenkonfiguration** Assistenten. Weitere Informationen finden Sie unter [Assistenten zur Datenquellenkonfiguration](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).  
  
### <a name="to-open-the-data-sources-window"></a>So öffnen Sie das Datenquellenfenster  
  
-   Wählen Sie auf der Menüleiste **Ansicht**, **Other Windows**, **Datenquellen** (oder drücken Sie die Tastenkombination Umschalt + Alt + D).  
  
    > [!IMPORTANT]
    >  Sie können eine Datenquelle einem Projekt nur hinzufügen, wenn es das Erstellen von Datenquellen und Arbeiten mit Datenquellen unterstützt. Angenommen, Sie können nicht geöffnet werden die **Datenquellen** Fenster in einem Projekt für eine Windows Store-app.
