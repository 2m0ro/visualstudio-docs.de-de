# <a name="connecting-to-data-in-visual-studio"></a>Herstellen von Datenverbindungen in Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] bietet Tools, die Ihre Anwendung mit Daten aus vielen verschiedenen Quellen, z. B. Datenbanken, Diensten und Objekten verbinden. Verbindungen zu Daten werden entweder programmgesteuert (im Code) erstellt, indem man Datenobjekte auf einem Formular oder Komponenten-Designer ablegt, oder mit dem Durchlaufen eines der Daten-Assistenten. Die folgenden Themen beschreiben, wie von Ihrer Anwendung Verbindungen zu Datenbanken, Diensten und Objekten hergestellt werden können.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Gewusst wie: Herstellen einer Verbindung zu Daten in einer Datenbank](../data-tools/how-to-connect-to-data-in-a-database.md)  
 Beschreibt das Erstellen einer Verbindungs zwischen Ihrer Anwendung und einer Datenbank über die **Assistenten zur Datenquellenkonfiguration**.  
  
 [Gewusst wie: Herstellen einer Verbindung mit Daten in einem Dienst](../data-tools/how-to-connect-to-data-in-a-service.md)  
 Beschreibt das Erstellen einer Verbindungs zwischen Ihrer Anwendung und die von einer Web-Dienst mit zurückgegebenen Daten die **Assistenten zur Datenquellenkonfiguration**.  
  
 [Vorgehensweise: Verbinden mit Daten in Objekten](http://msdn.microsoft.com/library/862fd351-0f4d-4220-9743-6103b87dc24b)  
 Beschreibt, wie man an Eigenschaftswerten in Objekten mithilfe der **Assistenten zur Datenquellenkonfiguration**.  
  
 [Gewusst wie: Speichern und Bearbeiten von Verbindungszeichenfolgen](~/E:/Repos/visualstudio-docs-pr/docs/data-tools/how-to-save-and-edit-connection-strings.md)  
 Beschreibt, wie Verbindungsinformationen in der Anwendung gespeichert und geändert werden.  
  
 [Herstellen einer Verbindung mit Daten in Windows Forms-Anwendungen](../data-tools/connecting-to-data-in-windows-forms-applications.md)  
 Enthält Informationen über die Verbindung von Windows Forms-Anwendungen zu Daten mit Entwurfszeittools.
