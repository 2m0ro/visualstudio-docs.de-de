# <a name="editing-data-in-your-application"></a>Bearbeiten von Daten in der Anwendung
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nachdem das Datenmodell mit Daten gefüllt ist, werden typischerweise Daten hinzugefügt, geändert oder gelöscht, ehe sie an die Datenquelle oder zu einem anderen Prozess bzw. einer anderen Anwendung zurückgesandt werden.  
  
 Die folgenden Themen enthalten Details über das Hinzufügen, Ändern und Löschen von Daten und wie man mit Daten in unterschiedlichen Stufen einer Anwendung umgeht.  
  
## <a name="related-topics"></a>Verwandte Themen  
 [Bearbeiten von Daten in Datasets](../data-tools/edit-data-in-datasets.md)  
 Enthält Informationen über viele Aufgaben zur Bearbeitung von Daten in einem Dataset.  
  
 [Ändern von Entitätsdaten](http://msdn.microsoft.com/library/b8673200-1219-4fd9-b879-373c87cec7ce)  
 Enthält Informationen über das Erstellen, Aktualisieren und Löschen von Daten im Entity Framework und in WCF Data Services.
