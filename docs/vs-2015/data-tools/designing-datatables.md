# <a name="designing-datatables"></a>Entwerfen von DataTables
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] bietet zur Entwurfszeit-Tools zum Erstellen und Bearbeiten von Tabellen (**DataTable**). Eine Übersicht, was eine **DataTable** ist, finden Sie in [DataTables](http://msdn.microsoft.com/library/52ff0e32-3e5a-41de-9a3b-7b04ea52b83e).  
  
 In den folgenden Themen wird erläutert, wie zum Erstellen von Datentabellen, die mit typisierten "Datasets" Hinzufügen der **Dataset-Designer**, und erstellen und bearbeiten Sie die Datenspalten (**DataColumn**), die sie definieren.  
  
> [!NOTE]
>  Je nach den aktiven Einstellungen oder der Version unterscheiden sich die Dialogfelder und Menübefehle auf Ihrem Bildschirm möglicherweise von den in der Hilfe beschriebenen. Klicken Sie im Menü **Extras** auf **Einstellungen importieren und exportieren** , um die Einstellungen zu ändern. Weitere Informationen finden Sie unter [Anpassen der Entwicklungseinstellungen in Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="reference"></a>Referenz  
 <xref:System.Data.DataSet>  
 Stellt einen speicherresidenten Datencache dar.  
  
 <xref:System.Data.DataTable>  
 Stellt eine Tabelle mit Daten im Arbeitsspeicher dar.  
  
 <xref:System.Data.DataColumn>  
 Stellt das Schema einer Spalte in einer <xref:System.Data.DataTable>.  
  
 <xref:System.Data.DataRelation>  
 Stellt eine über-/unterordnungsbeziehung zwischen zwei <xref:System.Data.DataTable> Objekte.
