# <a name="rtvs-current-version"></a>Aktuelle Version von RTVS

Bei der letzten Aktualisierung dieser Seite war die Version von R Tools für Visual Studio (RTVS) **1.0.30531.2148**. Ihre Version kann sich unterschieden, je nachdem welche Version von Visual Studio Sie verwenden und ob Sie einen Previewkanal nutzen.

Sie können die Version überprüfen, indem Sie in Visual Studio zu **Help > About** (Hilfe > Info) gehen und dann zum Eintrag **R Tools für Visual Studio** scrollen: 

![Überprüfen der RTVS-Version in Visual Studio (VS2017)](media/current-version.png)

Um Ihre Version von RTVS in Visual Studio 2017 zu aktualisieren, gehen Sie zum Visual Studio-Installer, und klicken Sie auf die Schaltfläche **Aktualisieren**, oder gehen Sie zu [aka.ms/rtvs-current](https://aka.ms/rtvs-current).

Informationen zu Visual Studio- und Windows-Anforderungen finden Sie unter [Installation](installation.md).

Updates zu Microsoft R Open und Microsoft R Server finden Sie unter [Produkte von Microsoft R](http://aka.ms/rtvs-msft-r).
