# <a name="rtvs-current-version"></a>Aktuelle Version von RTVS

Die neueste Version von R Tools für Visual Studio (RTVS) ist **1.0.30321.1407** (empfohlen).

Sie ist die 1.0-Releaseversion von RTVS, die unter [aka.ms/rtvs-current](https://aka.ms/rtvs-current) verfügbar ist.

Sie können die Version überprüfen, indem Sie in Visual Studio zu **Help > About** (Hilfe > Info) gehen und dann zum Eintrag **R Tools für Visual Studio** scrollen:

![Überprüfen der RTVS-Version in Visual Studio (VS2017)](~/rtvs/media/current-version.png)

Informationen zu Visual Studio- und Windows-Anforderungen finden Sie unter [Installation](installation.md).

Updates zu Microsoft R Open und Microsoft R Server finden Sie unter [Produkte zu Microsoft R](http://aka.ms/rtvs-msft-r).
