# <a name="code-generation-in-visual-studio"></a>Codegenerierung in Visual Studio
Visual Studio kann Sie beim Generieren von Teilen des Quellcodes unterstützen, sodass Sie sich auf die eigentliche Arbeit konzentrieren können.  Zu den gängigen Funktionen zur Codegenerierung gehören:  

* Generieren von Klassen, Methoden, Typen usw.
* Generieren von XML-Dokumentationskommentaren
* Implementieren von abstrakten Klassen oder Schnittstellen
* Einführen einer lokalen Variable in einen komplexen Ausdruck

Da die Logik der Codegenerierung eng mit der Sprachsyntax verbunden ist, stellt jeder Sprachdienst in Visual Studio eigene Funktionen zur Codegenerierung bereit. Weitere Informationen finden Sie auf den folgenden Seiten:  
  
* [Codegenerierung (C#)](../csharp-ide/code-generation-csharp.md)
* [Codegenerierung (Visual Basic)](../vb-ide/code-generation-vb.md)
