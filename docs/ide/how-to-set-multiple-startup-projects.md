# Gewusst wie: Festlegen mehrerer Startprojekte
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

In Visual Studio können Sie angeben, wie mehr als ein Projekt beim Starten des Debuggers ausgeführt wird.  
  
### So legen Sie mehrere Startprojekte fest  
  
1.  Im **Projektmappen\-Explorer** wählen Sie die Projektmappe \(der oberste Knoten\).  Klicken Sie mit der rechten Maustaste auf den Knoten, um das Kontextmenü aufzurufen.  
  
2.  **Eigenschaften** auswählen.  Das Dialogfeld **\-Eigenschaftenseiten**wird geöffnet.  
  
3.  Erweitern Sie die **Allgemeine Eigenschaften** Knoten und klicken Sie auf **Startprojekt**.  
  
4.  Klicken Sie auf **mehrere Startprojekte** und legen Sie die entsprechenden Aktionen fest.  Weitere Informationen finden Sie unter [Gewusst wie: Ändern von Projekteigenschaften und Konfigurationseinstellungen](http://msdn.microsoft.com/de-de/e7184bc5-2f2b-4b4f-aa9a-3ecfcbc48b67).
