# <a name="internet-resources-used-by-visual-studio"></a>Von Visual Studio verwendete Internetressourcen

Visual Studio ist eine vernetzte Umgebung:

- Sie können sich bei Visual Studio [anmelden](../ide/signing-in-to-visual-studio.md), um den Zugriff auf alle Ihrem Benutzerkonto zugeordneten und mit diesem verknüpften Dienste und Konten zu erhalten. Ihre Kontoeinstellungen "wandern" mit Ihnen mit, unabhängig davon, wo Sie Visual Studio verwenden.  
- Visual Studio [informiert Sie](../ide/visual-studio-notifications.md) darüber, wenn für eine Komponente Ihrer Installation, beispielsweise für ein SDK, ein Update verfügbar ist.  
- Sie können [NuGet-Pakete](/nuget/) herunterladen, um Bibliotheken abzurufen.  
- Sie können [Visual Studio-Erweiterungen](../ide/finding-and-using-visual-studio-extensions.md) herunterladen.  
- Sie können eine Verbindung mit Azure-Diensten herstellen.  
- Die Startseite wird regelmäßig mit Neuigkeiten für Entwickler von verschiedenen Websites usw.  
- aktualisiert.  

Die Themen in diesem Abschnitt beziehen sich auf die verschiedenen Möglichkeiten, wie Visual Studio Internetressourcen verwendet.
