# Hinzuf&#252;gen und Entfernen von Projektelementen
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie können Projekte und Projektelemente hinzufügen oder entfernen, indem Sie in **Projektmappen\-Explorer** mit der rechten Maustaste auf den Projekt\- oder Projektmappenknoten klicken und **Hinzufügen** oder **Entfernen** auswählen.  Wird ein Projekt aus einer Projektmappe oder ein Element aus einem Projekt entfernt, wird das Projekt oder Element nicht vom Datenträger oder aus der Quellcodeverwaltung gelöscht, es sei denn, Sie wählen die Option aus, es dauerhaft zu löschen.
