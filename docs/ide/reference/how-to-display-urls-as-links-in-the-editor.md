# Gewusst wie: Anzeigen von URLs als Links im Editor
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Sie können einstellen, dass Uniform Resource Locators \(URLs\) im Code vom Code\-Editor als aktive Links behandelt werden.  Wenn Sie dieses Feature verwenden, gilt für URLs Folgendes:  
  
-   Sie werden unterstrichen angezeigt.  
  
-   Es wird ein **QuickInfo** angezeigt, wenn Sie mit der Maus auf sie zeigen.  
  
-   Es wird versucht, die angegebene Website zu öffnen, wenn Sie bei gedrückter **STRG**\-TASTE mit der linken Maustaste auf den Link klicken.  Standardmäßig wird die Website im internen Webbrowser angezeigt.  
  
> [!NOTE]
>  Je nach den aktiven Einstellungen oder der Version unterscheiden sich die Dialogfelder und Menübefehle auf Ihrem Bildschirm möglicherweise von den in der Hilfe beschriebenen.  Wählen Sie im Menü **Extras** die Option **Einstellungen importieren und exportieren** aus, um die Einstellungen zu ändern.  Weitere Informationen finden Sie unter [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/de-de/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
## Verfahren  
  
#### So zeigen Sie URLs als Links an  
  
1.  Klicken Sie im Menü **Extras** auf **Optionen**.  
  
2.  Klicken Sie auf **Text\-Editor**.  
  
3.  Um die Option nur für eine Sprache zu ändern, erweitern Sie den Ordner für die gewünschte Sprache und wählen den Eintrag **Allgemein** aus.  
  
     \- oder \-  
  
     Wenn die Option für alle Sprachen geändert werden soll, erweitern Sie den Ordner **Alle Sprachen** und wählen den Eintrag **Allgemein** aus.  
  
4.  Aktivieren Sie unter **Anzeige** die Option **Einfaches Klicken für URLs aktivieren**.
