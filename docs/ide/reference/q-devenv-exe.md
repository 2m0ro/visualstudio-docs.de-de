# /? (devenv.exe)
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Zeigt ein Meldungsfeld an, in dem alle `devenv`\-Schalter mit jeweils einer kurzen Beschreibung aufgelistet sind.  
  
## Syntax  
  
```  
devenv /?  
```
