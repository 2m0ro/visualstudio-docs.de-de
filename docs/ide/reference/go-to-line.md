# Gehe zu Zeile
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Mithilfe dieses Dialogfelds können Sie zu einer bestimmten Zeile im aktiven Dokument wechseln.  Um auf dieses Dialogfeld zuzugreifen, öffnen Sie ein Dokument zur Bearbeitung und wählen dann im Menü **Bearbeiten** die Option **Gehe zu** aus.  
  
 Zeilennummer \(1 \- `n`\)  
 Ermöglicht es Ihnen, die Nummer der Zeile im aktiven Dokument anzugeben, zu der Sie wechseln möchten.  Die eingegebene Nummer muss zwischen 1 und `n`, d. h. der Anzahl der Zeilen im aktuellen Dokument, liegen.
