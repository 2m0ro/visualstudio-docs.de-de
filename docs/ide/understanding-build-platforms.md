# Grundlagen zu Buildplattformen
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie können verschiedene Versionen der Projektmappe und der Projekteigenschaften speichern, die für verschiedene Zielplattformen gelten.  Beispielsweise können Sie eine Debugkonfiguration erstellen, die eine x86\-Zielplattform und eine Debugkonfiguration vorgesehen, die eine x64\-Plattform abzielt.  Sie können die aktive Plattform schnell ändern, damit Sie mehrere Konfigurationen leicht erstellen können.
