# How to: Add Comments to Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Mithilfe von Kommentarshapes können Sie Klassendiagramme mit Anmerkungen versehen.  Ein Kommentarshape verfügt über eine Eigenschaft, **Text**, in die Sie Text eingeben können.  Kommentarshapes sind nur auf der Diagrammoberfläche und nicht im Code vorhanden.  
  
 Ein Kommentar ist einer Klassendiagrammansichtim **Klassen\-Designer** zugeordnet. Wenn Sie ein zweites Klassendiagramm für dasselbe Projekt öffnen, sind die in der ersten Ansicht erstellten Kommentare nicht sichtbar.  Wenn Sie ein Diagramm löschen, werden alle darin enthaltenen Kommentare ebenfalls gelöscht.  
  
 Sie können die Größe eines Kommentarshapes ändern, andere Aspekte der Darstellung, z. B. Hintergrundfarbe, Schriftart oder Schriftgrad, dagegen nicht.  
  
### So fügen Sie einen Kommentar hinzu  
  
1.  Ziehen Sie einen Kommentar aus der **Klassen\-Designer\-Toolbox** auf das Klassendiagramm.  
  
2.  Klicken Sie im Diagramm auf das neue Kommentarshape, und geben Sie den gewünschten Text ein.
