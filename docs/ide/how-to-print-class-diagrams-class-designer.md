# How to: Print Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie können ein Klassendiagramm mithilfe des Druckfeatures in Visual Studio drucken.  
  
### So drucken Sie ein Klassendiagramm  
  
1.  Zeigen Sie das Klassendiagramm an.  \(Klicken Sie ggf. auf die Registerkarte mit dem Klassendiagramm, um es anzuzeigen.\)  
  
2.  Klicken Sie im Menü **Datei** auf **Drucken**.  
  
     Das gesamte Klassendiagramm wird gedruckt.  Möglicherweise müssen Sie die Einstellungen im Dialogfeld Seite einrichten anpassen, damit das Klassendiagramm in einer geeigneten Größe gedruckt wird.
