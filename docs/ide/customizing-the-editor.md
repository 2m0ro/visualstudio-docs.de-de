# <a name="customize-the-editor"></a>Anpassen des Editors

Sie können die Formatierung, Registerkarten, Schriftarten und Farben sowie IntelliSense und viele weitere Funktionen im gewöhnlichen Text-Editor oder in allen oder nur in einer speziellen Programmiersprache anpassen. In den verschiedenen Abschnitten zu diesem Thema wird erläutert, wie Sie einige dieser Anpassungen vornehmen können.

## <a name="options-dialog-box"></a>Optionen (Dialogfeld)

Weitere Informationen zu den Optionen des Text-Editors finden Sie unter [Dialogfeld „Optionen“ des Text-Editors](../ide/reference/text-editor-options-dialog-box.md).
