# Anpassen von Projekt- und Elementvorlagen
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Nach Projekt\- und Elementvorlagen sind, haben Sie die Möglichkeit, sie noch weiter anpassen, um Ihren Anforderungen erstellt wurden.  
  
 Sie können z. B. folgende Aufgaben ausführen:  
  
-   Ändern und exportieren Sie eine vorhandene Vorlage als benutzerdefinierte Vorlage.  
  
     Weitere Informationen finden Sie unter [Gewusst wie: Aktualisieren vorhandener Vorlagen](../ide/how-to-update-existing-templates.md).  
  
-   Führen Sie benutzerdefinierte Parameter an eine Vorlage, um vorhandene Werte zu ersetzen.  
  
     Weitere Informationen finden Sie unter [Gewusst wie: Ersetzen von Parametern in einer Vorlage](../ide/how-to-substitute-parameters-in-a-template.md).  
  
-   Passen Sie die Assistenten an, die Projekte aus Vorlagen erstellen.  
  
     Weitere Informationen finden Sie unter [Gewusst wie: Verwenden von Assistenten mit Projektvorlagen](../extensibility/how-to-use-wizards-with-project-templates.md).
