# Projektbuildfehler PRJ0048
Ein Buildvorgang konnte nicht gestartet werden, da bereits ein Buildvorgang ausgeführt wird.  
  
 Sie können keinen Buildvorgang starten, wenn bereits ein anderer Buildvorgang ausgeführt wird. Warten Sie, bis der aktuelle Buildvorgang abgeschlossen ist, oder brechen Sie den Buildvorgang ab.
