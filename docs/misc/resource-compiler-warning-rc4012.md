# Ressourcencompiler: Warnung RC4012
Gleitkommakonstante bei einer plattformübergreifenden Kompilierung  
  
 Eine **Gleitkommakonstante** weist auf unterschiedlichen Zielplattformen möglicherweise nicht denselben Wert auf.
