# CA0055
Könnte file:path nicht laden \<\>  
  
 Das Codeanalysetool konnte die für die Analyse angegebene Zieldatei nicht laden.  Dieser Fehler wird angezeigt, wenn die Datei nicht gefunden oder der Zugriff auf die Datei verweigert wurde.  Normalerweise ist das der Fall, wenn der Benutzer versucht, die Eigenschaft für die Eingabeassembly in [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] zu überschreiben oder fxcopcmd.exe manuell aufzurufen.
