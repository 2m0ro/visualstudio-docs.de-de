# Division by zero.
Division durch 0 \(null\) ist mathematisch unzulässig.  Im Allgemeinen tritt dieser Fehler auf, wenn der Wert eines Ausdrucks, der als Divisor verwendet wird, 0 \(null\) ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie den Ausdruck auf Rechtschreibfehler, logische Fehler oder Fehler bei der Groß\-\/Kleinschreibung.
