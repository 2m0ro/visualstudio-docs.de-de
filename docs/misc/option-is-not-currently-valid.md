# Option is not currently valid.
Dieser Fehler tritt normalerweise auf, wenn ein Argument oder ein Schalter für einen Befehl angegeben wurde, der zurzeit nicht verfügbar ist.  Wenn Sie z. B. `>Edit.Find test /sel` eingegeben haben und in der Datei keine Auswahl getroffen wurde, steht der Schalter \/sel nicht zur Verfügung.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass die eingegebenen Argumente und Schalter zur Verfügung stehen.  Weitere Informationen finden Sie unter [Visual Studio\-Befehle mit Argumenten](../ide/reference/visual-studio-commands.md).
