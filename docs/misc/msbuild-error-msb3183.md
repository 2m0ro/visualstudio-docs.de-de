# MSBuild Error MSB3183
**MSB3183: Der \<Assembly\>\-Verweis ist eine Interop\-Assembly, die volle Vertrauenswürdigkeit benötigt.**  
  
 Diese Warnung wird generiert, wenn beim Generieren des Manifests für eine teilweise vertrauenswürdige Anwendung unter den Verweisen eine Interop\-Assembly vorgefunden wird.
