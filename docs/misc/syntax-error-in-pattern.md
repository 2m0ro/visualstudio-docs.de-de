# Syntax error in pattern.
Dieser Fehler tritt normalerweise auf, wenn eine Suchzeichenfolge einen oder mehrere reguläre Ausdrücke enthält, von denen jedoch mindestens einer einen Syntaxfehler aufweist.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie die Zeichenfolge erneut ein.  Weitere Informationen zur Syntax finden Sie unter [Reguläre Ausdrücke](../ide/using-regular-expressions-in-visual-studio.md).
