# MSBuild Error MSB1001
**Unbekannter Schalter.**  
  
 Ein unbekannter Schalter wurde in der Befehlszeile angegeben.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Rechtschreibung des Schalters.  Um die Liste der gültigen Schalter anzuzeigen, geben Sie `msbuild /help` ein.
