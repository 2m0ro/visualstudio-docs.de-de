# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
Der Unterabschnitt \(Unterabschnitt\) ist Include, und der Abschnitt \(Abschnitt\) ist Files.  
  
 Dieser Fehler wird meistens durch das manuelle Bearbeiten der Projektdatei verursacht.  
  
 **So beheben Sie diesen Fehler**  
  
-   Arbeiten Sie im Modus **Alle Dateien anzeigen** \(indem Sie auf der Symbolleiste des Projektmappen\-Explorers auf die entsprechende Schaltfläche klicken\), und fügen Sie die entsprechenden Dateien ein.  
  
     Wenn dieser Fehler auftritt, wird das Projekt zwar geöffnet, aber es werden keine Dateien hinzugefügt.
