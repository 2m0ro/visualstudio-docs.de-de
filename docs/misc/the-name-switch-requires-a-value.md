# The &lt;name&gt; switch requires a value.
Im Allgemeinen tritt dieser Fehler auf, wenn ein Schalter ohne einen erforderlichen Wert angegeben wird.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie den Befehl erneut ein, und geben Sie dabei einen Wert für den Schalter an.  
  
     – oder –  
  
     Geben Sie den Befehl ohne den Schalter erneut ein.
