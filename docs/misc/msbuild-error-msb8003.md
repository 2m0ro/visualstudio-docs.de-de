# MSBuild-Fehler MSB8003
Die *\<Variable\>*\-Variable wurde in der Registrierung nicht gefunden.  "TargetFrameworkVersion" oder "PlatformToolset" wurden möglicherweise auf eine ungültige Versionsnummer festgelegt.
