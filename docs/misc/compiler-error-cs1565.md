# Compilerfehler CS1565
Die angegebenen Optionen führen zu einem Konflikt: Win32\-Ressourcendatei; Win32\-Symbol  
  
 Es ist nicht zulässig, die beiden Compileroptionen [\/win32res](/dotnet/csharp/language-reference/compiler-options/win32res-compiler-option) und [\/win32ico](/dotnet/csharp/language-reference/compiler-options/win32icon-compiler-option) für dieselbe Kompilierung anzugeben.
