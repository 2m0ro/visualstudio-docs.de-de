# Visual Studio has encountered an unexpected error.
Dieser Fehler tritt normalerweise auf, wenn Visual Studio Daten in einem unerwarteten Format oder aus einer unerwarteten Quelle empfängt.  
  
### So beheben Sie diesen Fehler  
  
-   Schließen Sie Visual Studio, und starten Sie das Programm anschließend neu.
