# Disk full: &lt;name&gt;
Das Laufwerk mit dem Programm verfügt beim Schließen des Programms nicht über ausreichenden Speicherplatz zum Speichern von Programmdateien, z. B. **toolbox.tbd**.  
  
### So beheben Sie diesen Fehler  
  
1.  Entfernen Sie Dateien vom Laufwerk, und versuchen Sie erneut, das Programm zu schließen.
