# MSBuild Error MSB3484
**MSB3484: Das Signaturziel \<Datei\> wurde nicht gefunden.**  
  
 Dieser Fehler wird generiert, wenn die Datei, die Sie signieren möchten, nicht gefunden wurde.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass Sie eine gültige Datei für die Signierung angeben.
