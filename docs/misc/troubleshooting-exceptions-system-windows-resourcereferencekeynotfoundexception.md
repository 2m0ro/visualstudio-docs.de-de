# Problembehandlung bei Ausnahmen: System.Windows.ResourceReferenceKeyNotFoundException
Eine <xref:System.Windows.ResourceReferenceKeyNotFoundException>\-Ausnahme wird möglicherweise während Analyse oder der Serialisierung von Markuperweiterungsressourcen ausgelöst.
