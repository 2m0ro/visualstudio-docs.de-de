# Problembehandlung bei Ausnahmen: System.ServiceModel.Dispatcher.InvalidBodyAccessException
Eine <xref:System.ServiceModel.Dispatcher.InvalidBodyAccessException>\-Ausnahme stellt die abstrakte Basisklasse für die Ausnahmen dar, die ausgelöst werden, wenn unzulässigerweise versucht wird, auf den Text einer Meldung zuzugreifen.
