# Problembehandlung bei Ausnahmen: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
Die <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException>\-Ausnahme wird ausgelöst, wenn in der Pipelineverzeichnisstruktur ein Segmentverzeichnis fehlt.  
  
## Hinweise  
 Beim Aktivieren eines Add\-Ins wird die zugehörige Pipeline zum Host erstellt. Für das Add\-In\-System müssen die Pipelinesegmente in einer angegebenen Verzeichnisstruktur mit angegebenen Verzeichnisnamen platziert werden, wie in [Anforderungen für die Pipelineentwicklung](http://msdn.microsoft.com/de-de/ef9fa986-e80b-43e1-868b-247f4c1d9da5) beschrieben.
