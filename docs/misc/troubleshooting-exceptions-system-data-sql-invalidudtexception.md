# Problembehandlung bei Ausnahmen: System.Data.Sql.InvalidUdtException
Diese Ausnahme wird bei dem Versuch ausgelöst, eine SQL\-Operation auszuführen und die einheitliche Datenübertragung \(UDT, Uniform Data Transfer\) ungültig ist.
