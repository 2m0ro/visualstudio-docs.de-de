# Problembehandlung bei Ausnahmen: System.Configuration.SettingsPropertyNotFoundException
Diese Ausnahme wird bei dem Versuch ausgelöst, auf eine nicht vorhandene Konfigurationseigenschaft zuzugreifen oder eine solche zu ändern.
