# Unable to customize.
Dieser Fehler tritt beim Debuggen einer Anwendung auf, wenn Sie versuchen, Menüs oder Symbolleisten anzupassen.  
  
### So beheben Sie diesen Fehler  
  
-   Warten Sie mit der Anpassung von Symbolleisten oder Menüs, bis der Debugvorgang abgeschlossen ist.  
  
     \- oder \-  
  
     Beenden Sie den Debugvorgang, und nehmen Sie dann die Anpassungen an der Symbolleiste oder dem Menü vor.
