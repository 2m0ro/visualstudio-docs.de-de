# Problembehandlung bei Ausnahmen: System.Data.SqlTypes.SqlTruncateException
Diese Ausnahme wird ausgelöst, wenn das Festlegen eines Werts in einer <xref:System.Data.SqlTypes>\-Struktur zu einem Abschneiden dieses Werts führen würde.
