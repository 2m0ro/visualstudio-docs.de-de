# Problembehandlung bei Ausnahmen: System.Xml.Schema.XmlSchemaInferenceException
Gibt Informationen über Fehler zurück, auf die die <xref:System.Xml.Schema.XmlSchemaInference>\-Klasse beim Ableiten eines Schemas von einem XML\-Dokument stößt.
