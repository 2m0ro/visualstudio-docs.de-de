# Tab name &lt;name&gt; already exists.
Jede Registerkarte in der Toolbox muss einen eindeutigen Namen aufweisen.  Im Allgemeinen tritt dieser Fehler auf, wenn der eingegebene Name für die Registerkarte der Toolbox mit dem einer vorhandenen Registerkarte der Toolbox übereinstimmt.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie einen eindeutigen Namen für die Registerkarte ein.
