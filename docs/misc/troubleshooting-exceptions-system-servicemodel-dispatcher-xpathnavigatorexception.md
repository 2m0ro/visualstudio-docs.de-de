# Problembehandlung bei Ausnahmen: System.ServiceModel.Dispatcher.XPathNavigatorException
Eine <xref:System.ServiceModel.Dispatcher.XPathNavigatorException>\-Ausnahme wird ausgelöst, wenn der Anteil der Knoten, für die eine Prüfung von einem <xref:System.Xml.XPath.XPathNavigator> durchgeführt werden kann, überschritten wird.
