# Cannot find main menu commandbar in resources.
Dieser Fehler tritt auf, wenn der MenuBar\-Befehl von der integrierten Entwicklungsumgebung beim Start nicht gefunden wird.  Die DLL wurde gelöscht, verschoben oder ist beschädigt.  
  
### So beheben Sie diesen Fehler  
  
-   Verwenden Sie **Software**, um das Produkt zu reparieren oder neu zu installieren.
