# The following components could not be browsed: &lt;name&gt;
Im Allgemeinen tritt dieser Fehler auf, wenn Sie eine Datei hinzufügen, die das Durchsuchen von Objekten wie HTM\-Dateien und TXT\-Dateien im Dialogfeld **Komponentenauswahl** nicht unterstützt.  Der Objektkatalog unterstützt Dateitypen wie **BSC**, **OLB**, **TLB**, **DLL**, **EXE** und **OCX**.  
  
### So beheben Sie diesen Fehler  
  
1.  Entfernen Sie den nicht unterstützten Dateityp aus der Liste **Ausgewählte Komponenten**, und wählen Sie eine andere Datei aus.
