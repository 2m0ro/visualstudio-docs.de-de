# Path or drive is not valid.
Dieser Fehler tritt normalerweise auf, wenn das Feld **Durchsuchen** im Dialogfeld **Suchen nach** leer ist und Sie dann die EINGABETASTE drücken.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie einen Pfad oder ein Verzeichnis ein, und drücken Sie dann die EINGABETASTE.
