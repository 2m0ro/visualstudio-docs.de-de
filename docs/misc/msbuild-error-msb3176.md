# MSBuild Error MSB3176
**MSB3176: Die angegebene mindestens erforderliche Version ist höher als die aktuelle veröffentlichte Version.  Geben Sie eine Version an, die der aktuellen veröffentlichten Version entspricht oder eine ältere Version ist.**  
  
 Dieser Fehler wird generiert, wenn die mindestens erforderliche Anwendungsversion höher als die aktuelle Version des Bereitstellungsmanifests ist.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie eine mindestens erforderliche Anwendungsversion an, die niedriger oder gleich der derzeit veröffentlichten Version ist.
