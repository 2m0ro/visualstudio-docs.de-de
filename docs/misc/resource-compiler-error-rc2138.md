# Ressourcencompiler: Fehler RC2138
Nicht übereinstimmendes schließendes Kommentarzeichen "\/\*"  
  
 Für eine öffnende Kommentarsequenz \(`/*`\) gibt es keine übereinstimmende schließende Kommentarsequenz \(`*/`\). Dieser Fehler kann durch das Schachteln von Kommentaren verursacht werden.
