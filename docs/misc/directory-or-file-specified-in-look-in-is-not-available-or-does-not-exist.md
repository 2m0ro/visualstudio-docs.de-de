# Directory or file specified in Look In is not available or does not exist.
Im Allgemeinen tritt dieser Fehler auf, wenn der in der Liste **Durchsuchen** angegebene Pfad bzw. Dateiname nicht gefunden werden kann.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie Pfad und Dateinamen erneut ein.  
  
     \- oder \-  
  
2.  Navigieren Sie mithilfe der Liste **Durchsuchen** und des Felds **Verfügbare Elemente** zur richtigen Datei bzw. zum richtigen Ordner.
