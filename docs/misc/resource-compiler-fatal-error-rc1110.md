# Ressourcencompiler: Schwerwiegender Fehler RC1110
Dateiname konnte nicht geöffnet werden.  
  
 Der Ressourcencompiler konnte die angegebene Ressourcenskriptdatei nicht öffnen. Stellen Sie sicher, dass die Datei vorhanden ist.
