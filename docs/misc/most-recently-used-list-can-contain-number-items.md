# Most recently used list can contain &lt;number&gt; items.
MRU\-Listen können bis zu 24 Elemente enthalten.  Im Allgemeinen tritt dieser Fehler auf, wenn im Dialogfeld **Optionen** unter **Umgebung** im Bereich **Allgemein** im Feld **Zuletzt geöffnete Listen enthalten \<Anzahl\> Elemente** eine Zahl kleiner als 1 oder größer als 24 eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie im Feld eine Zahl zwischen 1 und 24 ein.
