# Property value is not valid. (Visual Studio) (2)
Im Allgemeinen tritt dieser Fehler auf, wenn ein ungültiger Wert für eine Eigenschaft eingegeben wird.  Dies kann beispielsweise eintreten, wenn das falsche Format für einen Ganzzahlwert oder ein zu großer Wert für einen `Short`\-Wert eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie einen Wert ein, der für die Eigenschaft gültig ist.  
  
    > [!TIP]
    >  Ermitteln Sie die gültigen Werte für diese Eigenschaft in der Onlinedokumentation.
