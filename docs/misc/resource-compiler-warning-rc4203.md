# Ressourcencompiler: Warnung RC4203
SHIFT oder CONTROL wurden ohne VIRTKEY verwendet.  
  
 In einer Zugriffstasten\-Tabellenressource ist für SHIFT oder CONTROL entsprechend VIRTKEY erforderlich. Da SHIFT und CONTROL in einer VIRTKEY\-Typzugriffstaste als Flagbits gekennzeichnet sind, können sie nicht unabhängig von VIRTKEY existieren.
