# MSBuild Error MSB3171
**MSB3171: Problem beim Generieren des Manifests. '  \<Fehler\>'**  
  
 Dies ist ein unerwarteter Fehler, der beim Abrufen von Informationen über die Datei oder Assembly, auf die verwiesen wird, generiert wird.  Diese Meldung kann bei der Erstellung des Anwendungsmanifests oder des Bereitstellungsmanifests generiert werden.  
  
### So beheben Sie diesen Fehler  
  
-   Weitere Informationen zur Ursache und zu möglichen Lösungen eines MSBuild\-Fehlers finden Sie unter [Additional Resources](../msbuild/additional-msbuild-resources.md).
