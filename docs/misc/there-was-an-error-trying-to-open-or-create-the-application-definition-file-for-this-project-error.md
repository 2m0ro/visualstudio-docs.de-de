# There was an error trying to open or create the application definition file for this project. &#39;&lt;error&gt;&#39;
Dieser Fehler tritt auf, wenn Sie versuchen, die Anwendungsdefinitionsdatei zu öffnen, diese jedoch nicht verfügbar ist.  Sie wurde möglicherweise gelöscht oder verschoben.  
  
### So beheben Sie diesen Fehler  
  
-   Erstellen Sie die Anwendungsdefinitionsdatei neu.
