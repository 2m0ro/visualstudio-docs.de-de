# Ressourcencompiler: Fehler RC2014
Präprozessorbefehl muss mit dem ersten Zeichen in der Zeile beginnen.  
  
 Vor dem Gatterzeichen \(**\#**\) einer Präprozessordirektive sind in der gleichen Zeile Zeichen vorhanden, die keine Leerzeichen sind.
