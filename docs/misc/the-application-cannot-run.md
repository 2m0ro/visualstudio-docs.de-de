# The application cannot run.
Zum Ausführen von Visual Studio ist nicht genügend Arbeitsspeicher vorhanden.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass der Computer die Mindestanforderungen zum Ausführen von Visual Studio erfüllt.  
  
2.  Schließen Sie andere Programme, die auf dem Computer ausgeführt werden, um Arbeitsspeicher freizugeben.  
  
     \- oder \-  
  
     Installieren Sie Visual Studio neu.
