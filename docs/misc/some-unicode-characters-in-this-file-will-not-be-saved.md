# Some Unicode characters in this file will not be saved.
Im Allgemeinen tritt dieser Fehler auf, wenn eine Datei Zeichen enthält, die nur gespeichert werden können, indem eine Unicode\-Codepage zugewiesen wird.  
  
### So speichern Sie die Datei mit Unicode\-Zeichen  
  
1.  Klicken Sie auf **Abbrechen**.  
  
2.  Wählen Sie im Menü **Datei** den Befehl **Erweiterte Speicheroptionen**.  
  
3.  Wählen Sie im Dialogfeld **Erweiterte Speicheroptionen** die für Ihre Datei geeignete Codierung aus.
