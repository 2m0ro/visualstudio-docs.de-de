# Problembehandlung bei Ausnahmen: System.ServiceModel.Channels.InvalidChannelBindingException
Eine <xref:System.ServiceModel.Channels.InvalidChannelBindingException>\-Ausnahme wird ausgelöst, wenn die angegebene Bindung nicht mit den Vetragsanforderungen für den Dienst übereinstimmt.
