# Problembehandlung bei Ausnahmen: System.DirectoryServices.Protocols.DsmlInvalidDocumentException
Diese Ausnahme wird ausgelöst, wenn ein DSML\-Dokument \(Anforderungs\- oder Antwortdokument\) in XML nicht wohlgeformt ist und mit dem DSMLV2\-Schema nicht überprüft werden kann \(DSML: Directory Services Markup Language\).
