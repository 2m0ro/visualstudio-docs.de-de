# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
Von einem Projekt konnte nicht auf ein bestimmtes anderes Projekt verwiesen werden.  
  
 Die Hauptursache für diesen Fehler stellt das Verweisen auf ein Projekt dar, das eine EXE\-Datei erstellt.  Dieser Fehler gibt außerdem an, dass ein Projekt auf ein Projekt verweist, dessen Ausgabe keine Assembly ist \(z. B. eine systemeigene C\+\+\-Anwendung\).
