# Ressourcencompiler: Fehler RC2131
Zeichenfolge in Anführungszeichen für Schlüssel erwartet.  
  
 Das Feld für diese Anweisung muss eine Zeichenfolge sein, die in doppelte Anführungszeichen eingeschlossen wird.
