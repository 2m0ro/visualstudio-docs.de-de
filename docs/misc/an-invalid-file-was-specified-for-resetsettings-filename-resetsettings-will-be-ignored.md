# An invalid file was specified for /resetsettings: &lt;filename&gt;.  /resetsettings will be ignored.
Dieser Fehler tritt im Allgemeinen auf, wenn die Einstellungsdatei nicht gelesen werden kann, nicht vorhanden ist oder fehlerhaften XML\-Code enthält.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie, ob der Dateiname richtig ist.  
  
2.  Stellen Sie sicher, dass die Datei das richtige Format aufweist.  
  
3.  Vergewissern Sie sich, dass die Datei fehlerfreien XML\-Code enthält.
