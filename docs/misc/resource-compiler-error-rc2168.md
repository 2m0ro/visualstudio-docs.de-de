# Ressourcencompiler: Fehler RC2168
Ressource ist zu groß.  
  
 Die Größenbegrenzung einer Ressource wurde überschritten. Dieser Grenzwert gilt nicht für Cursor, Symbole, Bitmaps oder andere dateibasierte Ressourcen.
