# &lt;number&gt; power closure argument &#39;n&#39; must be from 1 to 256.
Dieser Fehler tritt normalerweise auf, wenn die genannte Zahl nicht im gültigen Bereich von 1 bis 256 liegt.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie eine Zahl zwischen 1 und 256 ein, und versuchen Sie es erneut.  Weitere Informationen zur Syntax finden Sie unter [Reguläre Ausdrücke](../ide/using-regular-expressions-in-visual-studio.md).
