# MSBuild-Fehler MSB4137
**MSB4137: In der Konfigurationsdatei ist bei "\<x\>" ein ungültiger Wert angegeben.  Weder der Eigenschaftenname oder der ToolsVersion\-Name können eine leere Zeichenfolge sein.**  
  
 Eine Toolsetdefinition enthält ungültige Eigenschaftennamen oder \-werte.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass die Toolsetdefinition keine ungültigen oder leeren Eigenschaftennamen oder \-werte enthält.
