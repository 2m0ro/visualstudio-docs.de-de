# Funktionsspezifizierer
Sie können die Schlüsselwörter [inline](../misc/inline-inline-forceinline.md) und [virtual](/visual-cpp/cpp/virtual-specifier) als Bezeichner in Funktionsdeklarationen verwenden. Diese Verwendung des Schlüsselworts **virtual** unterscheidet sich von der Verwendung im Basisklassenspezifizierer einer Klassendefinition.
