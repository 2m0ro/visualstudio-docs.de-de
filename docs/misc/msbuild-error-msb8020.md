# MSBuild-Fehler MSB8020
Die Buildtools für *\<Plattform\>* \(Plattformtoolset \= "*\<Version\>*"\) wurden nicht gefunden. Installieren Sie zum Erstellen mithilfe der *\<Version\>*\-Buildtools die *\<Plattform\>*\-Buildtools.  Alternativ können Sie ein Upgrade auf die aktuellen Visual Studio\-Tools durchführen, indem Sie das Menü "Projekt" auswählen oder mit der rechten Maustaste auf die Projektmappe klicken und dann "Lösung neu zuordnen" auswählen.
