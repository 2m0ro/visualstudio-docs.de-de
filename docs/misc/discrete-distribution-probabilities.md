# discrete_distribution::probabilities
Gibt die gespeicherten Wahrscheinlichkeiten für die Verteilung zurück.  
  
## Syntax  
  
```  
std::vector<double> probabilities() const;  
```  
  
## Hinweise  
 Diese Memberfunktion gibt die gespeicherten Wahrscheinlichkeitswerte zurück.

 **Header:** \<random\>  
  
 **Namespace:** std
