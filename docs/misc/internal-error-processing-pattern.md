# Internal error processing pattern.
Dieser Fehler tritt normalerweise auf, wenn ein regulärer Ausdruck zu einer Suchzeichenfolge hinzugefügt wurde, die Syntax für den regulären Ausdruck jedoch Fehler enthält.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie die Suchzeichenfolge erneut unter Verwendung der korrekten Syntax für reguläre Ausdrücke ein.  Weitere Informationen finden Sie unter [Reguläre Ausdrücke](../ide/using-regular-expressions-in-visual-studio.md).
