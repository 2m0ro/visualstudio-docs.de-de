# Auslassungszeichen (...)
Der Auslassungspunkteoperator \(`...`\) wird je nach Kontext häufig in C und C\+\+ verwendet.  Er wurde ursprünglich in C als abstrakter Deklarator für Parameterlisten für variable Funktionen verwendet, wird aber auch in der Präprozessormakroprogrammierung in Form von variadic\-Makros eingesetzt.  In C\+\+ wird `...` bei der Ausnahmebehandlung von `catch`\-Blöcken und in C\+\+11 wird es für variadic\-Vorlagen verwendet.
