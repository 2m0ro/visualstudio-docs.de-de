# Problembehandlung bei Ausnahmen: System.ServiceModel.CommunicationObjectFaultedException
Eine <xref:System.ServiceModel.CommunicationObjectFaultedException>\-Ausnahme wird ausgelöst, wenn ein Aufruf für ein fehlgeschlagenes Kommunikationsobjekt erfolgt.
