# MSBuild Error MSB3121
**MSB3121: Im Dateizuordnungselement im Anwendungsmanifest fehlen eines oder mehrere der folgenden erforderlichen Attribute: Erweiterung, Beschreibung, Programm\-ID oder Standardsymbol.**  
  
 Das [ClickOnce Deployment Manifest](../deployment/clickonce-deployment-manifest.md) muss Werte für alle vier Attribute enthalten.  
  
### So beheben Sie diesen Fehler  
  
-   Legen Sie jedes [ClickOnce Deployment Manifest](../deployment/clickonce-deployment-manifest.md)\-Attribut auf einen gültigen Wert fest.
