# ^-Operator (C++)

-   [Bitweiser exklusiver OR\-Operator: ^](../Topic/Bitwise%20Exclusive%20OR%20Operator:%20%5E.md), für die Nutzung in allen Arten von C\+\+.  
  
-   [Handle für Objekt \(^\)](/visual-cpp/windows/handle-to-object-operator-hat-cpp-component-extensions), für die Nutzung in [!INCLUDE[cppwrt_short](../misc/includes/cppwrt_short_md.md)] und [!INCLUDE[cppcli](../misc/includes/cppcli_md.md)]
