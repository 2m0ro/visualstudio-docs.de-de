# Ressourcencompiler: Schwerwiegender Fehler RC1207
Standardcodepage ist ungültig  
  
 Die durch die Umgebungsvariable **RCCODEPAGE** angegebene Codepage ist ungültig. Weitere Informationen hierzu finden Sie unter [Codepages](/visual-cpp/c-runtime-library/code-pages) in der *Laufzeitbibliotheksreferenz*.
