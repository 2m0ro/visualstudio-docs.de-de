# Application did not register a class factory for class ID &lt;number&gt;.
Ein COM\-Objekt, das die IClassFactory\-Schnittstelle für die Anwendung implementiert, wurde nicht registriert.  Dies ist ein Fehler auf Betriebssystem\-Ebene.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie Visual Studio oder die betroffene Anwendung neu, und versuchen Sie es erneut.
