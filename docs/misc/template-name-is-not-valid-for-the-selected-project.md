# Template &lt;name&gt; is not valid for the selected project.
Im Allgemeinen tritt dieser Fehler auf, wenn ein unbekannter Name als Vorlage für den Befehl **Neue Datei**, `File.NewFile` oder für den Befehl **Neues Projektmappenelement hinzufügen** `File.AddNewSolutionItem` eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie den Befehl und seine Argumente neu ein.  
  
     – oder –  
  
2.  Schlagen Sie die korrekte Syntax in den Themen "Befehl **Neue Datei**" und "Befehl **Neues Projektmappenelement hinzufügen**" nach.
