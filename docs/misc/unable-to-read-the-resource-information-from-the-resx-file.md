# Unable to read the resource information from the resx file
Das Projektsystem konnte eine RESX\-Datei nicht lesen.  
  
 Eine für das Betriebssystem spezifische Ursache \(falls vorhanden\) wird an das Ende dieser Meldung angehängt.  Der Dateiname der Datei, die nicht gelesen werden konnte, wird ebenfalls angehängt.  
  
 Dieser Fehler kann z. B. durch fehlende Berechtigungen zum Lesen der Dateien oder \(bei Webprojekten\) durch eine unterbrochene Verbindung zum Webserver ausgelöst werden.  
  
 Durch diesen Fehler schlägt der Buildprozess fehl.
