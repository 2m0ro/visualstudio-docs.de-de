# Type library &lt;name&gt; was not found.
Visual Studio kann nicht ohne die Typbibliotheken DTE.olb und vsext.tlb ausgeführt werden.  Im Allgemeinen tritt dieser Fehler auf, wenn diese beiden Dateien entfernt oder gelöscht wurden.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie Visual Studio neu.
