# MSBuild Error MSB3158
**MSB3158: Ressourcen f√ºr Kultur \<Kultur\> konnten nicht gefunden werden.  Kultur \<Kultur\> wird verwendet.**  
  
 Diese Warnung wird generiert, wenn das angegebene Produkt keine Paketmanifestdatei \(package.xml\) finden kann, die die angegebene Kultur verwendet.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie die entsprechende Paketmanifestdatei an.
