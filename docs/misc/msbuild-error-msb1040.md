# MSBuild-Fehler MSB1040
**MSB1040: ToolsVersion ist nicht gültig.**  
  
 Der mit dem **\/ToolsVersion**\-Schalter angegebene `ToolsVersion`\-Wert ist ungültig.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie beim Erstellen einen gültigen `ToolsVersion`\-Wert an, indem Sie den **\/ToolsVersion**\-Schalter in der Befehlszeile verwenden.
