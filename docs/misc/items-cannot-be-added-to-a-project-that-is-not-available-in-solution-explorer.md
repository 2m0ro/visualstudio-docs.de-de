# Elemente k&#246;nnen nicht zu einem Projekt hinzugef&#252;gt werden, das im Projektmappen-Explorer nicht verf&#252;gbar ist.
Dieser Fehler tritt im Allgemeinen auf, wenn ein Projekt angegeben wurde, das im Projektmappen\-Explorer nicht geöffnet ist. Elemente können in Visual Studio nicht zu einem Projekt hinzugefügt werden, das im Projektmappen\-Explorer gerade nicht verfügbar ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Fügen Sie das angegebene Projekt der aktuellen Projektmappe hinzu, oder öffnen Sie die Projektmappe, in der sich das angegebene Projekt befindet.
