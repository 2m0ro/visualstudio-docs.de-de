# Problembehandlung bei Ausnahmen: System.Windows.Automation.ElementNotEnabledException
Eine <xref:System.Windows.Automation.ElementNotEnabledException>\-Ausnahme enthält Informationen über die Ausnahme, die bei dem Versuch ausgelöst wird, ein nicht aktiviertes Steuerelement zu ändern.
