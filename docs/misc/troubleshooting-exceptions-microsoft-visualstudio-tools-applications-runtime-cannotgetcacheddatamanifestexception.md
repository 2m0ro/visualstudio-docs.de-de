# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotGetCachedDataManifestException
Diese Ausnahme wird ausgelöst, wenn das Manifest, in dem alle zwischengespeicherten Datenobjekte aufgelistet sind, beschädigt ist oder fehlt.
