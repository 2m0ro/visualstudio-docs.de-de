# MSBuild Error MSB3189
**MSB3189: Der Updatepfad für diese Anwendung ist ein lokaler Pfad.**  
  
 Diese Warnung gilt nur für Bereitstellungsmanifeste.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass der Updatespeicherort eine gültige URL ist.
