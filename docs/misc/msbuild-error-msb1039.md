# MSBuild-Fehler MSB1039
**MSB1039: Geben Sie die Version des Toolsets an.**  
  
 Bei Verwendung des **\/ToolsVersion**\-Schalters wurde kein Wert angegeben.  
  
### So beheben Sie diesen Fehler  
  
-   Wenn Sie den **\/ToolsVersion**\-Schalter verwenden, stellen Sie sicher, dass Sie einen gültigen, nicht leeren Wert für `ToolsVersion` angeben.
