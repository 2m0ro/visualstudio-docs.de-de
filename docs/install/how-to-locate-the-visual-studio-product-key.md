# Gewusst wie: Suchen des Visual Studio Product Key
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Sie möchten unter Umständen den Product Key für [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] suchen, wenn Sie mit dem Produktsupport arbeiten, ein Upgrade von einer Testversion ausführen oder Ihre eigenen Datensätze aktualisieren.  
  
> [!NOTE]
>  Um den Product Key für [!INCLUDE[esprtfs](../code-quality/includes/esprtfs_md.md)] zu suchen, führen Sie die Schritte unter [Suchen oder Ändern des Product Key für Team Foundation Server](../Topic/Locate%20or%20Change%20the%20Product%20Key%20for%20Team%20Foundation%20Server.md) aus.  
  
### So suchen Sie den Product Key für Visual Studio  
  
-   Der Product Key befindet sich in der Regel an den folgenden Orten\/Stellen:  
  
    -   Auf den Installationsmedien \(falls zutreffend\).  
  
    -   Auf der Seite [MSDN\-Abonnements](http://go.microsoft.com/fwlink/?LinkId=206363) der Microsoft\-Website.  
  
    -   Der Dokumentation oder anderen Mitteilungen vom Anbieter, von dem Sie die Software gekauft haben.
