# <a name="modify-visual-studio-2017-rc"></a>Ändern von Visual Studio 2017 RC
Wir haben nicht nur das Personalisieren von Visual Studio entsprechend Ihren Aufgaben erleichtert, sondern haben es außerdem einfacher für Sie gemacht, Visual Studio zu ändern. Sie brauchen dazu nicht mehr die Systemsteuerung zu durchsuchen. Sie starten einfach den neuen Visual Studio-Installer und führen die gewünschten Änderungen aus.  

 Gehen Sie folgendermaßen vor:  

## <a name="modify-workloads"></a>Ändern von Arbeitsauslastungen  
 Arbeitsauslastungen enthalten die Features, die Sie für die verwendete Programmiersprache oder Plattform benötigen. Verwenden Sie Arbeitsauslastungen, um Visual Studio so zu ändern, dass die Arbeit, die Sie ausführen möchten zum gewünschten Zeitpunkt unterstützt werden.  

1.  Suchen Sie den Visual Studio-Installer auf Ihrem Computer.  

     Auf einem Computer mit Windows 10 Anniversary Update klicken Sie beispielsweise auf **Start** und blättern dann zum Buchstaben **M**, wo das Installationsprogramm als **Microsoft Visual Studio-Installer** angezeigt wird.  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png "Suchen des Microsoft Visual Studio-Installers")

2.  Starten Sie das Installationsprogramm, indem Sie darauf klicken, und klicken Sie dann auf **Ändern**.  

     ![Ändern von Visual Studio 2017 RC, starten oder ändern](media/vs2017modify-ModifyOrLaunch.png "Starten oder Ändern von Visual Studio 2017")  

3.  Aktivieren oder deaktivieren Sie auf dem Bildschirm **Arbeitsauslastungen** die Arbeitsauslastungen, die Sie installieren bzw. deinstallieren möchten.  

    ![Visual Studio 2017 RC-Setupdialogfeld](media/vs2017modify-ChooseWorkload.png "Auswählen einer Arbeitsauslastung in Visual Studio 2017")

4. Klicken Sie auf **Ändern**.  

5. Nachdem die neuen Arbeitsauslastungen und Komponenten installiert wurden, klicken Sie auf **Starten**.

## <a name="modify-individual-components"></a>Ändern einzelner Komponenten

Wenn Sie das praktische Arbeitsauslastungsfeature zum Anpassen Ihrer Visual Studio-Installation nicht verwenden möchten, wählen Sie im Visual Studio-Installer die Option **Einzelne Komponenten** und dann die gewünschten Komponenten aus, und folgen Sie den Anweisungen.  

> [!IMPORTANT]
> Während Visual Studio 2017 RC im Allgemeinen für die Verwendung in einer Produktionsumgebung unterstützt wird, werden die Arbeitsauslastungen und Komponenten, die in der Benutzeroberfläche der Installation als „Vorschau“ gekennzeichnet sind, nicht für die Verwendung in einer Produktionsumgebung unterstützt.
