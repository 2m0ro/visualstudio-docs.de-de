# Gewusst wie: Hinzuf&#252;gen einer Funktion zu SharePoint-L&#246;sungen
  Wenn Sie eine SharePoint\-Lösung erstellen, werden der Lösung eine Funktion und ein Paket hinzugefügt.  Sie können weitere Funktionselemente hinzufügen, um verschiedene Typen von SharePoint\-Dateien zu organisieren.  
  
### So fügen Sie eine Funktion über den Projektmappen\-Explorer hinzu  
  
1.  In **Projektmappen\-Explorer** öffnen Sie das Kontextmenü für **Funktionen**, und wählen Sie dann **Funktion hinzufügen** aus.
