# <a name="how-to-add-or-remove-sharepoint-connections"></a>Vorgehensweise: Hinzufügen oder Entfernen von SharePoint-Verbindungen
  Server-Explorer können Sie die SharePoint-Websites sowie datenverbindungen zu durchsuchen. Jedoch, bevor Sie den Inhalt der SharePoint-Website durchsuchen können Sie müssen diese hinzufügen, die **SharePoint-Verbindungen** Knoten.  
  
### <a name="to-add-a-sharepoint-site-to-the-sharepoint-connections-node"></a>So fügen Sie einer SharePoint-Website auf der SharePoint-Verbindungsknotens hinzu
  
1.  Wählen Sie in der Menüleiste **Ansicht**, **Server-Explorer**.  
  
2.  In **Server-Explorer**, wählen Sie die **SharePoint-Verbindungen** Knoten, und anschließend auf der Menüleiste die Optionen **Tools** > **SharePoint hinzufügen Verbindung**.  
  
3.  In der **SharePoint-Verbindung hinzufügen** Geben Sie die [!INCLUDE[TLA2#tla_url](../sharepoint/includes/tla2sharptla-url-md.md)] für die SharePoint-Website (z. B. http://testserver/sites/unittests).  
  
### <a name="to-delete-a-sharepoint-site-from-the-sharepoint-connections-node"></a>So löschen Sie eine SharePoint-Website aus der SharePoint-Verbindungsknotens
  
1.  Wählen Sie in der Menüleiste **Ansicht**, **Server-Explorer** öffnen **Server-Explorer**.  
  
2.  Erweitern Sie die **SharePoint-Verbindungen** Knoten, um die SharePoint-Website anzuzeigen, die Sie aus löschen möchten **Server-Explorer**.  
  
3.  Wählen Sie den Standort, und anschließend auf der Menüleiste die Optionen **bearbeiten** > **löschen**.  
  
    > [!NOTE]  
    >  Dieser Schritt ist nicht die zugrunde liegende Website löschen. Löscht nur die Verbindung von **Server-Explorer**.
