# Beispiele und exemplarische Vorgehensweisen f&#252;r SharePoint
  In diesem Abschnitt werden anhand von Beispielanwendungen, Beispielcode und schrittweise beschriebenen exemplarischen Vorgehensweisen die Syntax, die Struktur und die Verfahren erläutert, die Sie zum Erstellen von SharePoint\-Lösungen benötigen.  
  
 Der Beispielcode dient nur zur Veranschaulichung und sollte nicht unverändert in bereitgestellten Lösungen verwendet werden.  Vor allem müssen Sicherheitsaspekte stärker berücksichtigt werden.
