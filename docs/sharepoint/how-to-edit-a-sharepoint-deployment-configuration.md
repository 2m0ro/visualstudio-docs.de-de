# Gewusst wie: Bearbeiten einer SharePoint-Bereitstellungskonfiguration
  Sie können eine Bereitstellungskonfiguration erstellen oder eine vorhandene Bereitstellungskonfiguration ändern.  Beispielsweise können Sie einen einzelnen Schritt ausführen oder die Reihenfolge der Schritte im Bereitstellungsprozess ändern.  Vielleicht möchten Sie Bereitstellungskonfigurationen erstellen oder ändern, da die integrierten und programmgesteuert hinzugefügten Konfigurationen nicht geändert werden können.  
  
## Erstellen einer SharePoint\-Bereitstellungskonfiguration  
  
#### So erstellen Sie eine SharePoint\-Bereitstellungskonfiguration  
  
1.  Wählen Sie im **Projektmappen\-Explorer** ein SharePoint\-Projekt, und klicken Sie auf der Menüleiste, **Projekt** auswählen, *ProjectName***Eigenschaften** aus.  
  
2.  Auf der Registerkarte **SharePoint** wählen Sie die Schaltfläche **Neu** aus.  
  
     Das Dialogfeld **Neue Bereitstellungskonfiguration hinzufügen** wird angezeigt.  
  
3.  Im Textfeld **Name** geben Sie einen Namen für die neue Bereitstellungskonfiguration ein.  
  
4.  Im Bereich **Verfügbare Bereitstellungsschritte** wählen Sie die Schritte, die Sie der Bereitstellungskonfiguration hinzufügen möchten, wählen Sie die Schaltfläche **\>**\(\) und wählen Sie dann die Schaltfläche **OK** aus.  
  
    > [!NOTE]  
    >  Wenn Sie einen Befehl vor der Bereitstellung oder einen Befehl nach der Bereitstellung konfiguriert haben, werden diese Schritte nur ausgeführt, wenn Sie sie einer benutzerdefinierten Bereitstellungskonfiguration hinzufügen.  
  
## Ändern der aktiven Bereitstellungskonfiguration  
  
#### So ändern Sie die aktive Bereitstellungskonfiguration  
  
1.  Wählen Sie im **Projektmappen\-Explorer** ein SharePoint\-Projekt, und klicken Sie auf der Menüleiste, **Projekt** auswählen, *ProjectName***Eigenschaften** aus.  
  
2.  Wählen Sie die Registerkarte **SharePoint** aus.  
  
3.  Im Listenfeld **Aktive Bereitstellungskonfiguration** wählen Sie den Namen der Bereitstellungskonfiguration aus, die Sie verwenden möchten.
